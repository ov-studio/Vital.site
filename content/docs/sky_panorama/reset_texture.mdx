---
title: gfx.sky.panorama.reset_texture
description: Clears the currently assigned texture from the panorama sky material
badge: Client
---

### Syntax

```lua
local status = gfx.sky.panorama.reset_texture()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Reset panorama sky texture"
gfx.sky.panorama.reset_texture()
```
