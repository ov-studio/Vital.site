---
title: gfx.sky.panorama.get_energy_multiplier
description: Retrieves the current energy multiplier of the panorama sky material
badge: Client
---

### Syntax

```lua
local result = gfx.sky.panorama.get_energy_multiplier()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current energy multiplier value |

---

### Examples

```lua lineNumbers title="Print current panorama sky energy multiplier"
core.engine.print("info", "Panorama energy multiplier:", gfx.sky.panorama.get_energy_multiplier())
```
