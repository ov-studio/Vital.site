---
title: gfx.sky.panorama.get_texture
description: Retrieves the current texture path of the panorama sky material
badge: Client
---

### Syntax

```lua
local result = gfx.sky.panorama.get_texture()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` | result | Path of the currently assigned panorama texture |
| `bool` | result | `false` if no texture is currently assigned |

---

### Examples

```lua lineNumbers title="Print current panorama sky texture path"
core.engine.print("info", "Panorama texture:", gfx.sky.panorama.get_texture())
```
