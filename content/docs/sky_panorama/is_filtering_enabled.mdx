---
title: gfx.sky.panorama.is_filtering_enabled
description: Retrieves whether texture filtering is currently enabled on the panorama sky material
badge: Client
---

### Syntax

```lua
local result = gfx.sky.panorama.is_filtering_enabled()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if filtering is enabled, `false` otherwise |

---

### Examples

```lua lineNumbers title="Print whether panorama sky filtering is enabled"
core.engine.print("info", "Filtering enabled:", gfx.sky.panorama.is_filtering_enabled())
```
