---
title: self:get_type
description: Returns the type name of the instance
badge: Client
---

### Syntax

```lua
local result = self:get_type()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` \| `bool` | result | Type of the instance, `false` if unavailable |

---

### Examples

```lua lineNumbers title="Get the type of a texture instance"
local entity = texture.create("assets/sprite.png")

engine.print("info", entity:get_type()) -- 'texture'
```
