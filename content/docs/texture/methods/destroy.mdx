---
title: self:destroy
description: Destroys the texture instance and frees its resources
badge: Client
---

### Syntax

```lua
local status = self:destroy()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
local tex = texture.create("assets/sprite.png")

tex:destroy()
```
