### Values

Modes are ordered highest to lowest compression quality — or equivalently, least to most GPU compatibility.

| Constant | Description |
|----------|-------------|
| `core.texture.compression_mode.ASTC` | Flexible block sizes with best quality-to-ratio on supported hardware<br/>Arm Mali, Apple Silicon, and newer desktop GPUs |
| `core.texture.compression_mode.BPTC` | BC7 for color/alpha, BC6H for HDR<br/>Higher quality than S3TC at the same ratio — requires DX11-class GPU |
| `core.texture.compression_mode.S3TC` | BC1/BC3 — universal desktop standard<br/>Automatically uses BC1 for opaque textures and BC3 (DXT5) for alpha<br/>~6:1 compression over RGBA8 |
| `core.texture.compression_mode.ETC2` | Modern mobile standard (OpenGL ES 3.0+/Vulkan)<br/>Supports alpha — preferred for Android and iOS targets |
