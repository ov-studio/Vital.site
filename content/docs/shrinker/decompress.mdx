---
title: shrinker.decompress
description: Decodes a compressed string using zlib library
badge: Shared
---

### Syntax

```lua
local result = shrinker.decompress(input)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | input | Compressed string to be decompressed |


### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` / `bool` | result | Decompressed string on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
local input = shrinker.compress("Hello World")

engine.print("Input:", input)
engine.print("Decompressed:", shrinker.decompress(input))
```
