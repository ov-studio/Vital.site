---
title: util.shrinker.decompress
description: Decompresses a compressed string using zstd library
badge: Shared
---

### Syntax

```lua
local result = util.shrinker.decompress(
    input
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | input | Compressed string to be decompressed |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` \| `bool` | result | Decompressed string on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Decompresses an input string"
local input = util.shrinker.compress("Hello World")

engine.print("info", "Input:", input)
engine.print("info", "Decompressed:", util.shrinker.decompress(input))
```
