---
title: shrinker.decompress
description: Decodes a compressed string into equivalent buffer
badge: Shared
---

### Syntax

```lua
local buffer = shrinker.decompress(hash)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | hash | Compressed hash to be decompressed |


### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` or `bool` | buffer | Decompressed buffer on successful execution, or `false` on failure |

<hr/>

### Example

```lua
local input = shrinker.compress("Hello World")

engine.print("Input:", input)
engine.print("Decompressed:", shrinker.decompress(input))
```
