---
title: shrinker.compress
description: Compresses a string using zlib library
badge: Shared
---

### Syntax

```lua
local result = shrinker.compress(input)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | input | String containing data to be compressed |


### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` or `bool` | result | Compressed string on successful execution, or `false` on failure |

<hr/>

### Example

```lua
local input = "Hello World"

engine.print("Input:", input)
engine.print("Compressed:", shrinker.compress(input))
```
