---
title: string.void
description: Verifies whether the string is empty or contains only whitespaces
badge: Shared
---

### Syntax

```lua
local void = string.void(
    input
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | input | String to be verified |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | void | `true` if the string is empty or contains only whitespaces, `false` otherwise |

<hr/>

### Examples

```lua lineNumbers title="Return true for an empty string"
local void = string.void("")

engine.print("info", void) --true
```

```lua lineNumbers title="Return true for a whitespace-only string"
local void = string.void("      ")

engine.print("info", void) --true
```

```lua lineNumbers title="Return false for a non-empty string"
local void = string.void("Hello")

engine.print("info", void) --false
```
