---
title: string.kern
description: Inserts spacing between each character in a string
badge: Shared
---

### Syntax

```lua
local result = string.kern(
    input,
    kerner = " "
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | input | String to be kerned |
| `string` | kerner | Character(s) to insert between each character |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` | result | Kerned string |

<hr/>

### Examples
```lua lineNumbers title="Apply default single-space kerning"
local result = string.kern("HELLO")

engine.print("info", result) --'H E L L O'
```

```lua lineNumbers title="Apply custom dash kerning"
local result = string.kern("ABC", "-")

engine.print("info", result) --'A-B-C'
```

```lua lineNumbers title="Apply double-space kerning"
local result = string.kern("LUA", "  ")

engine.print("info", result) --'L  U  A'
```

```lua lineNumbers title="Apply dot kerning"
local result = string.kern("123", ".")

engine.print("info", result) --'1.2.3'
```
