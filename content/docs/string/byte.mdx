---
title: string.byte
description: Returns the internal numeric codes of characters
badge: Shared
---

### Syntax

```lua
local ... = string.byte(input, start_at = 1, end_at = #input)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | input | String to convert |
| `number` | start_at | Character position where extraction begins |
| `number` | end_at | Character position where extraction ends |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `...` | ... | Numeric byte values for the specified range |

<hr/>

### Example

```lua
--Get byte value of first character
local byte = string.byte("A")
engine.print(byte) --65
```

```lua
--Get byte value at specific position
local byte = string.byte("ABC", 2)
engine.print(byte) --66
```

```lua
--Get multiple byte values
local b1, b2, b3 = string.byte("ABC", 1, 3)
engine.print(b1, b2, b3) --65  66  67
```

```lua
--Using negative indices (from end)
local last = string.byte("Hello", -1)
engine.print(last) --111 (character 'o')
```
