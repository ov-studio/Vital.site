---
title: string.byte
description: Retrieves the internal numeric codes of characters
badge: Shared
---

### Syntax

```lua
local ... = string.byte(input, start_at = 1, end_at = -1)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | input | String to be converted |
| `int` | start_at | Character position where extraction begins - negative counts from end |
| `int` | end_at | Character position where extraction ends - negative counts from end |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `...` | ... | Numeric byte values for the specified range |

<hr/>

### Examples

```lua lineNumbers title="Get the byte value of the first character"
local byte = string.byte("A")

engine.print("info", byte) --65
```

```lua lineNumbers title="Get the byte value at a specific position"
local byte = string.byte("ABC", 2)

engine.print("info", byte) --66
```

```lua lineNumbers title="Get multiple byte values at once"
local b1, b2, b3 = string.byte("ABC", 1, 3)

engine.print("info", b1, b2, b3) --65  66  67
```

```lua lineNumbers title="Use a negative index to read from the end"
local last = string.byte("Hello", -1)

engine.print("info", last) --111 (character 'o')
```
