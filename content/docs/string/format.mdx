---
title: util.string.format
description: Returns a formatted string using format specifiers
badge: Shared
---

### Syntax

```lua
local result = util.string.format(
    format,
    ...
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | format | Format string containing specifiers<br/>Refer [**Specifiers**](#specifiers) section |
| `any` | ... | Values to format according to the format string |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` | result | Formatted string |

---

### Specifiers

| Specifier | Description |
|-----------|-------------|
| `%d` \| `%i` | Signed decimal integer |
| `%u` | Unsigned decimal integer |
| `%o` | Unsigned octal |
| `%x` | Unsigned hexadecimal (lowercase) |
| `%X` | Unsigned hexadecimal (uppercase) |
| `%f` | Decimal floating point |
| `%e` | Scientific notation (lowercase) |
| `%E` | Scientific notation (uppercase) |
| `%g` | Shortest of %e or %f |
| `%G` | Shortest of %E or %f |
| `%c` | Character (from byte value) |
| `%s` | String |
| `%q` | Quoted string |
| `%%` | Literal percent sign |

---

### Examples

```lua lineNumbers title="Format a basic string"
local str = util.string.format("Hello %s", "World")

core.engine.print("info", str) -- 'Hello World'
```

```lua lineNumbers title="Format an integer"
local num = util.string.format("Number: %d", 42)

core.engine.print("info", num) -- 'Number: 42'
```

```lua lineNumbers title="Format multiple values in one string"
local result = util.string.format("%d + %d = %d", 5, 3, 8)

core.engine.print("info", result) -- '5 + 3 = 8'
```

```lua lineNumbers title="Format a float with fixed decimal precision"
local pi = util.string.format("Pi: %.2f", util.math.pi)

core.engine.print("info", pi) -- 'Pi: 3.14'
```

```lua lineNumbers title="Format a float with more decimal places"
local precise = util.string.format("%.5f", util.math.pi)

core.engine.print("info", precise) -- '3.14159'
```

```lua lineNumbers title="Pad an integer with leading zeros"
local padded = util.string.format("%05d", 42)

core.engine.print("info", padded) -- '00042'
```

```lua lineNumbers title="Format a number as uppercase hexadecimal"
local hex = util.string.format("0x%X", 255)

core.engine.print("info", hex) -- '0xFF'
```

```lua lineNumbers title="Format a number as lowercase hexadecimal"
local hexLower = util.string.format("0x%x", 255)

core.engine.print("info", hexLower) -- '0xff'
```

```lua lineNumbers title="Format a number in scientific notation"
local sci = util.string.format("%e", 1234.5)

core.engine.print("info", sci) -- '1.234500e+03'
```

```lua lineNumbers title="Format a character from a byte value"
local char = util.string.format("%c", 65)

core.engine.print("info", char) -- 'A'
```

```lua lineNumbers title="Right-align a string within a fixed width"
local aligned = util.string.format("%10s", "right")

core.engine.print("info", aligned) -- '     right'
```

```lua lineNumbers title="Left-align a string within a fixed width"
local leftAligned = util.string.format("%-10s", "left")

core.engine.print("info", leftAligned) -- 'left     '
```

```lua lineNumbers title="Format a quoted string with escaped characters"
local quoted = util.string.format("%q", "hello\nworld")

core.engine.print("info", quoted) -- '"hello\nworld"'
```

```lua lineNumbers title="Combine multiple format specifiers"
local info = util.string.format("Name: %s, Age: %d, Score: %.1f%%", "Alice", 25, 95.5)

core.engine.print("info", info) -- 'Name: Alice, Age: 25, Score: 95.5%'
```

```lua lineNumbers title="Format a number as octal"
local oct = util.string.format("%o", 8)

core.engine.print("info", oct) -- '10'
```

```lua lineNumbers title="Include a literal percent sign"
local percent = util.string.format("Progress: 100%%")

core.engine.print("info", percent) -- 'Progress: 100%'
```
