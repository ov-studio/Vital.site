---
title: string.format
description: Returns a formatted string using format specifiers
badge: Shared
---

### Syntax

```lua
local result = string.format(
    format,
    ...
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | format | Format string containing specifiers<br/>Refer [**Specifiers**](#specifiers) section |
| `...` | ... | Values to be formatted according to the format string |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` | result | Formatted string |

<hr/>

### Specifiers

| Specifier | Description |
|-----------|-------------|
| `%d` \| `%i` | Signed decimal integer |
| `%u` | Unsigned decimal integer |
| `%o` | Unsigned octal |
| `%x` | Unsigned hexadecimal (lowercase) |
| `%X` | Unsigned hexadecimal (uppercase) |
| `%f` | Decimal floating point |
| `%e` | Scientific notation (lowercase) |
| `%E` | Scientific notation (uppercase) |
| `%g` | Shortest of %e or %f |
| `%G` | Shortest of %E or %f |
| `%c` | Character (from byte value) |
| `%s` | String |
| `%q` | Quoted string |
| `%%` | Literal percent sign |

<hr/>

### Examples

```lua lineNumbers title="Format a basic string"
local str = string.format("Hello %s", "World")

engine.print("info", str) --'Hello World'
```

```lua lineNumbers title="Format an integer"
local num = string.format("Number: %d", 42)

engine.print("info", num) --'Number: 42'
```

```lua lineNumbers title="Format multiple values in one string"
local result = string.format("%d + %d = %d", 5, 3, 8)

engine.print("info", result) --'5 + 3 = 8'
```

```lua lineNumbers title="Format a float with fixed decimal precision"
local pi = string.format("Pi: %.2f", math.pi)

engine.print("info", pi) --'Pi: 3.14'
```

```lua lineNumbers title="Format a float with more decimal places"
local precise = string.format("%.5f", math.pi)

engine.print("info", precise) --'3.14159'
```

```lua lineNumbers title="Pad an integer with leading zeros"
local padded = string.format("%05d", 42)

engine.print("info", padded) --'00042'
```

```lua lineNumbers title="Format a number as uppercase hexadecimal"
local hex = string.format("0x%X", 255)

engine.print("info", hex) --'0xFF'
```

```lua lineNumbers title="Format a number as lowercase hexadecimal"
local hexLower = string.format("0x%x", 255)

engine.print("info", hexLower) --'0xff'
```

```lua lineNumbers title="Format a number in scientific notation"
local sci = string.format("%e", 1234.5)

engine.print("info", sci) --'1.234500e+03'
```

```lua lineNumbers title="Format a character from a byte value"
local char = string.format("%c", 65)

engine.print("info", char) --'A'
```

```lua lineNumbers title="Right-align a string within a fixed width"
local aligned = string.format("%10s", "right")

engine.print("info", aligned) --'     right'
```

```lua lineNumbers title="Left-align a string within a fixed width"
local leftAligned = string.format("%-10s", "left")

engine.print("info", leftAligned) --'left     '
```

```lua lineNumbers title="Format a quoted string with escaped characters"
local quoted = string.format("%q", "hello\nworld")

engine.print("info", quoted) --'"hello\nworld"'
```

```lua lineNumbers title="Combine multiple format specifiers"
local info = string.format("Name: %s, Age: %d, Score: %.1f%%", "Alice", 25, 95.5)

engine.print("info", info) --'Name: Alice, Age: 25, Score: 95.5%'
```

```lua lineNumbers title="Format a number as octal"
local oct = string.format("%o", 8)

engine.print("info", oct) --'10'
```

```lua lineNumbers title="Include a literal percent sign"
local percent = string.format("Progress: 100%%")

engine.print("info", percent) --'Progress: 100%'
```
