---
title: string.parse
description: Converts a string to its appropriate primitive type
badge: Shared
---

### Syntax

```lua
local result = string.parse(
    input
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `any` | input | Input to parse and convert to its appropriate type |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `any` | result | Converted value:<br/>• `nil` - if input is "nil"<br/>• `bool` - if input is "true" or "false"<br/>• `number` - if input represents a valid number<br/>• `default` - if input cannot be converted to other types |

---

### Examples
```lua lineNumbers title="Parse a nil string"
local result = string.parse("nil")

core.engine.print("info", result) -- nil
```

```lua lineNumbers title="Parse a boolean true string"
local result = string.parse("true")

core.engine.print("info", result) -- true
```

```lua lineNumbers title="Parse a boolean false string"
local result = string.parse("false")

core.engine.print("info", result) -- false
```

```lua lineNumbers title="Parse an integer string"
local result = string.parse("42")

core.engine.print("info", result) -- 42
```

```lua lineNumbers title="Parse a float string"
local result = string.parse("3.14")

core.engine.print("info", result) -- 3.14
```

```lua lineNumbers title="Parse a negative number string"
local result = string.parse("-10")

core.engine.print("info", result) -- -10
```

```lua lineNumbers title="Return unchanged string when no conversion applies"
local result = string.parse("hello")

core.engine.print("info", result) -- 'hello'
```

```lua lineNumbers title="Return false when input is nil"
local result = string.parse(nil)

core.engine.print("info", result) -- false
```
