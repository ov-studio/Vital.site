---
title: string.pack
description: Packs values into a binary string according to a format
badge: Shared
---

### Syntax

```lua
local binary = string.pack(format, ...)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | format | Format string containing specifiers describing how to pack the values<br/>Refer [**Specifiers**](#specifiers) section |
| `...` | ... | Values to pack |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` | binary | Binary string containing the packed values |

<hr/>

### Specifiers

| Specifier | Description |
|--------|-------------|
| `b` | Signed byte (char) |
| `B` | Unsigned byte (unsigned char) |
| `h` | Signed short |
| `H` | Unsigned short |
| `l` | Signed long |
| `L` | Unsigned long |
| `j` | lua_Integer |
| `J` | lua_Unsigned |
| `T` | size_t |
| `i[n]` | Signed int with n bytes |
| `I[n]` | Unsigned int with n bytes |
| `f` | Float |
| `d` | Double |
| `n` | Lua number |
| `c[n]` | Fixed-size string of n bytes |
| `z` | Zero-terminated string |
| `s[n]` | String preceded by length |
| `x` | One byte of padding |
| `<` | Set little endian |
| `>` | Set big endian |
| `=` | Set native endian |

<hr/>

### Examples

```lua lineNumbers title="Pack three integers into a binary string"
local packed = string.pack("iii", 10, 20, 30)

engine.print("info", #packed) --12
```

```lua lineNumbers title="Pack a fixed-size string"
local packed = string.pack("c5", "hello")

engine.print("info", #packed) --5
```

```lua lineNumbers title="Pack a zero-terminated string"
local packed = string.pack("z", "world")

engine.print("info", #packed) --6
```

```lua lineNumbers title="Pack mixed types into one binary string"
local packed = string.pack("ifd", 42, 3.14, 2.718)
```

```lua lineNumbers title="Pack a value in little endian byte order"
local packed = string.pack("<I4", 0x12345678)
```

```lua lineNumbers title="Pack a value in big endian byte order"
local packed = string.pack(">I4", 0x12345678)
```

```lua lineNumbers title="Pack raw bytes as a string"
local packed = string.pack("BBB", 65, 66, 67)

engine.print("info", packed) --'ABC'
```

```lua lineNumbers title="Pack values with padding bytes"
local packed = string.pack("ixxi", 1, 2)
```

```lua lineNumbers title="Pack an unsigned short"
local packed = string.pack("H", 1000)
```

```lua lineNumbers title="Pack a double precision float"
local packed = string.pack("d", math.pi)
```

```lua lineNumbers title="Pack a string with a length prefix"
local packed = string.pack("s1", "hello")
```

```lua lineNumbers title="Pack multiple zero-terminated strings"
local packed = string.pack("zz", "first", "second")
```

```lua lineNumbers title="Combine different types in one pack call"
local packed = string.pack("i4f", 100, 1.5)
```

```lua lineNumbers title="Pack a value in network byte order"
local packed = string.pack(">I2", 0xABCD)
```

```lua lineNumbers title="Pack struct-like player data"
local packed = string.pack("c10i4f", "PlayerName", 100, 99.5)
```

```lua lineNumbers title="Pack a boolean as a byte"
local packed = string.pack("b", (true and 1) or 0)
```
