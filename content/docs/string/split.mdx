---
title: util.string.split
description: Splits a string into a table of substrings using a separator
badge: Shared
---

### Syntax

```lua
local parts = util.string.split(
    input,
    separator
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | input | String to be splitted |
| `string` | separator | Delimiter string to split by |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `table` | parts | Array of split substrings |

---

### Examples
```lua lineNumbers title="Split a string by comma"
local parts = util.string.split("apple,banana,orange", ",")

core.engine.print("info", parts[1]) -- 'apple'
core.engine.print("info", parts[2]) -- 'banana'
core.engine.print("info", parts[3]) -- 'orange'
```

```lua lineNumbers title="Split a string by a multi-character delimiter"
local parts = util.string.split("one::two::three", "::")

core.engine.print("info", parts[1]) -- 'one'
core.engine.print("info", parts[2]) -- 'two'
core.engine.print("info", parts[3]) -- 'three'
```

```lua lineNumbers title="Split a string by space"
local words = util.string.split("hello world lua", " ")

for i, word in ipairs(words) do
    core.engine.print("info", word)
end

--[[
Output:
'hello'
'world'
'lua'
]]
```
