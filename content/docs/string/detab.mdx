---
title: string.detab
description: Converts tab characters to spaces in a string
badge: Shared
---

### Syntax
```lua
local result = string.detab(input)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | input | String containing tab characters |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` | result | Detabbed string |

<hr/>

### Examples
```lua lineNumbers title="Convert a single tab to spaces"
local result = string.detab("Hello\tWorld")

engine.print("info", result) --"Hello    World"
```

```lua lineNumbers title="Convert multiple tabs to spaces"
local result = string.detab("A\tB\tC")

engine.print("info", result) --"A    B    C"
```

```lua lineNumbers title="Format tab-separated values"
local tsv = "Name\tAge\tCity"
local formatted = string.detab(tsv)

engine.print("info", formatted) --"Name    Age    City"
```
