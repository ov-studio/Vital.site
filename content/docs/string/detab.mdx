---
title: util.string.detab
description: Converts tab characters to spaces in a string
badge: Shared
---

### Syntax

```lua
local result = util.string.detab(
    input
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | input | String containing tab characters |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` | result | Detabbed string |

---

### Examples
```lua lineNumbers title="Convert a single tab to spaces"
local result = util.string.detab("Hello\tWorld")

core.engine.print("info", result) -- 'Hello    World'
```

```lua lineNumbers title="Convert multiple tabs to spaces"
local result = util.string.detab("A\tB\tC")

core.engine.print("info", result) -- 'A    B    C'
```

```lua lineNumbers title="Format tab-separated values"
local tsv = "Name\tAge\tCity"
local formatted = util.string.detab(tsv)

core.engine.print("info", formatted) -- 'Name    Age    City'
```
