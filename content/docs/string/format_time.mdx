---
title: string.format_time
description: Formats milliseconds into HH:MM:SS time format
badge: Shared
---

### Syntax
```lua
local result = string.format_time(
    milliseconds
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `int` | milliseconds | Time in milliseconds |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` | result | Formatted string in `HH:MM:SS` format |

<hr/>

### Examples
```lua lineNumbers title="Format 1 hour into HH:MM:SS"
local time = string.format_time(3600000)

engine.print("info", time) --'01:00:00'
```

```lua lineNumbers title="Format 1 minute 30 seconds into HH:MM:SS"
local time = string.format_time(90000)

engine.print("info", time) --'00:01:30'
```

```lua lineNumbers title="Format 30 seconds into HH:MM:SS"
local time = string.format_time(30000)

engine.print("info", time) --'00:00:30'
```
