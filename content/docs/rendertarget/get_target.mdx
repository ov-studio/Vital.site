---
title: rendertarget.get_target
description: Retrieves the currently active rendertarget
badge: Client
---

### Syntax

```lua
local result = rendertarget.get_target()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `rendertarget` / `bool` | result | Currently active rendertarget instance, or `false` if none is set |

<hr/>

### Examples

```lua
local rt = rendertarget.get_target()

if rt then
    engine.print("info", "Active target size:", rt:get_size())
end
```
