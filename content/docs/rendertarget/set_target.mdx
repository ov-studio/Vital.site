---
title: rendertarget.set_target
description: Sets the active rendertarget
badge: Client
---

### Syntax

```lua
local status = rendertarget.set_target(instance = false, clear = false, instant = false)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `rendertarget` | instance | Rendertarget instance to set as active<br/>Omitting this argument will clear the currently active target |
| `bool` | clear | Whether to clear the rendertarget |
| `bool` | instant | Whether to clear immediately |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
local rt = rendertarget.create(1920, 1080, false)

--Bind the rendertarget with a clear
rendertarget.set_target(rt, true)

--Unset the active rendertarget
rendertarget.set_target()
```
