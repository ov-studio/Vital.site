---
title: self:get_size
description: Retrieves the size of the rendertarget
badge: Client
---

### Syntax

```lua
local size = self:get_size()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `vector2` | size | Size of the rendertarget as `{width, height}` |

---

### Examples

```lua lineNumbers title="Get the size of a rendertarget"
local entity = core.rendertarget.create(1920, 1080, false)
local size = entity:get_size()

core.engine.print("info", "Rendertarget size:", util.table.unpack(size))
```
