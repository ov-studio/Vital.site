---
title: self:get_size
description: Retrieves the size of the rendertarget
badge: Client
---

### Syntax

```lua
local x, y = self:get_size()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `int` | x | Width of the rendertarget in pixels |
| `int` | y | Height of the rendertarget in pixels |

<hr/>

### Examples

```lua lineNumbers title="Get the size of a rendertarget"
local rt = rendertarget.create(1920, 1080, false)

engine.print("info", "Rendertarget size:", rt:get_size())
```
