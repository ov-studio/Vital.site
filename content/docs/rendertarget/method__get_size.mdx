---
title: self:get_size
description: Retrieves the size of the rendertarget
badge: Client
---

### Syntax

```lua
local x, y = self:get_size()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `vector2` | size | Size of the rendertarget as `{width, height}` |

<hr/>

### Examples

```lua lineNumbers title="Get the size of a rendertarget"
local rt = rendertarget.create(1920, 1080, false)
local size = rt:get_size()

engine.print("info", "Rendertarget size:", size[1], size[2])
```
