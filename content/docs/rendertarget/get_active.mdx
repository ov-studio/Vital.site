---
title: rendertarget.get_active
description: Retrieves the currently active rendertarget
badge: Client
---

### Syntax

```lua
local result = rendertarget.get_active()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `rendertarget` \| `bool` | result | Currently active rendertarget instance, `false` if none is set |

---

### Examples

```lua lineNumbers title="Get the currently active rendertarget"
local entity = rendertarget.get_active()

if entity then
    engine.print("info", "Active rendertarget size:", entity:get_size())
end
```
