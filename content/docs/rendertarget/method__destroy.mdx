---
title: self:destroy
description: Destroys the rendertarget instance and frees its resources
badge: Client
---

### Syntax

```lua
local status = self:destroy()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Destroy a rendertarget instance"
local rt = rendertarget.create(1920, 1080, false)

rt:destroy()
```
