---
title: ssao.get_intensity
description: Retrieves the current intensity of the SSAO post-processing effect
badge: Client
---

### Syntax

```lua
local result = ssao.get_intensity()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current intensity value |

<hr/>

### Examples

```lua lineNumbers title="Get the current ssao intensity"
engine.print("info", "SSAO intensity:", ssao.get_intensity())
```
