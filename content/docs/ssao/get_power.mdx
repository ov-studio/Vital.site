---
title: ssao.get_power
description: Retrieves the current power curve of the SSAO post-processing effect
badge: Client
---

### Syntax

```lua
local result = ssao.get_power()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current power value |

<hr/>

### Examples

```lua lineNumbers title="Get the current ssao power value"
engine.print("info", "SSAO power:", ssao.get_power())
```
