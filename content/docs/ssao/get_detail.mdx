---
title: ssao.get_detail
description: Retrieves the current detail level of the SSAO post-processing effect
badge: Client
---

### Syntax

```lua
local result = ssao.get_detail()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current detail value |

<hr/>

### Examples

```lua lineNumbers title="Get the current ssao detail level"
engine.print("info", "SSAO detail:", ssao.get_detail())
```
