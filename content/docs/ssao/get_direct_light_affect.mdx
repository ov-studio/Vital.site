---
title: ssao.get_direct_light_affect
description: Retrieves the current direct lighting influence of the SSAO post-processing effect
badge: Client
---

### Syntax

```lua
local result = ssao.get_direct_light_affect()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current direct light influence value |

<hr/>

### Examples

```lua lineNumbers title="Get the current ssao direct light affect"
engine.print("info", "SSAO direct light affect:", ssao.get_direct_light_affect())
```
