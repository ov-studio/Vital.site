---
title: ssao.set_detail
description: Sets the detail level of the SSAO post-processing effect
badge: Client
---

### Syntax

```lua
local status = ssao.set_detail(value)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Detail value to be applied<br/>Higher values capture finer geometric details in the occlusion pass |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Maximize occlusion detail"
ssao.set_detail(1.0)
```
