---
title: ssao.get_horizon
description: Retrieves the current horizon threshold of the SSAO post-processing effect
badge: Client
---

### Syntax

```lua
local result = ssao.get_horizon()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current horizon value |

<hr/>

### Examples

```lua lineNumbers title="Get the current ssao horizon value"
engine.print("info", "SSAO horizon:", ssao.get_horizon())
```
