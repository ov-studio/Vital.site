---
title: ssao.get_radius
description: Retrieves the current sampling radius of the SSAO post-processing effect
badge: Client
---

### Syntax

```lua
local result = ssao.get_radius()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current radius value |

<hr/>

### Examples

```lua lineNumbers title="Get the current ssao sampling radius"
engine.print("info", "SSAO radius:", ssao.get_radius())
```
