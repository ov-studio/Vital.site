---
title: ssao.get_channel_affect
description: Retrieves the current AO channel influence of the SSAO post-processing effect
badge: Client
---

### Syntax

```lua
local result = ssao.get_channel_affect()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current channel influence value |

<hr/>

### Examples

```lua lineNumbers title="Get the current ssao channel affect"
engine.print("info", "SSAO channel affect:", ssao.get_channel_affect())
```
