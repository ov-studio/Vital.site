---
title: gfx.sdfgi.set_use_occlusion
description: Toggles occlusion for the SDFGI post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.sdfgi.set_use_occlusion(state)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `bool` | state | Determines whether occlusion is used:<br/>• When `true` - occlusion is enabled, preventing light from passing through solid geometry<br/>• When `false` - occlusion is disabled |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
--Enable occlusion for more accurate GI shadowing
gfx.sdfgi.set_use_occlusion(true)
```
