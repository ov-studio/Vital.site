### Syntax

```lua
local result = gfx.sdfgi.get_max_distance()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current maximum distance |

<hr/>

### Examples

```lua
gfx.print("SDFGI max distance:", gfx.sdfgi.get_max_distance())
```
