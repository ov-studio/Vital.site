
### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current depth tolerance value |

### Examples

```lua
gfx.print("SSR depth tolerance:", gfx.ssr.get_depth_tolerance())
```
