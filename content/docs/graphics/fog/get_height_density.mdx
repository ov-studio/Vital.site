```lua
local result = gfx.fog.get_height_density()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current height density value |

<hr/>

### Examples

```lua
gfx.print("Fog height density:", gfx.fog.get_height_density())
```
