---
title: gfx.fog.set_light_color
description: The light color to be set for the fog post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.fog.set_light_color(r, g, b, a)
local status = gfx.fog.set_light_color(hex)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | r | Red channel value of the fog light color |
| `float` | g | Green channel value of the fog light color |
| `float` | b | Blue channel value of the fog light color |
| `float` | a | Alpha channel value of the fog light color |
| `string` | hex | Alternatively, a valid HTML hex color string (e.g. `"#ffffff"`) |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
--Set fog light color using RGBA
gfx.fog.set_light_color(0.5, 0.6, 0.7, 1.0)
```

```lua
--Set fog light color using a hex string
gfx.fog.set_light_color("#8fa3b1")
```
