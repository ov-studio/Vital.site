---
title: gfx.volumetric_fog.set_emission
description: The emission color to be set for the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.volumetric_fog.set_emission(r, g, b, a)
local status = gfx.volumetric_fog.set_emission(hex)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | r | Red channel value of the emission color |
| `float` | g | Green channel value of the emission color |
| `float` | b | Blue channel value of the emission color |
| `float` | a | Alpha channel value of the emission color |
| `string` | hex | Alternatively, a valid HTML hex color string (e.g. `"#ffffff"`) |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
--Set a warm orange emission color using RGBA
gfx.volumetric_fog.set_emission(1.0, 0.4, 0.1, 1.0)
```

```lua
--Set emission color using a hex string
gfx.volumetric_fog.set_emission("#ff6619")
```
