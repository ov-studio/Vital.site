---
title: gfx.adjustment.get_lut
description: Retrieves the path of the currently active LUT on the adjustment post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.adjustment.get_lut()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` \| `bool` | result | Path of the active LUT, `false` if no LUT is currently applied |

---

### Examples

```lua lineNumbers title="Retrieve currently active LUT"
local lut = gfx.adjustment.get_lut()

if lut then
    core.engine.print("info", "Active LUT:", lut)
else
    core.engine.print("info", "No LUT is currently applied")
end
```
