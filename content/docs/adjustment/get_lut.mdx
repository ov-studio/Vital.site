---
title: adjustment.get_lut
description: Retrieves the path of the currently active LUT on the adjustment post-processing effect
badge: Client
---

### Syntax

```lua
local result = adjustment.get_lut()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` / `bool` | result | Path of the active LUT, or `false` if no LUT is currently applied |

<hr/>

### Examples

```lua lineNumbers title="Retrieve currently active LUT"
local lut = adjustment.get_lut()

if lut then
    engine.print("info", "Active LUT:", lut)
else
    engine.print("info", "No LUT is currently applied")
end
```
