---
title: adjustment.get_contrast
description: Retrieves the current contrast level of the adjustment post-processing effect
badge: Client
---

### Syntax

```lua
local result = adjustment.get_contrast()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current contrast value |

<hr/>

### Examples

```lua lineNumbers title="Print current contrast level"
engine.print("info", "Current contrast:", adjustment.get_contrast())
```
