---
title: adjustment.reset_lut
description: Removes the currently active LUT on the adjustment post-processing effect
badge: Client
---

### Syntax

```lua
local status = adjustment.reset_lut()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Remove the currently active LUT"
adjustment.reset_lut()
```
