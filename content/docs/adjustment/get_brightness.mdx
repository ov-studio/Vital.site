---
title: gfx.adjustment.get_brightness
description: Retrieves the current brightness level of the adjustment post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.adjustment.get_brightness()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current brightness value |

---

### Examples

```lua lineNumbers title="Print current brightness level"
core.engine.print("info", "Current brightness:", gfx.adjustment.get_brightness())
```
