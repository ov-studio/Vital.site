---
title: adjustment.is_enabled
description: Retrieves whether the adjustment post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local result = adjustment.is_enabled()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if adjustment is enabled, `false` otherwise |

<hr/>

### Examples

```lua lineNumbers title="Check if adjustment is enabled"
engine.print("info", "Adjustment enabled:", adjustment.is_enabled())
```
