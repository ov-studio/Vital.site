---
title: gfx.adjustment.get_saturation
description: Retrieves the current saturation level of the adjustment post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.adjustment.get_saturation()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current saturation value |

---

### Examples

```lua lineNumbers title="Print current saturation level"
core.engine.print("info", "Current saturation:", gfx.adjustment.get_saturation())
```
