---
title: fog.set_light_color
description: The light color to be set for the fog post-processing effect
badge: Client
---

### Syntax

```lua
local status = fog.set_light_color(
    color
)

local status = fog.set_light_color(
    hex
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `color` | color | Light color of the fog as `{r, g, b, a}` |
| `string` | hex | Alternatively, a valid HTML hex color string (e.g. `"#ffffff"`) |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Set fog light color using a color table"
fog.set_light_color({0.5, 0.6, 0.7, 1.0})
```

```lua lineNumbers title="Set fog light color using a hex string"
fog.set_light_color("#8fa3b1")
```
