---
title: fog.get_mode
description: Retrieves the current fog mode of the fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = fog.get_mode()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `int` | result | Current fog mode |

<hr/>

### Examples

```lua lineNumbers title="Print current fog mode"
engine.print("info", "Fog mode:", fog.get_mode())
```
