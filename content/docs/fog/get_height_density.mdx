---
title: fog.get_height_density
description: Retrieves the current height density of the fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = fog.get_height_density()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current height density value |

<hr/>

### Examples

```lua lineNumbers title="Print current fog height density"
engine.print("info", "Fog height density:", fog.get_height_density())
```
