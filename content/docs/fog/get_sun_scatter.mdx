---
title: fog.get_sun_scatter
description: Retrieves the current sun scatter amount of the fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = fog.get_sun_scatter()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current sun scatter value |

<hr/>

### Examples

```lua lineNumbers title="Print current fog sun scatter amount"
engine.print("info", "Fog sun scatter:", fog.get_sun_scatter())
```
