---
title: gfx.fog.set_light_energy
description: Sets the light energy of the fog post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.fog.set_light_energy(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Light energy value to apply<br/>Controls the brightness of the fog's light color. Higher values produce a more luminous, glowing fog appearance |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set a bright fog light"
gfx.fog.set_light_energy(2.0)
```
