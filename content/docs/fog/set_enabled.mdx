---
title: fog.set_enabled
description: Toggles the fog post-processing effect
badge: Client
---

### Syntax

```lua
local status = fog.set_enabled(state)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `bool` | state | Determines fog's state:<br/>• When `true` - fog is enabled<br/>• When `false` - fog is disabled |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Enable and disable fog"
--Enable fog
fog.set_enabled(true)

--Disable fog
fog.set_enabled(false)
```
