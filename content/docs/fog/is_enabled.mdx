---
title: fog.is_enabled
description: Retrieves whether the fog post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local result = fog.is_enabled()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if fog is enabled, `false` otherwise |

<hr/>

### Examples

```lua lineNumbers title="Print whether fog is enabled"
engine.print("info", "Fog enabled:", fog.is_enabled())
```
