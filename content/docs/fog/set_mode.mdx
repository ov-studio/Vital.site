---
title: gfx.fog.set_mode
description: The fog mode to be set for the fog post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.fog.set_mode(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `enum` | value | Fog mode to be applied<br/>Refer [**gfx.fog.fog_mode**](/docs/fog/enum/fog_mode) enum |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set exponential and depth-based fog modes"
-- Set exponential fog mode
gfx.fog.set_mode(gfx.fog.fog_mode.EXPONENTIAL)

-- Set depth-based fog mode
gfx.fog.set_mode(gfx.fog.fog_mode.DEPTH)
```
