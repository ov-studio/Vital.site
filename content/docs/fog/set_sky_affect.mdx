---
title: fog.set_sky_affect
description: The sky influence amount to be set for the fog post-processing effect
badge: Client
---

### Syntax

```lua
local status = fog.set_sky_affect(value)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Sky influence value to be applied<br/>Controls how much the fog blends into and affects the appearance of the sky. A value of `0` leaves the sky unaffected while higher values cause the fog to visibly tint and obscure the skybox |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Allow fog to partially tint the sky"
fog.set_sky_affect(0.5)
```
