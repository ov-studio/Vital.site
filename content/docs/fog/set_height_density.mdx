---
title: gfx.fog.set_height_density
description: Sets the height density of the fog post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.fog.set_height_density(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Height density value to apply<br/>Controls how quickly fog density increases as height decreases. Use a negative value to instead increase fog density as height increases |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Increase fog density toward the ground"
gfx.fog.set_height_density(10.0)
```

```lua lineNumbers title="Increase fog density toward the sky"
gfx.fog.set_height_density(-10.0)
```
