---
title: fog.get_depth_begin
description: Retrieves the current depth begin distance of the fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = fog.get_depth_begin()
```

<Callout type="warn">
  **Only available when the fog mode is set to depth mode.** Refer [`fog.set_mode`](/docs/fog/set_mode) to configure the active fog mode.

  - **Depth mode required** — calling this function in any other fog mode will return an unexpected result
  - **Recommended usage** — use alongside [`fog.get_depth_end`](/docs/fog/get_depth_end) to inspect the full fog transition range
</Callout>

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current depth begin value |

<hr/>

### Examples

```lua lineNumbers title="Print current fog depth begin distance"
engine.print("info", "Fog depth begin:", fog.get_depth_begin())
```
