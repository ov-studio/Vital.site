---
title: fog.get_light_color
description: Retrieves the current light color of the fog post-processing effect
badge: Client
---

### Syntax

```lua
local color = fog.get_light_color()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `color` | color | Light color of the fog as `{r, g, b, a}` |

<hr/>

### Examples

```lua lineNumbers title="Print current fog light color"
engine.print("info", "Fog light color:", table.unpack(fog.get_light_color()))
```
