---
title: fog.get_sky_affect
description: Retrieves the current sky influence amount of the fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = fog.get_sky_affect()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current sky influence value |

<hr/>

### Examples

```lua lineNumbers title="Print current fog sky influence amount"
engine.print("info", "Fog sky affect:", fog.get_sky_affect())
```
