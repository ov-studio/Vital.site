### Values

| Constant | Description |
|----------|-------------|
| `gfx.fog.fog_mode.EXPONENTIAL` | Fog density increases exponentially with distance from the camera<br/>Refer [`gfx.fog.set_density`](/docs/fog/set_density) |
| `gfx.fog.fog_mode.DEPTH` | Fog is applied across a defined depth range<br/>Refer [`gfx.fog.set_depth_begin`](/docs/fog/set_depth_begin) and [`gfx.fog.set_depth_end`](/docs/fog/set_depth_end) |
