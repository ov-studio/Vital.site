---
title: fog.get_aerial_perspective
description: Retrieves the current aerial perspective blend amount of the fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = fog.get_aerial_perspective()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current aerial perspective value |

<hr/>

### Examples

```lua lineNumbers title="Print current fog aerial perspective"
engine.print("info", "Fog aerial perspective:", fog.get_aerial_perspective())
```
