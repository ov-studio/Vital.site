---
title: engine.get_timestamp
description: Retrieves engine's timestamp
badge: Shared
---

### Syntax

```lua
local timestamp = engine.get_timestamp()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|


### Returns

| Type | Name | Description |
|------|------|-------------|
| `table` | timestamp | Engine's current timestamp containing the following fields:<br/>• `hour` (number) - Current hour<br/>• `minute` (number) - Current minute<br/>• `second` (number) - Current second<br/>• `day` (number) - Current day<br/>• `month` (number) - Current month<br/>• `year` (number) - Current year |

<hr/>

### Example

```lua
engine.print("Current timestamp:")
engine.print(table.inspect(engine.get_timestamp()))
```
