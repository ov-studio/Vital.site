---
title: core.engine.get_timestamp
description: Retrieves engine's current timestamp
badge: Shared
---

### Syntax

```lua
local timestamp = core.engine.get_timestamp()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `table` | timestamp | Engine's current timestamp containing the following fields:<br/>• `hour` (number) - current hour<br/>• `minute` (number) - current minute<br/>• `second` (number) - current second<br/>• `day` (number) - current day<br/>• `month` (number) - current month<br/>• `year` (number) - current year |

---

### Examples

```lua lineNumbers title="Prints engine's current timestamp"
core.engine.print("info", "current timestamp:")
core.engine.iprint(core.engine.get_timestamp())
```
