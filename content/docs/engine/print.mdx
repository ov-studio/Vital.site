---
title: engine.print
description: Prints info on engine's console
badge: Shared
---

### Syntax

```lua
local status = engine.print(...)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `...` | ... | Values to print on engine's console |


### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | Returns `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
engine.print("Hello", "World", 123)
--Output: Hello World 123
```
