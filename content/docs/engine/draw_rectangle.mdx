---
title: core.engine.draw_rectangle
description: Draws a filled rectangle on the canvas
badge: Client
---

### Syntax

```lua
local status = core.engine.draw_rectangle(
    position, 
    size, 
    color = {1, 1, 1, 1}, 
    stroke = 0, 
    stroke_color = {1, 1, 1, 1}, 
    rotation = 0, 
    pivot = {0, 0}
)
```

<Callout type="warn">
  **Must be called within the `"sandbox:draw"` util.event.** Invoking this function outside of that event will have no effect.

  - **Inside the event** — executes as expected, rendering the filled rectangle to the canvas each frame
  - **Outside the event** — the call will be silently ignored and nothing will be drawn
  - **Recommended usage** — register a handler via `util.event.on("sandbox:draw", ...)` and place all draw calls inside it
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `vector2` | position | Top-left corner of the rectangle |
| `vector2` | size | Width and height of the rectangle |
| `color` | color | Fill color of the rectangle |
| `float` | stroke | Outline thickness in pixels<br/>Set to `0` to disable |
| `color` | stroke_color | Color of the outline |
| `float` | rotation | Rotation angle in degrees |
| `vector2` | pivot | Pivot point for rotation, relative to the rectangle's position |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Draw a plain white rectangle"
local resolution = core.engine.get_resolution()
local center = {resolution[1] * 0.5, resolution[2] * 0.5}
local size = {200, 100}

util.event.on("sandbox:draw", function()
    core.engine.draw_rectangle(
        {center[1] - size[1] * 0.5, center[2] - 200},
        size
    )
end)
```

```lua lineNumbers title="Draw a filled rectangle with an outline"
local resolution = core.engine.get_resolution()
local center = {resolution[1] * 0.5, resolution[2] * 0.5}
local size = {200, 100}

util.event.on("sandbox:draw", function()
    core.engine.draw_rectangle(
        {center[1] - size[1] * 0.5, center[2] - size[2] * 0.5},
        size,
        {0, 1, 0, 1},
        2,
        {0, 0, 0, 1}
    )
end)
```

```lua lineNumbers title="Draw a rectangle rotated around its center"
local resolution = core.engine.get_resolution()
local center = {resolution[1] * 0.5, resolution[2] * 0.5}
local size = {200, 100}

util.event.on("sandbox:draw", function()
    core.engine.draw_rectangle(
        {center[1] - size[1] * 0.5, center[2] + 120},
        size,
        {1, 0.5, 0, 1},
        0,
        {1, 1, 1, 1},
        30,
        {size[1] * 0.5, size[2] * 0.5}
    )
end)
```
