---
title: engine.fog.set_mode
description: The fog mode to be set for the fog post-processing effect
badge: Client
---

### Syntax

```lua
local status = engine.fog.set_mode(value)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `int` | value | Fog mode to be applied. Determines how fog density is calculated across the scene:<br/>• `0` - Exponential mode, fog density increases exponentially with distance from the camera<br/>• `1` - Depth mode, fog is applied based on a defined depth range with configurable start and end points |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | Returns `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
--Set exponential fog mode
engine.fog.set_mode(0)

--Set depth-based fog mode
engine.fog.set_mode(1)
```
