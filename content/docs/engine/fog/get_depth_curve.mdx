---
title: engine.fog.get_depth_curve
description: Retrieves the current depth curve of the fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = engine.fog.get_depth_curve()
```

<Callout type="warn">
Only available when the fog mode is set to depth mode. See [`engine.fog.set_mode`](/docs/engine/fog/set_mode) for more information.
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current depth curve value |

<hr/>

### Examples

```lua
engine.print("Fog depth curve:", engine.fog.get_depth_curve())
```
