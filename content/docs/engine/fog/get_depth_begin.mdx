```lua

```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Returns the current depth begin value |

<hr/>

### Examples

```lua
engine.print("Fog depth begin:", engine.fog.get_depth_begin())
```
