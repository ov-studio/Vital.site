---
title: engine.fog.get_sun_scatter

badge: Client
---

### Syntax

```lua
local result = engine.fog.get_sun_scatter()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Returns the current sun scatter value |

<hr/>

### Examples

```lua
engine.print("Fog sun scatter:", engine.fog.get_sun_scatter())
```
