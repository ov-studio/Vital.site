---
title: engine.fog.get_aerial_perspective
description: Retrieves the current aerial perspective blend amount of the fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = engine.fog.get_aerial_perspective()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Returns the current aerial perspective value |

<hr/>

### Examples

```lua
engine.print("Fog aerial perspective:", engine.fog.get_aerial_perspective())
```
