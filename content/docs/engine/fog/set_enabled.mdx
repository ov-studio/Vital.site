---
title: engine.fog.set_enabled
description: Toggles the fog post-processing effect
badge: Client
---

### Syntax

```lua
local status = engine.fog.set_enabled(state)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `bool` | state | Determines fog's state:<br/>• When `true` - fog is enabled<br/>• When `false` - fog is disabled |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | Returns `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
--Enable fog
engine.fog.set_enabled(true)

--Disable fog
engine.fog.set_enabled(false)
```
