---
title: engine.fog.get_depth_end
description: Retrieves the current depth end distance of the fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = engine.fog.get_depth_end()
```

<Callout type="warn">
Only available when the fog mode is set to depth mode. See [`engine.fog.set_mode`](/docs/engine/fog/set_mode) for more information.
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current depth end value |

<hr/>

### Examples

```lua
engine.print("Fog depth end:", engine.fog.get_depth_end())
```
