---
title: engine.fog.set_depth_begin
description: The depth begin distance to be set for the fog post-processing effect
badge: Client
---

### Syntax

```lua
local status = engine.fog.set_depth_begin(value)
```

<Callout type="warn">
Only available when the fog mode is set to depth mode. See [`engine.fog.set_mode`](/docs/engine/fog/set_mode) for more information.
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Depth begin value to be applied<br/>Sets the distance from the camera at which depth-based fog starts to appear. Geometry closer than this distance will not be affected by fog |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | Returns `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
--Start fog at 10 units from the camera
engine.fog.set_depth_begin(10.0)
```
