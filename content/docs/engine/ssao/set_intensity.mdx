---
title: engine.ssao.set_intensity
description: Sets the intensity of the SSAO post-processing effect
badge: Client
---

### Syntax

```lua
local status = engine.ssao.set_intensity(value)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | The intensity value to apply.<br/>Higher values produce darker and more pronounced occlusion shadows |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | Returns `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
--Set a strong occlusion intensity
engine.ssao.set_intensity(3.0)
```
