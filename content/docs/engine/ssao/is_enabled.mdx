---
title: engine.ssao.is_enabled
description: Retrieves whether the SSAO post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local result = engine.ssao.is_enabled()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if SSAO is enabled, `false` otherwise |

<hr/>

### Examples

```lua
engine.print("SSAO enabled:", engine.ssao.is_enabled())
```
