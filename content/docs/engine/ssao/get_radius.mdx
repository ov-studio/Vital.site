```lua

```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current radius value |

### Examples

engine.print("SSAO radius:", engine.ssao.get_radius())
