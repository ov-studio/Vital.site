---
title: engine.ssao.get_direct_light_affect

badge: Client
---

### Syntax

```lua
local result = engine.ssao.get_direct_light_affect()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current direct light influence value |

<hr/>

### Examples

```lua
engine.print("SSAO direct light affect:", engine.ssao.get_direct_light_affect())
```
