---
title: engine.ssao.set_enabled
description: Toggles the SSAO post-processing effect
badge: Client
---

### Syntax

```lua
local status = engine.ssao.set_enabled(state)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `bool` | state | Determines SSAO's state:<br/>• When `true` - SSAO is enabled<br/>• When `false` - SSAO is disabled |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | Returns `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
--Enable SSAO
engine.ssao.set_enabled(true)

--Disable SSAO
engine.ssao.set_enabled(false)
```
