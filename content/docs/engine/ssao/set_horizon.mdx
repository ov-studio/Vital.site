---
title: engine.ssao.set_horizon
description: Sets the horizon threshold of the SSAO post-processing effect
badge: Client
---

### Syntax

```lua
local status = engine.ssao.set_horizon(value)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Horizon value to be applied.<br/>Controls how much occlusion is applied to surfaces facing away from the camera, reducing halo artifacts on silhouettes |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | Returns `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
-- Reduce silhouette halos
engine.ssao.set_horizon(0.06)
```
