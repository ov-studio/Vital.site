---
title: engine.draw_line
description: Draws a polyline through a series of 2D points on the canvas
badge: Client
---

### Syntax

```lua
local status = engine.draw_line(
    points, 
    stroke = 0, 
    color = {1, 1, 1, 1}
)
```

<Callout type="warn">
  **Must be called within the `"sandbox:draw"` event.** Invoking this function outside of that event will have no effect.

  - **Inside the event** — executes as expected, rendering the polyline to the canvas each frame
  - **Outside the event** — the call will be silently ignored and nothing will be drawn
  - **Recommended usage** — register a handler via `event.on("sandbox:draw", ...)` and place all canvas draw calls inside it
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `vector2[]` | points | Array of 2D points defining the line path |
| `float` | stroke | Line thickness in pixels — set to `0` to disable |
| `color` | color | Color of the line |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Draw lines with various styles"
local resolution = engine.get_resolution()
local center = {resolution[1] * 0.5, resolution[2] * 0.5}

event.on("sandbox:draw", function()
    -- Draw a simple horizontal line across the center of the screen
    engine.draw_line(
        {
            {center[1] - 200, center[2] - 100}, 
            {center[1] + 200, center[2] - 100}
        },
        2
    )

    -- Draw a thick red diagonal line below it
    engine.draw_line(
        {
            {center[1] - 100, center[2]}, 
            {center[1] + 100, center[2] + 100}
        },
        4,
        {1, 0, 0, 1}
    )

    -- Draw a green multi-segment polyline at the bottom section
    engine.draw_line(
        {
            {center[1] - 150, center[2] + 150},
            {center[1] - 50,  center[2] + 80},
            {center[1] + 50,  center[2] + 150},
            {center[1] + 150, center[2] + 80}
        },
        2,
        {0, 1, 0, 1}
    )
end)
```
