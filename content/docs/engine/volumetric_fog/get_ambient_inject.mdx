---
title: engine.volumetric_fog.get_ambient_inject

---

### Syntax

```lua
local result = engine.volumetric_fog.get_ambient_inject()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current ambient inject value |

<hr/>

### Examples

```lua
engine.print("Volumetric fog ambient inject:", engine.volumetric_fog.get_ambient_inject())
```
