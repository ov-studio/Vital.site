---
title: engine.volumetric_fog.is_enabled
description: Retrieves whether the volumetric fog post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local result = engine.volumetric_fog.is_enabled()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | Returns `true` if volumetric fog is enabled, `false` otherwise |

<hr/>

### Examples

```lua
engine.print("Volumetric fog enabled:", engine.volumetric_fog.is_enabled())
```
