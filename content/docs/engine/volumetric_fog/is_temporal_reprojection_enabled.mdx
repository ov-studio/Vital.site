---
title: engine.volumetric_fog.is_temporal_reprojection_enabled
description: Retrieves whether temporal reprojection is currently enabled for the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = engine.volumetric_fog.is_temporal_reprojection_enabled()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if temporal reprojection is enabled, `false` otherwise |

<hr/>

### Examples

```lua
engine.print("Temporal reprojection enabled:", engine.volumetric_fog.is_temporal_reprojection_enabled())
```
