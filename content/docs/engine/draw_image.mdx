---
title: engine.draw_image
description: Draws an image on the canvas from a file path or a texture resource
badge: Client
---

### Syntax

```lua
local status = engine.draw_image(
    position, 
    size, 
    texture, 
    rotation = 0, 
    pivot = {0, 0}, 
    color = {1, 1, 1, 1}
)
```

<Callout type="warn">
  **Must be called within the `"sandbox:draw"` event.** Invoking this function outside of that event will have no effect.

  - **Inside the event** — executes as expected, rendering the image to the canvas each frame
  - **Outside the event** — the call will be silently ignored and nothing will be drawn
  - **Recommended usage** — register a handler via `event.on("sandbox:draw", ...)` and place all canvas draw calls inside it
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `vector2` | position | Top-left position to draw the image |
| `vector2` | size | Width and height to render the image at |
| `string` \| `texture` \| `svg` \| `rendertarget` | texture | Image source:<br/>• `string` — file path to an image asset<br/>• `texture` — pre-loaded texture instance<br/>• `rendertarget` — rendertarget instance<br/>• `svg` — svg instance |
| `float` | rotation | Rotation angle in degrees |
| `vector2` | pivot | Pivot point for rotation, relative to the image's position |
| `color` | color | Tint color to multiply over the image |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Draw images from various sources"
local resolution = engine.get_resolution()
local center = {resolution[1] * 0.5, resolution[2] * 0.5}
local my_texture = texture.create("assets/logo.png")
local my_svg = svg.create("assets/icon.svg")
local my_rt = rendertarget.create(1920, 1080)

event.on("sandbox:draw", function()
    -- Draw an image from a file path at the top-left region
    engine.draw_image(
        {center[1] - 400, center[2] - 300},
        {256, 256},
        "assets/logo.png"
    )

    -- Draw a tinted image next to it
    engine.draw_image(
        {center[1] + 150, center[2] - 300},
        {256, 256},
        "assets/icon.png",
        0,
        {0, 0},
        {1, 0.5, 0.5, 1}
    )

    -- Draw an image rotated 90 degrees around its center
    engine.draw_image(
        {center[1] - 64, center[2] - 64},
        {128, 128},
        "assets/arrow.png",
        90,
        {64, 64}
    )

    -- Draw from a texture instance below the previous row
    engine.draw_image(
        {center[1] - 400, center[2] + 50},
        {256, 256},
        my_texture
    )

    -- Draw from an svg instance next to it
    engine.draw_image(
        {center[1] - 100, center[2] + 50},
        {256, 256},
        my_svg
    )

    -- Draw from a rendertarget instance spanning the lower portion
    engine.draw_image(
        {center[1] - 200, center[2] + 320},
        {400, 225},
        my_rt
    )
end)
```
