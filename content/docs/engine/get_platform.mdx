---
title: engine.get_platform
description: Retrieves sandbox's platform type
badge: Shared
---

### Syntax

```lua
local platform = engine.get_platform()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|


### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` | platform | Sandbox's current platform type:<br/>• `"Client"` - indicates client-side execution<br/>• `"Server"` - indicates server-side execution |

<hr/>

### Example

```lua
local platform = engine.get_platform()

if platform == "Client" then
    print("Running on client")
elseif platform == "Server" then
    print("Running on server")
end
```
