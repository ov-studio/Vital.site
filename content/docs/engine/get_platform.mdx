---
title: engine.get_platform
description: Retrieves the current platform type
badge: Shared
---

### Syntax

```lua
local platform = engine.get_platform()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|


### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` | platform | <br/>Possible values are: `"Client"` (indicates client-side execution) or `"Server"` (indicates server-side execution) |

<hr/>

### Example

```lua
local platform = engine.get_platform()

if platform == "Client" then
    print("Running on client")
elseif platform == "Server" then
    print("Running on server")
end
```
