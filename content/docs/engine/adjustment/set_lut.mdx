---
title: engine.adjustment.set_lut
description: Applies a specified LUT on the adjustment post-processing effect
badge: Client
---

### Syntax

```lua
local status = engine.adjustment.set_lut(path)
```

<Callout type="info">
LUTs (Look-Up Tables) are used for color grading and correction — such as cinematic tones, warm/cool shifts, or stylized filters.

Supports variable LUT sizes with two layouts: a **square grid** (e.g. 8×8, 16×16, 64×64) or a **strip layout** (single row of N columns).
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | path | Path of the LUT to be applied |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
--Apply a LUT for cinematic color grading
engine.adjustment.set_lut("lut/cinematic.png")
```
