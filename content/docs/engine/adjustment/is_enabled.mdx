---
title: engine.adjustment.is_enabled
description: Retrieves whether the adjustment post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local state = engine.adjustment.is_enabled()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | state | Returns `true` if adjustment is enabled, `false` otherwise |

<hr/>

### Examples

```lua
engine.print("Adjustment enabled:", engine.adjustment.is_enabled())
```
