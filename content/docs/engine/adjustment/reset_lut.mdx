---
title: engine.adjustment.reset_lut
description: Removes the currently active LUT on the adjustment post-processing effect
badge: Client
---

### Syntax

```lua
local status = engine.adjustment.reset_lut()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
--Remove active LUT
engine.adjustment.reset_lut()
```
