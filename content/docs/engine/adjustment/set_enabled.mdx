---
title: engine.adjustment.set_enabled
description: Enables or disables the adjustment post-processing effect
badge: Client
---

### Syntax

```lua
local status = engine.adjustment.set_enabled(state)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `bool` | state | Determines adjustment's state:<br/>• When `true` - the adjustment is enabled<br/>• When `false` - the adjustment is disabled |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | Returns `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
--Enable adjustment
engine.adjustment.set_enabled(true)

--Disable adjustment
engine.adjustment.set_enabled(false)
```
