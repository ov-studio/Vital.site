---
title: engine.adjustment.get_lut
description: Retrieves the path of the currently active LUT texture
badge: Client
---

### Syntax

```lua
local result = engine.adjustment.get_lut()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` / `bool` | result | Returns the path of the active LUT texture, or `false` if no LUT is currently applied |

<hr/>

### Examples

```lua
local lut = engine.adjustment.get_lut()

if lut then
    engine.print("Active LUT:", lut)
else
    engine.print("No LUT applied")
end
```
