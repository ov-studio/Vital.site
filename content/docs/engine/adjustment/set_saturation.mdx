---
title: engine.adjustment.set_saturation
description: Sets the saturation level of the adjustment post-processing effect
badge: Client
---

### Syntax

```lua
local status = engine.adjustment.set_saturation(value)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Saturation value to be applied |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | Returns `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
--Set saturation to default
engine.adjustment.set_saturation(1.0)

--Desaturate (greyscale)
engine.adjustment.set_saturation(0.0)
```
