---
title: engine.get_tick
description: Retrieves the current tickcount
badge: Shared
---

### Syntax

```lua
local tick = engine.get_tick()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|


### Returns

| Type | Name | Description |
|------|------|-------------|
| `int` | tick | Platform's current tickcount |

<hr/>

### Example

```lua
local tickcount = engine.get_tick()
print("Current tickcount", engine.get_tick())
```
