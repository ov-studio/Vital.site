---
title: engine.get_tick
description: Retrieves engine's current tickcount
badge: Shared
---

### Syntax

```lua
local tick = engine.get_tick()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `int` | tick | Engine's current tickcount |

<hr/>

### Examples

```lua lineNumbers title="Prints engine's current tickcount"
engine.print("info", "Current tickcount:", engine.get_tick())
```
