---
title: core.engine.get_tick
description: Retrieves engine's current tickcount
badge: Shared
---

### Syntax

```lua
local tick = core.engine.get_tick()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `int` | tick | Engine's current tickcount |

---

### Examples

```lua lineNumbers title="Prints engine's current tickcount"
core.engine.print("info", "Current tickcount:", core.engine.get_tick())
```
