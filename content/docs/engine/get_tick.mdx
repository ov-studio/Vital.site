---
title: engine.get_tick
description: Retrieves engine's current tickcount
badge: Shared
---

### Syntax

```lua
local tick = engine.get_tick()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|


### Returns

| Type | Name | Description |
|------|------|-------------|
| `int` | tick | Engine's current tickcount |

<hr/>

### Examples

```lua
engine.print("Current tickcount:", engine.get_tick())
```
