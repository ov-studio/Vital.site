---
title: engine.ssil.set_intensity
description: The intensity to be set for the SSIL post-processing effect
badge: Client
---

### Syntax

```lua
local status = engine.ssil.set_intensity(value)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Intensity value to be applied<br/>Higher values produce stronger indirect light contributions, making bounced light more visible across surfaces |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
--Set a strong indirect illumination intensity
engine.ssil.set_intensity(2.0)
```
