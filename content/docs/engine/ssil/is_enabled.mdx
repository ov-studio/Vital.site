---
title: engine.ssil.is_enabled
description: Retrieves whether the SSIL post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local result = engine.ssil.is_enabled()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | Returns `true` if SSIL is enabled, `false` otherwise |

<hr/>

### Examples

```lua
engine.print("SSIL enabled:", engine.ssil.is_enabled())
```
