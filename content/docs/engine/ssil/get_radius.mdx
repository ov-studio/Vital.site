```lua

```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current radius value |

### Examples

engine.print("SSIL radius:", engine.ssil.get_radius())
