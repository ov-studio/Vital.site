---
title: engine.ssil.set_enabled
description: Toggles the SSIL post-processing effect
badge: Client
---

### Syntax

```lua
local status = engine.ssil.set_enabled(state)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `bool` | state | Determines SSIL's state:<br/>• When `true` - SSIL is enabled<br/>• When `false` - SSIL is disabled |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
--Enable SSIL
engine.ssil.set_enabled(true)

--Disable SSIL
engine.ssil.set_enabled(false)
```
