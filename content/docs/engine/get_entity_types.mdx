---
title: core.engine.get_entity_types
description: Retrieves all currently registered entity category names
badge: Shared
---

### Syntax

```lua
local result = core.engine.get_entity_types()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `table` | result | An array of available entity category names<br/>Each entry is usable as the `category` argument to [`core.engine.get_entities`](/docs/engine/get_entities) |

---

### Examples

```lua lineNumbers title="Prints all registered entity categories"
core.engine.iprint(core.engine.get_entity_types())
```

```lua lineNumbers title="Counts entities across every category"
for index, category in ipairs(core.engine.get_entity_types()) do
    core.engine.print("info", category, "count:", #core.engine.get_entities(category))
end
```

```lua lineNumbers title="Registers a monitor for every entity category"
for index, category in ipairs(core.engine.get_entity_types()) do
    util.monitor.register(
        category .. "_entity_count",
        util.string.upper(category) .. "_ENTITY_COUNT",
        function() return #core.engine.get_entities(category) end,
        util.monitor.stat_format.QUANTITY
    )
end
```
