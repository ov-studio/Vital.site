```lua

```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current glow map strength value |

### Examples

```lua
engine.print("Emissive map strength:", engine.emissive.get_map_strength())
```
