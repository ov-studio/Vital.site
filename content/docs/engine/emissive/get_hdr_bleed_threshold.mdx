```lua
local result = engine.emissive.get_hdr_bleed_threshold()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current HDR bleed threshold value |

<hr/>

### Examples

```lua
engine.print("Emissive HDR bleed threshold:", engine.emissive.get_hdr_bleed_threshold())
```
