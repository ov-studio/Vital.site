title: engine.emissive.get_mix

### Syntax

```lua
local result = engine.emissive.get_mix()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current mix value |

<hr/>

### Examples

```lua
engine.print("Emissive mix:", engine.emissive.get_mix())
```
