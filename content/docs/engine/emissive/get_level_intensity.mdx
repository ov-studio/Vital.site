---
title: engine.emissive.get_level_intensity
description: Retrieves the current intensity of a specific glow level of the emissive post-processing effect
badge: Client
---

### Syntax

```lua
local result = engine.emissive.get_level_intensity(idx)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `int` | idx | The glow level index to retrieve the intensity for |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current intensity of the specified glow level |

<hr/>

### Examples

```lua
engine.print("Glow level 1 intensity:", engine.emissive.get_level_intensity(1))
```
