---
title: engine.emissive.get_hdr_bleed_scale

badge: Client
---

### Syntax

```lua
local result = engine.emissive.get_hdr_bleed_scale()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current HDR bleed scale value |

<hr/>

### Examples

```lua
engine.print("Emissive HDR bleed scale:", engine.emissive.get_hdr_bleed_scale())
```
