---
title: engine.emissive.get_hdr_luminance_cap
description: Retrieves the current HDR luminance cap of the emissive post-processing effect
badge: Client
---

### Syntax

```lua
local result = engine.emissive.get_hdr_luminance_cap()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current HDR luminance cap value |

<hr/>

### Examples

```lua
engine.print("Emissive HDR luminance cap:", engine.emissive.get_hdr_luminance_cap())
```
