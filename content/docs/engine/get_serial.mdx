---
title: engine.get_serial
description: Retrieves engine's serial
badge: Client
---

### Syntax

```lua
local tick = engine.get_serial()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|


### Returns

| Type | Name | Description |
|------|------|-------------|
| `int` | tick | Engine's current serial |

<hr/>

### Example

```lua
engine.print("Current serial:", engine.get_serial())
```
