---
title: core.engine.get_serial
description: Retrieves engine's current serial
badge: Client
---

### Syntax

```lua
local serial = core.engine.get_serial()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` | serial | Engine's current serial |

---

### Examples

```lua lineNumbers title="Prints engine's current serial"
core.engine.print("info", "Current serial:", core.engine.get_serial())
```
