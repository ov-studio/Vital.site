---
title: engine.get_entities
description: Retrieves all registered entities of a specified type
badge: Shared
---

### Syntax

```lua
local result = engine.get_entities(type)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | type | The entity type to query |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `table` | result | A table containing all registered entities of the specified type |

<hr/>

### Entities

| Entity Type | Availability |
|-------------|--------------|
| `"database"` | `Server` |
| `"database_query"` | `Server` |
| `"model"` | `Client` \| `Server` |
| `"webview"` | `Client` |
| `"font"` | `Client` |
| `"texture"` | `Client` |
| `"svg"` | `Client` |
| `"rendertarget"` | `Client` |
| `"timer"` | `Client` \| `Server` |
| `"promise"` | `Client` \| `Server` |
| `"thread"` | `Client` \| `Server` |

<hr/>

### Examples

```lua lineNumbers title="Prints all entities of a given type"
engine.iprint(engine.get_entities("model"))
```

```lua lineNumbers title="Counts entities of a given type"
engine.print("info", "Total webviews:", #engine.get_entities("webview"))
```
