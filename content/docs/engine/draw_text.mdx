---
title: engine.draw_text
description: Draws text within a bounding box on the canvas
badge: Canvas
---

### Syntax

```lua
local status = engine.draw_text(
    text, 
    start_at, 
    end_at, 
    font, 
    font_size, 
    color = {1, 1, 1, 1}, 
    alignment = {"left", "top"}, 
    clip = false, 
    wordwrap = false, 
    stroke = 0, 
    stroke_color = {1, 1, 1, 1}, 
    rotation = 0, 
    pivot = {0, 0}
)
```

<Callout type="warn">
  **Must be called within the `vital.sandbox:draw` event.** Invoking this function outside of that event will have no effect.

  - **Inside the event** — executes as expected, rendering text to the canvas each frame
  - **Outside the event** — the call will be silently ignored and nothing will be drawn
  - **Recommended usage** — register a handler via `network:fetch("vital.sandbox:draw", true):on(...)` and place all canvas draw calls inside it
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | text | Text string to draw |
| `vector2` | start_at | Top-left corner of the bounding box |
| `vector2` | end_at | Bottom-right corner of the bounding box |
| `font` | font | Font to render the text with |
| `int` | font_size | Font size in points |
| `color` | color | Color of the text |
| `alignment[horizontal, vertical]` | alignment | Horizontal alignment: <br/>•`"left"` / `0` <br/>•`"center"` / `1` <br/>•`"right"` / `2` <br/>•`"fill"` / `3` <br/><br/>Vertical alignment: <br/>•`"top"` / `0` <br/>•`"center"` / `1` <br/>•`"bottom"` / `2` <br/>•`"fill"` / `3` |
| `bool` | clip | When `true`, text is clipped to the bounding box |
| `bool` | wordwrap | When `true`, text wraps at the bounding box width |
| `int` | stroke | Outline thickness in pixels — set to `0` to disable |
| `color` | stroke_color | Color of the text outline |
| `float` | rotation | Rotation angle in radians |
| `vector2` | pivot | Pivot point for rotation, relative to `start_at` |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
local fontface = font.create("fonts/Roboto-Regular.ttf")

network:fetch("vital.sandbox:draw", true):on(function()
    --Draw simple text
    engine.draw_text("Hello World", {100, 100}, {400, 150}, fontface, 24)

    --Draw centered text with a custom color
    engine.draw_text("Centered", {100, 100}, {500, 200}, fontface, 32, {1, 1, 0, 1}, {"center", "center"})

    --Draw word-wrapped text clipped to its box
    engine.draw_text("This is a long string that will wrap.", {50, 50}, {300, 300}, fontface, 18, {1, 1, 1, 1}, {"left", "top"}, true, true)

    --Draw text with an outline
    engine.draw_text("Outlined", {100, 100}, {400, 160}, fontface, 28, {1, 1, 1, 1}, {"center", "center"}, false, false, 2, {0, 0, 0, 1})

    --Draw rotated text
    engine.draw_text("Rotated", {200, 200}, {400, 250}, fontface, 20, {1, 1, 1, 1}, {"left", "top"}, false, false, 0, {1, 1, 1, 1}, math.pi/6, {0, 0})
end)
```
