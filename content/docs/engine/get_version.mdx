---
title: engine.get_version
description: Retrieves engine's current build version
badge: Shared
---

### Syntax

```lua
local version = engine.get_version()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` | version | Engine's current build version |

<hr/>

### Examples

```lua lineNumbers title="Prints engine's current build version"
engine.print("info", "Build Version:", engine.get_version())
```
