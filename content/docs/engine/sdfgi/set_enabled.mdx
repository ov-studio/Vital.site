---
title: engine.sdfgi.set_enabled
description: Toggles the SDFGI post-processing effect
badge: Client
---

### Syntax

```lua
local status = engine.sdfgi.set_enabled(state)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `bool` | state | Determines SDFGI's state:<br/>• When `true` - SDFGI is enabled<br/>• When `false` - SDFGI is disabled |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
--Enable SDFGI
engine.sdfgi.set_enabled(true)

--Disable SDFGI
engine.sdfgi.set_enabled(false)
```
