---
title: engine.sdfgi.is_enabled
description: Retrieves whether the SDFGI post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local result = engine.sdfgi.is_enabled()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | Returns `true` if SDFGI is enabled, `false` otherwise |

<hr/>

### Examples

```lua
engine.print("SDFGI enabled:", engine.sdfgi.is_enabled())
```
