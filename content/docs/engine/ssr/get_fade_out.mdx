---
title: engine.ssr.get_fade_out
description: Retrieves the current fade-out distance of the SSR post-processing effect
badge: Client
---

### Syntax

```lua
local result = engine.ssr.get_fade_out()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Returns the current fade-out value |

<hr/>

### Examples

```lua
engine.print("SSR fade out:", engine.ssr.get_fade_out())
```
