---
title: engine.ssr.get_fade_in
description: Retrieves the current fade-in distance of the SSR post-processing effect
badge: Client
---

### Syntax

```lua
local result = engine.ssr.get_fade_in()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Returns the current fade-in value |

<hr/>

### Examples

```lua
engine.print("SSR fade in:", engine.ssr.get_fade_in())
```
