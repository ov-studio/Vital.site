---
title: engine.ssr.is_enabled
description: Retrieves whether the SSR post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local result = engine.ssr.is_enabled()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if SSR is enabled, `false` otherwise |

<hr/>

### Examples

```lua
engine.print("SSR enabled:", engine.ssr.is_enabled())
```
