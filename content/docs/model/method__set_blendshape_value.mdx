---
title: self:set_blendshape_value
description: Sets the value of a blend shape on a specific component of the model
badge: Shared
---

### Syntax

```lua
local status = self:set_blendshape_value(component, blendshape, value)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | component | Path of the component containing the blend shape |
| `string` | blendshape | Name of the blend shape to update |
| `float` | value | Value to apply to the blend shape |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
local entity = model.create("character")

entity:set_blendshape_value("Skeleton3D/Head", "Smile", 1.0)
```
