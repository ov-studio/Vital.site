---
title: self:get_rotation
description: Retrieves the current rotation of the model in 3D world-space
badge: Shared
---

### Syntax

```lua
local rotation = self:get_rotation()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `vector3` | rotation | 3D world-space rotation of the model as `{x, y, z}` (in degrees) |

<hr/>

### Examples

```lua lineNumbers title="Get the world-space rotation of a model"
local entity = model.create("character")
local rotation = entity:get_rotation()

engine.print("info", "Rotation:", rotation[1], rotation[2], rotation[3])
```
