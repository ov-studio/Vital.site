---
title: self:get_blendshape_value
description: Retrieves the current value of a blend shape on a specific component
badge: Shared
---

### Syntax

```lua
local value = self:get_blendshape_value(component, blendshape)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | component | Path of the component containing the blend shape |
| `string` | blendshape | Name of the blend shape to retrieve |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Current value of the blend shape |

<hr/>

### Examples

```lua lineNumbers title="Get the current value of a blend shape on a component"
local entity = model.create("character")

engine.print("info", "Blend shape value:", entity:get_blendshape_value("Skeleton3D/Head", "smile"))
```
