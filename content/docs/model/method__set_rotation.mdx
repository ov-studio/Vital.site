---
title: self:set_rotation
description: Sets the rotation of the model in 3D world-space
badge: Shared
---

### Syntax

```lua
local status = self:set_rotation(
    rotation
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `vector3` | rotation | 3D world-space rotation to apply to the model, as `{x, y, z}` (in degrees) |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Rotate a model 25 degrees on the Y axis"
local entity = model.create("character")

entity:set_rotation({0, 25, 0})
```
