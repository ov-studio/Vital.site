---
title: self:stop_animation
description: Stops the currently playing animation on the model
badge: Shared
---

### Syntax

```lua
local status = self:stop_animation()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Stop currently playing animation on a model"
local entity = model.create("character")

entity:play_animation("idle")
entity:stop_animation()
```
