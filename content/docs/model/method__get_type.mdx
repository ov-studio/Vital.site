---
title: self:get_type
description: Returns the type name of the instance
badge: Shared
---

### Syntax

```lua
local result = self:get_type()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` / `bool` | result | Type of the instance, or `false` if unavailable |

<hr/>

### Examples

```lua lineNumbers title="Print the type of a model instance"
local entity = model.create("character")

engine.print("info", entity:get_type()) --"model"
```
