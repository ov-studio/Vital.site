---
title: self:get_blendshapes
description: Retrieves a list of all blend shapes available on a specific component
badge: Shared
---

### Syntax

```lua
local blendshapes = self:get_blendshapes(component)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | component | Path of the component to retrieve blend shapes from |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `table` | blendshapes | Array of blend shape name strings |

<hr/>

### Examples

```lua
local entity = model.create("character")
local blendshapes = entity:get_blendshapes("Skeleton3D/Torso_vest_")

for i = 1, #blendshapes do
    engine.print("info", blendshapes[i])
end
```
