---
title: self:set_rotation
description: Sets the rotation of the model in 3D space
badge: Shared
---

### Syntax

```lua
local status = self:set_rotation(rotation)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `vector3` | rotation | Rotation to apply to the model, as `{x, y, z}` (in degrees) |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
local entity = model.create("character")

entity:set_rotation({0, 25, 0})
```
