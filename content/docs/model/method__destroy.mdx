---
title: self:destroy
description: Destroys the model instance and removes it from the scene
badge: Shared
---

### Syntax

```lua
local status = self:destroy()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Destroy a model instance"
local entity = model.create("character")

entity:destroy()
```
