---
title: model.create
description: Creates a model instance from a previously loaded model
badge: Shared
---

### Syntax

```lua
local instance = model.create(name)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | name | Name of the loaded model to instantiate |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `model` | instance | The created model instance |

<hr/>

### Examples

```lua
model.load("character", "models/character.glb")
local instance = model.create("character")
```
