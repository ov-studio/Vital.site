---
title: self:pause_animation
description: Pauses the currently playing animation on the model
badge: Shared
---

### Syntax

```lua
local status = self:pause_animation()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Pause currently playing animation on a model"
local entity = model.create("character")

entity:play_animation("idle")
entity:pause_animation()
```
