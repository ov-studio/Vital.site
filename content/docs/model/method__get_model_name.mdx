---
title: self:get_model_name
description: Retrieves the registered name of the model
badge: Shared
---

### Syntax

```lua
local name = self:get_model_name()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` | name | Registered name of the model |

<hr/>

### Examples

```lua lineNumbers title="Get the registered name of a model instance"
local entity = model.create("character")

engine.print("info", "Model name:", entity:get_model_name()) --'character'
```
