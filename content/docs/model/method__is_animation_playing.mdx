---
title: self:is_animation_playing
description: Retrieves whether an animation is currently playing on the model
badge: Shared
---

### Syntax

```lua
local result = self:is_animation_playing()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if an animation is playing, `false` otherwise |

<hr/>

### Examples

```lua lineNumbers title="Check whether an animation is currently playing"
local entity = model.create("character")

engine.print("info", "Animation playing:", entity:is_animation_playing())
```
