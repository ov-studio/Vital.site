---
title: core.model.unload
description: Unloads a previously loaded model from memory
badge: Shared
---

### Syntax

```lua
local status = core.model.unload(
    name
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | name | Name of the model to unload |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Unload a loaded model from memory"
core.model.unload("character")
```
