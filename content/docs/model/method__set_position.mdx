---
title: self:set_position
description: Sets the position of the model in 3D world-space
badge: Shared
---

### Syntax

```lua
local status = self:set_position(position)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `vector3` | position | 3D world-space position to move the model to, as `{x, y, z}` |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Move a model to the world origin"
local entity = model.create("character")

entity:set_position({0, 0, 0})
```
