---
title: self:get_components
description: Retrieves a list of all component paths on the model
badge: Shared
---

### Syntax

```lua
local components = self:get_components()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `table` | components | Array of component path strings |

<hr/>

### Examples

```lua
local instance = model.create("character")
local components = instance:get_components()
for i = 1, #components do
    engine.print("info", components[i])
end
```
