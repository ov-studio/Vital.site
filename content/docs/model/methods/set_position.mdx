---
title: self:set_position
description: Sets the position of the model in 3D space
badge: Shared
---

### Syntax

```lua
local status = self:set_position(x, y, z)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | x | Position on the X axis |
| `float` | y | Position on the Y axis |
| `float` | z | Position on the Z axis |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
local instance = model.create("character")

instance:set_position(0, 0, 0)
```
