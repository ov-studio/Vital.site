---
title: self:get_blend_shape_value
description: Retrieves the current value of a blend shape on a specific component
badge: Shared
---

### Syntax

```lua
local value = self:get_blend_shape_value(component, blend_shape)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | component | Path of the component containing the blend shape |
| `string` | blend_shape | Name of the blend shape to retrieve |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Current value of the blend shape |

<hr/>

### Examples

```lua
local instance = model.create("character")
local value = instance:get_blend_shape_value("Skeleton3D/Torso_vest_", "Smile")
engine.print("info", "Blend shape value:", value)
```
