---
title: self:is_animation_playing
description: Retrieves whether an animation is currently playing on the model
badge: Shared
---

### Syntax

```lua
local result = self:is_animation_playing()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if an animation is playing, `false` otherwise |

<hr/>

### Examples

```lua
local instance = model.create("character")

engine.print("info", "Animation playing:", instance:is_animation_playing())
```
