---
title: self:get_blend_shapes
description: Retrieves a list of all blend shapes available on a specific component
badge: Shared
---

### Syntax

```lua
local blend_shapes = self:get_blend_shapes(component)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | component | Path of the component to retrieve blend shapes from |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `table` | blend_shapes | Array of blend shape name strings |

<hr/>

### Examples

```lua
local instance = model.create("character")
local blend_shapes = instance:get_blend_shapes("Skeleton3D/Torso_vest_")

for i = 1, #blend_shapes do
    engine.print("info", blend_shapes[i])
end
```
