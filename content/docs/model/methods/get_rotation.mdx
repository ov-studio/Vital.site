---
title: self:get_rotation
description: Retrieves the current rotation of the model in 3D space
badge: Shared
---

### Syntax

```lua
local x, y, z = self:get_rotation()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | x | Rotation on the X axis (in radians) |
| `float` | y | Rotation on the Y axis (in radians) |
| `float` | z | Rotation on the Z axis (in radians) |

<hr/>

### Examples

```lua
local instance = model.create("character")
local x, y, z = instance:get_rotation()
engine.print("info", "Rotation:", x, y, z)
```
