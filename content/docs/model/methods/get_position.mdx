---
title: self:get_position
description: Retrieves the current position of the model in 3D space
badge: Shared
---

### Syntax

```lua
local x, y, z = self:get_position()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | x | Position on the X axis |
| `float` | y | Position on the Y axis |
| `float` | z | Position on the Z axis |

<hr/>

### Examples

```lua
local entity = model.create("character")

engine.print("info", "Entity position:", entity:get_position())
```
