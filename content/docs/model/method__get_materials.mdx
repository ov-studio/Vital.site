---
title: self:get_materials
description: Retrieves a list of all material names on a specific component
badge: Shared
---

### Syntax

```lua
local materials = self:get_materials(
    component
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | component | Path of the component to retrieve materials from |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `table` | materials | Array of material name strings |

<hr/>

### Examples

```lua lineNumbers title="Print all material names on a specific component"
local entity = model.create("character")
local materials = entity:get_materials("Skeleton3D/Head")

engine.iprint(materials)
```
