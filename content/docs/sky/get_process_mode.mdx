title: gfx.sky.get_process_mode

badge: Client

### Syntax

```lua
local result = gfx.sky.get_process_mode()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| [`gfx.sky.process_mode`](/docs/sky/enum/process_mode) | result | Current sky process mode |

---

### Examples

```lua lineNumbers title="Print current sky process mode"
core.engine.print("info", "Sky process mode:", gfx.sky.get_process_mode())
```
