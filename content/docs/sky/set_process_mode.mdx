---
title: gfx.sky.set_process_mode
description: Sets the process mode of the sky
badge: Client
---

### Syntax

```lua
local status = gfx.sky.set_process_mode(
    mode
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `int` | mode | Sky process mode enum value to apply<br/>Refer [**gfx.sky.process_mode**](/docs/sky/enum/process_mode) enum |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set sky process mode to automatic"
gfx.sky.set_process_mode(gfx.sky.process_mode.AUTOMATIC)
```
