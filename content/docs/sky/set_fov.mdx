---
title: gfx.sky.set_fov
description: Sets the custom field of view of the sky
badge: Client
---

### Syntax

```lua
local result = gfx.sky.set_fov(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Sky custom field of view to apply |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set sky field of view"
gfx.sky.set_fov(0.0)
```
