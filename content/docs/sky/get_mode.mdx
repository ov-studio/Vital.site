---
title: gfx.sky.get_mode
description: Retrieves the current material type of the sky
badge: Client
---

### Syntax

```lua
local result = gfx.sky.get_mode()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` \| `bool` | result | Current sky material type, `false` otherwise<br/>Refer [**Modes**](#modes) section |

---

<include>./set_mode.mdx#modes</include>

### Examples

```lua lineNumbers title="Print current sky material type"
core.engine.print("info", "Sky mode:", gfx.sky.get_mode())
```
