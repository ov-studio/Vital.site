---
title: gfx.sky.set_radiance_size
description: Sets the radiance cubemap size of the sky
badge: Client
---

### Syntax

```lua
local status = gfx.sky.set_radiance_size(
    size
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `int` | size | Radiance size enum value to apply<br/>Refer [**gfx.sky.radiance_size**](/docs/sky/enum/radiance_size) enum |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set sky radiance size to 128"
gfx.sky.set_radiance_size(gfx.sky.radiance_size.SIZE_128)
```
