---
title: self:get_size
description: Retrieves the camera's orthogonal size in meters
badge: Client
---

### Syntax

```lua
local result = self:get_size()
```

<Callout type="warn">
  **Only applicable in orthogonal projection mode.** Refer [`core.camera.set_projection`](/docs/camera/method/set_projection) to configure the active projection mode.

  - **Orthogonal mode required** — calling this function in any other projection mode will have no effect
  - **Recommended usage** — combine with `core.camera.projection.ORTHOGONAL` to switch projection before adjusting size
</Callout>

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current orthogonal size of the camera |

---

### Examples

```lua lineNumbers title="Get the orthogonal camera size"
local entity = core.camera.create()

entity:set_size(10)
core.engine.print("info", entity:get_size()) -- 10.0
```
