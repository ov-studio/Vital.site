---
title: self:get_auto_exposure_max_sensitivity
description: Retrieves the camera's maximum auto exposure sensitivity
badge: Client
---

### Syntax

```lua
local result = self:get_auto_exposure_max_sensitivity()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current maximum auto exposure sensitivity (ISO) |

---

### Examples

```lua lineNumbers title="Get the maximum auto exposure sensitivity"
local entity = core.camera.create()

core.engine.print("info", entity:get_auto_exposure_max_sensitivity())
```
