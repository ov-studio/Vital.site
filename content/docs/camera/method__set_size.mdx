---
title: self:set_size
description: Sets the camera's orthogonal size in meters
badge: Client
---

### Syntax

```lua
local status = self:set_size(
    size
)
```

<Callout type="warn">
  **Only applicable in orthogonal projection mode.** Refer [`core.camera.set_projection`](/docs/camera/method/set_projection) to configure the active projection mode.

  - **Orthogonal mode required** — calling this function in any other projection mode will have no effect
  - **Recommended usage** — combine with `core.camera.projection.ORTHOGONAL` to switch projection before adjusting size
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | size | Orthogonal view size |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set the orthogonal size"
local entity = core.camera.create()

entity:set_projection(core.camera.projection.ORTHOGONAL)
entity:set_size(20)
```
