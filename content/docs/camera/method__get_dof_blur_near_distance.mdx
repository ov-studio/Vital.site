---
title: self:get_dof_blur_near_distance
description: Retrieves the camera's near depth of field blur start distance
badge: Client
---

### Syntax

```lua
local result = self:get_dof_blur_near_distance()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Distance from the camera at which near DOF blur begins |

---

### Examples

```lua lineNumbers title="Get the near DOF blur distance"
local entity = core.camera.create()

core.engine.print("info", entity:get_dof_blur_near_distance())
```
