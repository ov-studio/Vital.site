---
title: self:set_orthogonal
description: Configures the camera for orthogonal projection with the given parameters
badge: Client
---

### Syntax

```lua
local status = self:set_orthogonal(
    size,
    z_near,
    z_far
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | size | Orthogonal view size |
| `float` | z_near | Near clip distance |
| `float` | z_far | Far clip distance |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set orthogonal projection"
local entity = core.camera.create()

entity:set_orthogonal(10, 0.1, 1000)
entity:set_active()
```
