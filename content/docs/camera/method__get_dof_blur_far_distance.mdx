---
title: self:get_dof_blur_far_distance
description: Returns the camera's far depth of field blur start distance
badge: Client
---

### Syntax

```lua
local result = self:get_dof_blur_far_distance()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Distance from the camera at which far DOF blur begins |

---

### Examples

```lua lineNumbers title="Get the far DOF blur distance"
local entity = core.camera.create()

core.engine.print("info", entity:get_dof_blur_far_distance())
```
