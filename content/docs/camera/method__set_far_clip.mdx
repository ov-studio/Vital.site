---
title: self:set_far_clip
description: Sets the camera's far clip distance
badge: Client
---

### Syntax

```lua
local status = self:set_far_clip(
    far_clip
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | far_clip | Far clip distance to apply<br/>Objects beyond this distance will not be rendered |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set the far clip distance"
local entity = core.camera.create()

entity:set_far_clip(2000)
```
