---
title: self:get_doppler_tracking
description: Retrieves the camera's current doppler tracking mode
badge: Client
---

### Syntax

```lua
local result = self:get_doppler_tracking()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| [`core.camera.doppler_tracking`](/docs/camera/enum/doppler_tracking) | result | Current doppler tracking mode of the camera |

---

### Examples

```lua lineNumbers title="Get the doppler tracking mode"
local entity = core.camera.create()

core.engine.print("info", entity:get_doppler_tracking()) -- core.camera.doppler_tracking.DISABLED
```
