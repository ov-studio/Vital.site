---
title: self:get_fov
description: Retrieves the camera's field of view
badge: Client
---

### Syntax

```lua
local result = self:get_fov()
```

<Callout type="warn">
  **Only applicable in perspective projection mode.** Refer [`core.camera.set_projection`](/docs/camera/method/set_projection) to configure the active projection mode.

  - **Perspective mode required** — calling this function in any other projection mode will have no effect.
  - **Recommended usage** — combine with `core.camera.projection.PERSPECTIVE` to switch projection before adjusting FOV.
</Callout>

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current field of view in degrees |

---

### Examples

```lua lineNumbers title="Get the camera FOV"
local entity = core.camera.create()

core.engine.print("info", entity:get_fov())
```
