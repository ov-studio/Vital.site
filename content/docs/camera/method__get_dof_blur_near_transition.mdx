---
title: self:get_dof_blur_near_transition
description: Returns the camera's near depth of field blur transition distance
badge: Client
---

### Syntax

```lua
local result = self:get_dof_blur_near_transition()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Distance over which the near DOF blur gradually fades in |

---

### Examples

```lua lineNumbers title="Get the near DOF blur transition distance"
local entity = core.camera.create()

core.engine.print("info", entity:get_dof_blur_near_transition())
```
