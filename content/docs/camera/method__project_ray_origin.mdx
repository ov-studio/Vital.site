---
title: self:project_ray_origin
description: Retrieves the world-space origin point of a ray cast from a given screen point
badge: Client
---

### Syntax

```lua
local result = self:project_ray_origin(
    position
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `vector2` | position | Position on the screen in pixels |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `vector3` | result | World-space origin of the ray cast from the given screen point |

---

### Examples

```lua lineNumbers title="Get a ray origin from a screen point"
local entity = core.camera.create()
local position = {960, 540}

entity:set_active()

local ray_origin = entity:project_ray_origin(position)
local ray_normal = entity:project_ray_normal(position)

core.engine.print("info", "Ray from:", ray_origin[1], ray_origin[2], ray_origin[3])
core.engine.print("info", "Ray towards:", ray_normal[1], ray_normal[2], ray_normal[3])
```
