---
title: self:project_ray_normal
description: Retrieves a normalized direction vector from the camera through a given screen point
badge: Client
---

### Syntax

```lua
local result = self:project_ray_normal(
    position
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `vector2` | position | Position on the screen in pixels |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `vector3` | result | Normalized direction vector from the camera through the given screen point |

---

### Examples

```lua lineNumbers title="Get a ray direction from the screen center"
local resolution = core.engine.get_resolution()
local center = {resolution[1] * 0.5, resolution[2] * 0.5}
local entity = core.camera.create()

entity:set_active()

local ray_dir = entity:project_ray_normal(center)
core.engine.print("info", "Ray direction:", ray_dir[1], ray_dir[2], ray_dir[3])
```
