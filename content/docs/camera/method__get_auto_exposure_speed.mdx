---
title: self:get_auto_exposure_speed
description: Retrieves the camera's auto exposure speed
badge: Client
---

### Syntax

```lua
local result = self:get_auto_exposure_speed()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current auto exposure speed<br/>Affects how quickly the camera adapts to lighting changes |

---

### Examples

```lua lineNumbers title="Get the auto exposure speed"
local entity = core.camera.create()

core.engine.print("info", entity:get_auto_exposure_speed())
```
