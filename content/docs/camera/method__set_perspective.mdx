---
title: self:set_perspective
description: Configures the camera for perspective projection with the given parameters
badge: Client
---

### Syntax

```lua
local status = self:set_perspective(
    fov,
    near_clip,
    far_clip
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | fov | Field of view in degrees |
| `float` | near_clip | Near clip distance |
| `float` | far_clip | Far clip distance |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set perspective projection"
local entity = core.camera.create()

entity:set_perspective(75, 0.1, 1000)
entity:set_active()
```
