---
title: self:get_v_offset
description: Returns the camera's vertical offset
badge: Client
---

### Syntax

```lua
local result = self:get_v_offset()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current vertical offset of the camera |

---

### Examples

```lua lineNumbers title="Get the vertical offset"
local entity = core.camera.create()

core.engine.print("info", entity:get_v_offset())
```
