---
title: ssil.get_radius
description: Retrieves the current sampling radius of the SSIL post-processing effect
badge: Client
---

### Syntax

```lua
local result = ssil.get_radius()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current radius value |

<hr/>

### Examples

```lua lineNumbers title="Print current ssil radius"
engine.print("info", "SSIL radius:", ssil.get_radius())
```
