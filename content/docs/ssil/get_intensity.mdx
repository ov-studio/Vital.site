---
title: ssil.get_intensity
description: Retrieves the current intensity of the SSIL post-processing effect
badge: Client
---

### Syntax

```lua
local result = ssil.get_intensity()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current intensity value |

<hr/>

### Examples

```lua lineNumbers title="Print current ssil intensity"
engine.print("info", "SSIL intensity:", ssil.get_intensity())
```
