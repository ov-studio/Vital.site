---
title: ssil.get_sharpness
description: Retrieves the current sharpness of the SSIL post-processing effect
badge: Client
---

### Syntax

```lua
local result = ssil.get_sharpness()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current sharpness value |

<hr/>

### Examples

```lua lineNumbers title="Print current ssil sharpness"
engine.print("info", "SSIL sharpness:", ssil.get_sharpness())
```
