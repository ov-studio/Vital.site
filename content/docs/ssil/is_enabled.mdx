---
title: ssil.is_enabled
description: Retrieves whether the SSIL post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local result = ssil.is_enabled()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if SSIL is enabled, `false` otherwise |

<hr/>

### Examples

```lua lineNumbers title="Print whether ssil is enabled"
engine.print("info", "SSIL enabled:", ssil.is_enabled())
```
