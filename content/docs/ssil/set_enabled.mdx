---
title: ssil.set_enabled
description: Toggles the SSIL post-processing effect
badge: Client
---

### Syntax

```lua
local status = ssil.set_enabled(
    state
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `bool` | state | Determines SSIL's state:<br/>• When `true` - SSIL is enabled<br/>• When `false` - SSIL is disabled |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Toggle ssil effect"
--Enable SSIL
ssil.set_enabled(true)

--Disable SSIL
ssil.set_enabled(false)
```
