---
title: gfx.ssil.set_radius
description: Sets the sampling radius of SSIL post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.ssil.set_radius(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Radius value to apply<br/>Larger values extend the reach of indirect light contribution, illuminating a wider area around surfaces |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set a wider indirect light radius"
gfx.ssil.set_radius(5.0)
```
