---
title: ssil.set_radius
description: The sampling radius to be set for the SSIL post-processing effect
badge: Client
---

### Syntax

```lua
local status = ssil.set_radius(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Radius value to be applied<br/>Larger values extend the reach of indirect light contribution, illuminating a wider area around surfaces |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Set a wider indirect light radius"
ssil.set_radius(5.0)
```
