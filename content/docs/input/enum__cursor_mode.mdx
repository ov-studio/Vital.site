---
title: util.input.cursor_mode
description: Enumeration of supported cursor visibility and confinement modes
badge: Client
---

### Values

| Constant | Description |
|----------|-------------|
| `util.input.cursor_mode.VISIBLE` | Cursor is visible and moves freely — default OS behavior |
| `util.input.cursor_mode.HIDDEN` | Cursor is hidden but still moves freely and can leave the window |
| `util.input.cursor_mode.CAPTURED` | Cursor is hidden and locked to the window — ideal for FPS-style camera control<br/>Raw mouse delta is still reported via [`util.input.get_cursor_velocity`](/docs/input/get_cursor_velocity) |
