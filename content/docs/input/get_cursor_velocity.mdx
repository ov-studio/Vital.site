---
title: util.input.get_cursor_velocity
description: Retrieves the cursor velocity from the last input frame
badge: Client
---

### Syntax

```lua
local velocity = util.input.get_cursor_velocity()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `vector2` | velocity | Mouse movement delta as `{x, y}` in pixels per second |

---

### Examples

```lua lineNumbers title="Print the current cursor velocity"
core.engine.print("info", "Cursor velocity:", util.table.unpack(util.input.get_cursor_velocity()))
```

```lua lineNumbers title="Use cursor velocity for FPS camera rotation"
util.input.set_cursor_mode(util.input.cursor_mode.CAPTURED)

local sensitivity = 0.1
local velocity = util.input.get_cursor_velocity()
local yaw = velocity[1] * sensitivity
local pitch = velocity[2] * sensitivity

core.engine.print("info", "Yaw delta:", yaw, "Pitch delta:", pitch)
```
