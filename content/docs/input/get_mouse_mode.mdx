---
title: util.input.get_mouse_mode
description: Returns the current cursor visibility and confinement mode
badge: Client
---

### Syntax

```lua
local mode = util.input.get_mouse_mode()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `int` | mode | Current mouse mode<br/>Refer [**util.input.mouse_mode**](/docs/input/enum/mouse_mode) enum |

---

### Examples

```lua lineNumbers title="Print the current mouse mode"
local mode = util.input.get_mouse_mode()

core.engine.print("info", "Mouse mode:", mode)
```

```lua lineNumbers title="Toggle captured mode only when not already active"
if util.input.get_mouse_mode() ~= util.input.mouse_mode.CAPTURED then
    util.input.set_mouse_mode("CAPTURED")
end
```
