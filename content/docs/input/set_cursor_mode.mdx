---
title: util.input.set_cursor_mode
description: Sets the cursor visibility and confinement mode
badge: Client
---

### Syntax

```lua
local status = util.input.set_cursor_mode(
    mode
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `int` | mode | Target cursor mode to apply<br/>Refer [**util.input.cursor_mode**](/docs/input/enum/cursor_mode) enum |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Capture the cursor for FPS camera control"
util.input.set_cursor_mode(util.input.cursor_mode.CAPTURED)
```

```lua lineNumbers title="Restore the cursor when opening a menu"
util.input.set_cursor_mode(util.input.cursor_mode.VISIBLE)
```

```lua lineNumbers title="Hide the cursor while still allowing free movement"
util.input.set_cursor_mode(util.input.cursor_mode.HIDDEN)
```
