---
title: util.input.is_pressed
description: Checks whether a keyboard or mouse button is currently held down
badge: Client
---

### Syntax

```lua
local result = util.input.is_pressed(
    key
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | key | Key or button identifier to query<br/>Refer [**util.input.key**](/docs/input/enum/key) enum |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if the key or button is currently held down, `false` otherwise |

---

### Examples

```lua lineNumbers title="Check if the W key is held"
if util.input.is_pressed("W") then
    core.engine.print("info", "W is held")
end
```

```lua lineNumbers title="Check if the left mouse button is held"
if util.input.is_pressed("MOUSE_LEFT") then
    core.engine.print("info", "Left mouse button is held")
end
```

```lua lineNumbers title="Check multiple keys for a sprint condition"
local moving = util.input.is_pressed("W")
local sprinting = moving and util.input.is_pressed("SHIFT")

core.engine.print("info", "Sprinting:", sprinting)
```
