---
title: util.input.mouse_mode
description: Enumeration of supported cursor visibility and confinement modes
badge: Client
---

### Values

| Constant | Description |
|----------|-------------|
| `util.input.mouse_mode.VISIBLE` | Cursor is visible and moves freely — default OS behavior |
| `util.input.mouse_mode.HIDDEN` | Cursor is hidden but still moves freely and can leave the window |
| `util.input.mouse_mode.CAPTURED` | Cursor is hidden and locked to the window — ideal for FPS-style camera control<br/>Raw mouse delta is still reported via [`util.input.get_mouse_velocity`](/docs/input/get_mouse_velocity) |
| `util.input.mouse_mode.CONFINED` | Cursor is visible but cannot leave the window bounds |
| `util.input.mouse_mode.CONFINED_HIDDEN` | Cursor is hidden and cannot leave the window bounds |
