---
title: util.input.get_cursor_position
description: Retrieves the current cursor position relative to the root viewport
badge: Client
---

### Syntax

```lua
local position = util.input.get_cursor_position()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `vector2` | position | Cursor position as `{x, y}` in pixels, relative to the root viewport |

---

### Examples

```lua lineNumbers title="Print the current cursor position"
core.engine.print("info", "Cursor position:", util.table.unpack(util.input.get_cursor_position()))
```

```lua lineNumbers title="Convert the cursor position to a world-space ray"
local cursor_position = util.input.get_cursor_position()
local world_position = core.engine.screen_to_world(position, 10)

core.engine.print("info", "World position:", util.table.unpack(world_position))
```
