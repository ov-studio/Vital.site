---
title: gfx.glow.reset_map
description: Removes the currently active glow map on the glow post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.glow.reset_map()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Remove the currently active glow map"
gfx.glow.reset_map()
```
