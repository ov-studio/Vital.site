---
title: gfx.glow.set_strength
description: Sets the strength of the glow post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.glow.set_strength(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Strength value to apply<br/>Controls the glow's spread or filter strength. Higher values produce a softer, wider glow while lower values keep the glow tighter around glow surfaces |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set a soft wide glow spread"
gfx.glow.set_strength(2.0)
```
