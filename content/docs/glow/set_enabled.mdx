---
title: gfx.glow.set_enabled
description: Toggles the glow post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.glow.set_enabled(
    state
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `bool` | state | Determines glow's state:<br/>• When `true` - glow is enabled<br/>• When `false` - glow is disabled |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Toggle glow"
-- Enable glow
gfx.glow.set_enabled(true)

-- Disable glow
gfx.glow.set_enabled(false)
```
