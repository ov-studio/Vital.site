---
title: gfx.glow.set_level_intensity
description: Sets the intensity of a specific level of the glow post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.glow.set_level_intensity(
    idx,
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `int` | idx | Level index to target<br/>Glow is applied in multiple passes at different scales; each level corresponds to a blur pass of increasing size |
| `float` | value | Intensity value to apply to the specified level<br/>Higher values increase the contribution of that level to the final result |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set intensity of a specific level"
-- Set a strong contribution on level 1
gfx.glow.set_level_intensity(1, 1.0)

-- Disable contribution on level 3
gfx.glow.set_level_intensity(3, 0.0)
```
