---
title: gfx.glow.get_level_intensity
description: Retrieves the current intensity of a specific glow level of the glow post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.glow.get_level_intensity(
    idx
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `int` | idx | Glow level index to be targetted |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current intensity of the specified glow level |

---

### Examples

```lua lineNumbers title="Print intensity of a specific glow level"
core.engine.print("info", "Glow level 1 intensity:", gfx.glow.get_level_intensity(1))
```
