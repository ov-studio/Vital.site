---
title: gfx.glow.is_enabled
description: Retrieves whether the glow glow post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local result = gfx.glow.is_enabled()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if glow glow is enabled, `false` otherwise |

---

### Examples

```lua lineNumbers title="Check if glow glow is enabled"
core.engine.print("info", "Glow enabled:", gfx.glow.is_enabled())
```
