---
title: gfx.glow.get_intensity
description: Retrieves the current overall intensity of the glow post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.glow.get_intensity()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current intensity value |

---

### Examples

```lua lineNumbers title="Print glow overall intensity"
core.engine.print("info", "Emissive intensity:", gfx.glow.get_intensity())
```
