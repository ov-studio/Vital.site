---
title: gfx.glow.set_hdr_bleed_scale
description: The HDR bleed scale to be set for the glow post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.glow.set_hdr_bleed_scale(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | HDR bleed scale value to be applied<br/>Controls how strongly pixels above the bleed threshold contribute to the glow. Higher values amplify the bleed from bright HDR areas, producing a more intense halo around very bright surfaces |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Amplify bleed from bright HDR surfaces"
gfx.glow.set_hdr_bleed_scale(2.0)
```
