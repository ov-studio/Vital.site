---
title: gfx.glow.set_map_strength
description: The glow map strength to be set for the glow post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.glow.set_map_strength(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Glow map strength value to be applied<br/>Controls how strongly the assigned glow map texture influences the glow effect. Higher values give the glow map more influence over which areas of the scene produce glow |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Give the glow map full influence"
gfx.glow.set_map_strength(1.0)
```
