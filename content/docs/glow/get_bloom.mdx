---
title: gfx.glow.get_bloom
description: Retrieves the current bloom threshold of the glow post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.glow.get_bloom()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current bloom value |

---

### Examples

```lua lineNumbers title="Print glow bloom threshold"
core.engine.print("info", "Glow bloom:", gfx.glow.get_bloom())
```
