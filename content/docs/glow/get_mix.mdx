---
title: gfx.glow.get_mix
description: Retrieves the current mix amount of the glow post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.glow.get_mix()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current mix value |

---

### Examples

```lua lineNumbers title="Print glow mix amount"
core.engine.print("info", "Glow mix:", gfx.glow.get_mix())
```
