---
title: gfx.glow.get_hdr_bleed_threshold
description: Retrieves the current HDR bleed threshold of the glow post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.glow.get_hdr_bleed_threshold()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current HDR bleed threshold value |

---

### Examples

```lua lineNumbers title="Print glow HDR bleed threshold"
core.engine.print("info", "Glow HDR bleed threshold:", gfx.glow.get_hdr_bleed_threshold())
```
