---
title: gfx.glow.get_hdr_bleed_scale
description: Retrieves the current HDR bleed scale of the glow post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.glow.get_hdr_bleed_scale()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current HDR bleed scale value |

---

### Examples

```lua lineNumbers title="Print glow HDR bleed scale"
core.engine.print("info", "Glow HDR bleed scale:", gfx.glow.get_hdr_bleed_scale())
```
