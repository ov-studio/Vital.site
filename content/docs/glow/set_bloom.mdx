---
title: gfx.glow.set_bloom
description: The bloom threshold to set for the glow post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.glow.set_bloom(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Bloom value to apply<br/>Controls how much of the glow effect bleeds onto non-glow surfaces in the scene. Higher values cause the glow to spread further and illuminate surrounding geometry |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Allow glow to bleed onto surrounding surfaces"
gfx.glow.set_bloom(0.5)
```
