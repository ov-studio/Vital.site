---
title: gfx.glow.get_blend_mode
description: Retrieves the current blend mode of the glow post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.glow.get_blend_mode()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `int` | result | Current blend mode |

---

### Examples

```lua lineNumbers title="Print glow blend mode"
core.engine.print("info", "Glow blend mode:", gfx.glow.get_blend_mode())
```
