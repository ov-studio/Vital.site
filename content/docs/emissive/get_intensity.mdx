---
title: emissive.get_intensity
description: Retrieves the current overall intensity of the emissive post-processing effect
badge: Client
---

### Syntax

```lua
local result = emissive.get_intensity()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current intensity value |

<hr/>

### Examples

```lua lineNumbers title="Print emissive overall intensity"
engine.print("info", "Emissive intensity:", emissive.get_intensity())
```
