---
title: gfx.emissive.get_strength
description: Retrieves the current strength of the emissive post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.emissive.get_strength()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current strength value |

---

### Examples

```lua lineNumbers title="Print emissive strength"
core.engine.print("info", "Emissive strength:", gfx.emissive.get_strength())
```
