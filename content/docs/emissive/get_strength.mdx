---
title: emissive.get_strength
description: Retrieves the current strength of the emissive post-processing effect
badge: Client
---

### Syntax

```lua
local result = emissive.get_strength()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current strength value |

<hr/>

### Examples

```lua lineNumbers title="Print emissive strength"
engine.print("info", "Emissive strength:", emissive.get_strength())
```
