---
title: emissive.is_enabled
description: Retrieves whether the emissive glow post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local result = emissive.is_enabled()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if emissive glow is enabled, `false` otherwise |

<hr/>

### Examples

```lua lineNumbers title="Check if emissive glow is enabled"
engine.print("info", "Emissive enabled:", emissive.is_enabled())
```
