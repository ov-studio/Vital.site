---
title: emissive.get_hdr_bleed_scale
description: Retrieves the current HDR bleed scale of the emissive post-processing effect
badge: Client
---

### Syntax

```lua
local result = emissive.get_hdr_bleed_scale()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current HDR bleed scale value |

<hr/>

### Examples

```lua lineNumbers title="Print emissive HDR bleed scale"
engine.print("info", "Emissive HDR bleed scale:", emissive.get_hdr_bleed_scale())
```
