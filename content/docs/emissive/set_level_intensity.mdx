---
title: emissive.set_level_intensity
description: The intensity to be set for a specific glow level of the emissive post-processing effect
badge: Client
---

### Syntax

```lua
local status = emissive.set_level_intensity(idx, value)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `int` | idx | Glow level index to be targetted<br/>Glow is applied in multiple passes at different scales; each level corresponds to a blur pass of increasing size |
| `float` | value | Intensity value to be applied to the specified level<br/>Higher values increase the contribution of that glow level to the final result |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Set intensity of a specific glow level"
--Set a strong contribution on glow level 1
emissive.set_level_intensity(1, 1.0)

--Disable contribution on glow level 3
emissive.set_level_intensity(3, 0.0)
```
