---
title: emissive.get_blend_mode
description: Retrieves the current blend mode of the emissive post-processing effect
badge: Client
---

### Syntax

```lua
local result = emissive.get_blend_mode()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `int` | result | Current blend mode |

<hr/>

### Examples

```lua lineNumbers title="Print emissive blend mode"
engine.print("info", "Emissive blend mode:", emissive.get_blend_mode())
```
