---
title: emissive.get_map_strength
description: Retrieves the current glow map strength of the emissive post-processing effect
badge: Client
---

### Syntax

```lua
local result = emissive.get_map_strength()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current glow map strength value |

<hr/>

### Examples

```lua lineNumbers title="Print emissive glow map strength"
engine.print("info", "Emissive map strength:", emissive.get_map_strength())
```
