---
title: self:update
description: Refreshes the webview's size and visibility state
badge: Client
---

### Syntax

```lua
local status = self:update()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
local browser = webview.create()

--Apply after repositioning or resizing
browser:set_size(1280, 720)
browser:update()
```
