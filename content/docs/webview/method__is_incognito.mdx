---
title: self:is_incognito
description: Retrieves whether the webview is in incognito mode
badge: Client
---

### Syntax

```lua
local result = self:is_incognito()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if the webview is fullscreen, `false` otherwise |

---

### Examples

```lua lineNumbers title="Check if a webview is in incognito mode"
local entity = core.webview.create()

core.engine.print("info", "Webview incognito:", entity:is_incognito())
```
