---
title: self:set_devtools_visible
description: Toggles the developer tools panel on the webview
badge: Client
---

### Syntax

```lua
local status = self:set_devtools_visible(
    state
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `bool` | state | Determines the devtools visibility:<br/>• When `true` - the devtools panel is opened<br/>• When `false` - the devtools panel is closed |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Toggle devtools's visibility on a webview"
local wv = webview.create()

--Open devtools
wv:set_devtools_visible(true)

--Close devtools
wv:set_devtools_visible(false)
```
