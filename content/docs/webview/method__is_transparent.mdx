---
title: self:is_transparent
description: Retrieves whether the webview background is transparent
badge: Client
---

### Syntax

```lua
local result = self:is_transparent()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if the webview is transparent, `false` otherwise |

<hr/>

### Examples

```lua lineNumbers title="Check if a webview has a transparent background"
local browser = webview.create()

engine.print("info", "Webview transparent:", browser:is_transparent())
```
