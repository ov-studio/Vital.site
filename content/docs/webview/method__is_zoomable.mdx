---
title: self:is_zoomable
description: Retrieves whether zoom hotkeys are enabled on the webview
badge: Client
---

### Syntax

```lua
local result = self:is_zoomable()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if zoom hotkeys are enabled, `false` otherwise |

<hr/>

### Examples

```lua lineNumbers title="Check if zoom hotkeys are enabled on a webview"
local browser = webview.create()

engine.print("info", "Webview zoomable:", browser:is_zoomable())
```
