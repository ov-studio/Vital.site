---
title: webview:focus
description: Focuses the webview, directing keyboard input to it
badge: Client
---

### Syntax

```lua
local status = webview:focus()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
view:focus()
```
