---
title: webview:is_autoplay
description: Retrieves whether autoplay is enabled on the webview
badge: Client
---

### Syntax

```lua
local result = webview:is_autoplay()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if autoplay is enabled, `false` otherwise |

<hr/>

### Examples

```lua
engine.print("info", "Webview autoplay:", view:is_autoplay())
```
