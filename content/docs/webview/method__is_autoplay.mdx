---
title: self:is_autoplay
description: Retrieves whether autoplay is enabled on the webview
badge: Client
---

### Syntax

```lua
local result = self:is_autoplay()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if autoplay is enabled, `false` otherwise |

<hr/>

### Examples

```lua lineNumbers title="Check if autoplay is enabled on a webview"
local browser = webview.create()

engine.print("info", "Webview autoplay:", browser:is_autoplay())
```
