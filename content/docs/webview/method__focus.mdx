---
title: self:focus
description: Focuses the webview, directing keyboard input to it
badge: Client
---

### Syntax

```lua
local status = self:focus()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Focus a webview to capture keyboard input"
local browser = webview.create()

browser:focus()
```
