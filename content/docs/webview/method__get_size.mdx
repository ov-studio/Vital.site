---
title: self:get_size
description: Retrieves the current size of the webview
badge: Client
---

### Syntax

```lua
local size = self:get_size()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `vector2` | size | Size of the webview as `{width, height}` |

---

### Examples

```lua lineNumbers title="Get the size of a webview"
local entity = core.webview.create()
local size = entity:get_size()

core.engine.print("info", "Webview size:", util.table.unpack(size))
```
