---
title: self:load_url
description: Loads a URL in the webview
badge: Client
---

### Syntax

```lua
local status = self:load_url(
    url
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | url | URL to load in the webview |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Load a URL in a webview"
local wv = webview.create()

wv:load_url("https://google.com")
```
