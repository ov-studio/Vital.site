---
title: self:eval
description: Evaluates a JavaScript expression inside the webview
badge: Client
---

### Syntax

```lua
local status = self:eval(
    input
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | input | JavaScript code to evaluate inside the webview |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Evaluate JavaScript expressions inside a webview"
local entity = core.webview.create()

-- Change the page background color
entity:eval("document.body.style.background = 'red'")

-- Post a message back to Lua
entity:eval("window.ipc.postMessage('hello from JS')")
```
