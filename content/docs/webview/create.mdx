title: core.webview.create
description: Creates a new webview instance
badge: Client
---

### Syntax

```lua
local instance = core.webview.create(
    options = {}
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `table` | options | Optional configuration table<br/>Refer [**Options**](#options) section |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `webview` | instance | Created webview instance |

---

### Options

All fields are optional and fall back to their defaults if not specified.

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `fullscreen` | `bool` | `true` | Webview will always match the size of the viewport |
| `transparent` | `bool` | `true` | Webview background will be transparent |
| `incognito` | `bool` | `true` | Runs the webview in incognito mode, no data is persisted to disk |
| `autoplay` | `bool` | `false` | Media can be played without user interaction |
| `zoomable` | `bool` | `false` | Enables page zooming hotkeys |
| `forward_input` | `bool` | `false` | Allows the webview to become the global input forwarder, forwarding keyboard and mouse input through to the game |

---

### Examples

```lua lineNumbers title="Create a webview with default options"
local entity = core.webview.create()

entity:set_visible(true)
entity:load_url("https://google.com")
```

```lua lineNumbers title="Create a webview with custom options"
local entity = core.webview.create({
    fullscreen = true,
    transparent = true,
    incognito = false,
    autoplay = false,
    zoomable = false,
    forward_input = true
})

entity:set_visible(true)
entity:load_url("https://google.com")
```
