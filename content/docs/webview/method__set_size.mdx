---
title: self:set_size
description: Sets the size of the webview
badge: Client
---

### Syntax

```lua
local status = self:set_size(size)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `vector2` | size | Size to apply to the webview, as `{x, y}` |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Set the size of a webview"
local browser = webview.create()

--Set webview to 800x600
browser:set_size({800, 600})
```
