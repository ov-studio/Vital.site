---
title: self:set_size
description: Sets the size of the webview
badge: Client
---

### Syntax

```lua
local status = self:set_size(
    size
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `vector2` | size | Size to apply to the webview, as `{width, height}` |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set the size of a webview"
local entity = webview.create()

-- Set webview to 800x600
entity:set_size({800, 600})
```
