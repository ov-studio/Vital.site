---
title: self:is_devtools_visible
description: Retrieves whether the developer tools panel is currently open
badge: Client
---

### Syntax

```lua
local result = self:is_devtools_visible()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if devtools is open, `false` otherwise |

<hr/>

### Examples

```lua lineNumbers title="Check if devtools is open on a webview"
local browser = webview.create()

engine.print("info", "Devtools open:", browser:is_devtools_visible())
```
