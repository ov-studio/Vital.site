---
title: self:is_visible
description: Retrieves whether the webview is currently visible
badge: Client
---

### Syntax

```lua
local result = self:is_visible()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if the webview is visible, `false` otherwise |

<hr/>

### Examples

```lua lineNumbers title="Check if a webview is currently visible"
local wv = webview.create()

engine.print("info", "Webview visible:", wv:is_visible())
```
