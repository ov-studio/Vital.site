---
title: self:set_message_handler
description: Registers a handler function to receive messages posted from the webview
badge: Client
---

### Syntax

```lua
local status = self:set_message_handler(
    handler
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `function` | handler | Callback function invoked when the webview posts a message via `window.ipc.postMessage()` |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Register a message handler on a webview"
local browser = webview.create()

browser:set_message_handler(function(message)
    engine.print("info", "Received message:", message)
end)
```
