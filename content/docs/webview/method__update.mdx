---
title: self:update
description: Refreshes the webview's size and visibility state
badge: Client
---

### Syntax

```lua
local status = self:update()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Update a webview after resizing"
local wv = webview.create()

--Apply after repositioning or resizing
wv:set_size(1280, 720)
wv:update()
```
