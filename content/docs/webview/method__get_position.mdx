---
title: self:get_position
description: Retrieves the current position of the webview
badge: Client
---

### Syntax

```lua
local position = self:get_position()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `vector2` | position | Position of the webview as `{x, y}` |

<hr/>

### Examples

```lua lineNumbers title="Print the position of a webview"
local browser = webview.create()
local position = browser:get_position()

engine.print("info", "Webview position:", position[1], position[2])
```
