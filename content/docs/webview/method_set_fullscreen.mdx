---
title: self:set_fullscreen
description: Toggles fullscreen mode on the webview
badge: Client
---

### Syntax

```lua
local status = self:set_fullscreen(state)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `bool` | state | Determines the webview's fullscreen state:<br/>• When `true` - the webview fills the entire window<br/>• When `false` - the webview uses its defined size |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
local browser = webview.create()

--Enable fullscreen
browser:set_fullscreen(true)

--Disable fullscreen
browser:set_fullscreen(false)
```
