---
title: self:is_fullscreen
description: Retrieves whether the webview is in fullscreen mode
badge: Client
---

### Syntax

```lua
local result = self:is_fullscreen()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if the webview is fullscreen, `false` otherwise |

<hr/>

### Examples

```lua lineNumbers title="Check if a webview is in fullscreen mode"
local browser = webview.create()

engine.print("info", "Webview fullscreen:", browser:is_fullscreen())
```
