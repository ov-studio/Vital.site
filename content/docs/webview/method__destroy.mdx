---
title: self:destroy
description: Destroys the webview instance and frees its memory
badge: Client
---

### Syntax

```lua
local status = self:destroy()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Destroy a webview instance"
local wv = webview.create()

wv:destroy()
```
