---
title: self:reload
description: Reloads the current page in the webview
badge: Client
---

### Syntax

```lua
local status = self:reload()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
view:reload()
```
