---
title: self:is_transparent
description: Retrieves whether the webview background is transparent
badge: Client
---

### Syntax

```lua
local result = self:is_transparent()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if the webview is transparent, `false` otherwise |

<hr/>

### Examples

```lua
engine.print("info", "Webview transparent:", view:is_transparent())
```
