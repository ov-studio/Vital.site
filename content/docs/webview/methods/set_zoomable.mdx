---
title: self:set_zoomable
description: Toggles zoom hotkeys on the webview
badge: Client
---

### Syntax

```lua
local status = self:set_zoomable(state)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `bool` | state | Determines the webview's zoom hotkey state:<br/>• When `true` - zoom hotkeys are enabled<br/>• When `false` - zoom hotkeys are disabled |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
--Enable zoom hotkeys
view:set_zoomable(true)

--Disable zoom hotkeys
view:set_zoomable(false)
```
