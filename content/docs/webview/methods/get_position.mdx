---
title: self:get_position
description: Retrieves the current position of the webview
badge: Client
---

### Syntax

```lua
local x, y = self:get_position()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | x | Horizontal position of the webview |
| `float` | y | Vertical position of the webview |

<hr/>

### Examples

```lua
local browser = webview.create()

local x, y = browser:get_position()
engine.print("info", "Position:", x, y)
```
