---
title: self:get_size
description: Retrieves the current size of the webview
badge: Client
---

### Syntax

```lua
local x, y = self:get_size()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | x | Width of the webview |
| `float` | y | Height of the webview |

<hr/>

### Examples

```lua
local browser = webview.create()

local w, h = browser:get_size()
engine.print("info", "Webview size:", w, h)
```
