---
title: self:set_transparent
description: Toggles background transparency on the webview
badge: Client
---

### Syntax

```lua
local status = self:set_transparent(state)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `bool` | state | Determines the webview's transparency:<br/>• When `true` - the background is transparent<br/>• When `false` - the background is opaque |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
local browser = webview.create()

--Enable transparency
browser:set_transparent(true)

--Disable transparency
browser:set_transparent(false)
```
