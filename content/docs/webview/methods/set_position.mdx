---
title: self:set_position
description: Sets the position of the webview
badge: Client
---

### Syntax

```lua
local status = self:set_position(x, y)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | x | Horizontal position to move the webview to |
| `float` | y | Vertical position to move the webview to |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
local browser = webview.create()

--Move webview to top-left corner
browser:set_position(0, 0)

--Move webview to a custom position
browser:set_position(100, 200)
```
