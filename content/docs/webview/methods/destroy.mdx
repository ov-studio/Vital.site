---
title: self:destroy
description: Destroys the webview instance and frees its memory
badge: Client
---

### Syntax

```lua
local status = self:destroy()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
local view = webview.create()
view:destroy()
```
