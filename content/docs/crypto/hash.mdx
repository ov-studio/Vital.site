---
title: crypto.hash
description: Retrieves computed cryptographic hash of a string
badge: Shared
---

### Syntax

```lua
local hash = crypto.hash(
    mode,
    input
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | mode | Hash mode<br/>Refer [**Modes**](#modes) section |
| `string` | input | String containing data to be hashed |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` / `bool` | hash | Computed hash on successful execution, or `false` on failure |

<hr/>

### Modes

| Mode | Description |
|------|-------------|
| `"SHA1"` | Secure Hash Algorithm 1 |
| `"SHA224"` | Secure Hash Algorithm 224-bit |
| `"SHA256"` | Secure Hash Algorithm 256-bit |
| `"SHA384"` | Secure Hash Algorithm 384-bit |
| `"SHA512"` | Secure Hash Algorithm 512-bit |

<hr/>

### Examples

```lua lineNumbers title="Hashes an input string"
local input = "Hello World"

engine.print("info", "Input:", input)
engine.print("info", "Hash:", crypto.hash("SHA256", input))
```
