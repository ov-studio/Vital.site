---
title: crypto.encode
description: Encodes a buffer into equivalent base64 string
badge: Shared
---

### Syntax

```lua
local hash = crypto.encode(buffer)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | buffer | Buffer containing data to be encoded |


### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` or `bool` | hash | Encoded base64 string on successful execution, or `false` on failure |

<hr/>

### Example

```lua
local input = "Hello World"

engine.print("Input:", input)
engine.print("Encoded:", crypto.encode(input))
```
