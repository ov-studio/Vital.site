---
title: crypto.decrypt
description: Decodes a cryptographically encrypted string into buffer
badge: Shared
---

### Syntax

```lua
local buffer = crypto.decrypt(mode, hash, key, iv)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | mode | Encryption mode:<br/>• `"AES128"` - Advanced Encryption Standard 128-bit<br/>• `"AES192"` - Advanced Encryption Standard 192-bit<br/>• `"AES256"` - Advanced Encryption Standard 256-bit |
| `string` | hash | Cryptographic hash to be decrypted |
| `string` | key | Key to be used for decryption |
| `string` | iv | Generated initialization vector |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` or `bool` | buffer | Computed buffer from cryptographic hash on successful execution, or `false` on failure |

<hr/>

### Example

```lua
local key = "MySecretKey12345"
local hash, iv = crypto.encrypt("AES256", "Hello World", key)

engine.print("Input:", hash)
engine.print("Decrypted:", crypto.decrypt("AES256", hash, key, iv))
```
