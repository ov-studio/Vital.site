---
title: crypto.decode
description: Decodes a base64 string
badge: Shared
---

### Syntax

```lua
local result = crypto.decode(hash)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | hash | Base64 hash to be decoded |


### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` or `bool` | result | Decoded string on successful execution, or `false` on failure |

<hr/>

### Example

```lua
local input = "SGVsbG8gV29ybGQ="

engine.print("Input:", input)
engine.print("Decoded:", crypto.decode(input))
```
