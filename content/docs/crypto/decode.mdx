---
title: crypto.decode
description: Decodes a base64 string
badge: Shared
---

### Syntax

```lua
local result = crypto.decode(
    input
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | input | Encoded string to be decoded |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` \| `bool` | result | Decoded string on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Decodes an input string"
local input = "SGVsbG8gV29ybGQ="

engine.print("info", "Input:", input)
engine.print("info", "Decoded:", crypto.decode(input))
```
