---
title: self:is_type
description: Checks whether the instance belongs to the specified type
badge: Server
---

### Syntax

```lua
local result = self:is_type(
    name
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | name | Type to check against |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if the instance matches the given type, `false` otherwise |

---

### Examples

```lua lineNumbers title="Check the type of a db_query instance"
local db = core.database.create("127.0.0.1", "root", "", "vital_sandbox")
local query = db:table("players")

core.engine.print("info", query:is_type("db_query")) -- true
core.engine.print("info", query:is_type("database")) -- false
```
