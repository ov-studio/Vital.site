---
title: sdfgi.set_max_distance
description: The maximum draw distance to be set for the SDFGI post-processing effect
badge: Client
---

### Syntax

```lua
local status = sdfgi.set_max_distance(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Maximum distance value to be applied<br/>Defines how far from the camera SDFGI global illumination is computed. Larger values cover greater distances but require more cascades to maintain detail |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Extend GI coverage for large outdoor scenes"
sdfgi.set_max_distance(500.0)
```
