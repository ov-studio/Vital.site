---
title: gfx.sdfgi.set_y_scale
description: The Y-axis scale mode to be set for the SDFGI post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.sdfgi.set_y_scale(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `enum` | value | Y scale mode to be applied<br/>Refer [**gfx.sdfgi.y_scale**](/docs/sdfgi/enum/y_scale) enum |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set tighter vertical packing for a tall vertical scene"
gfx.sdfgi.set_y_scale(gfx.sdfgi.y_scale.PERCENT_50)
```
