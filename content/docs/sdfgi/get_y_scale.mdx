---
title: gfx.sdfgi.get_y_scale

badge: Client
---

### Syntax

```lua
local result = gfx.sdfgi.get_y_scale()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| [`gfx.sdfgi.y_scale`](/docs/sdfgi/enum/y_scale) | result | Current Y scale mode |

---

### Examples

```lua lineNumbers title="Print current SDFGI Y scale mode"
core.engine.print("info", "SDFGI Y scale:", gfx.sdfgi.get_y_scale())
```
