---
title: sdfgi.get_y_scale
description: Retrieves the current Y-axis scale mode of the SDFGI post-processing effect
badge: Client
---

### Syntax

```lua
local result = sdfgi.get_y_scale()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `int` | result | Current Y scale mode |

<hr/>

### Examples

```lua lineNumbers title="Print current SDFGI Y scale mode"
engine.print("info", "SDFGI Y scale:", sdfgi.get_y_scale())
```
