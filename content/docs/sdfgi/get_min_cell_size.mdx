---
title: sdfgi.get_min_cell_size
description: Retrieves the current minimum cell size of the SDFGI post-processing effect
badge: Client
---

### Syntax

```lua
local result = sdfgi.get_min_cell_size()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current minimum cell size |

<hr/>

### Examples

```lua lineNumbers title="Print current SDFGI minimum cell size"
engine.print("info", "SDFGI min cell size:", sdfgi.get_min_cell_size())
```
