---
title: sdfgi.get_bounce_feedback
description: Retrieves the current bounce feedback strength of the SDFGI post-processing effect
badge: Client
---

### Syntax

```lua
local result = sdfgi.get_bounce_feedback()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current bounce feedback value |

<hr/>

### Examples

```lua lineNumbers title="Print current SDFGI bounce feedback"
engine.print("info", "SDFGI bounce feedback:", sdfgi.get_bounce_feedback())
```
