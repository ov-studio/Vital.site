---
title: sdfgi.get_max_distance
description: Retrieves the current maximum draw distance of the SDFGI post-processing effect
badge: Client
---

### Syntax

```lua
local result = sdfgi.get_max_distance()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current maximum distance |

<hr/>

### Examples

```lua lineNumbers title="Print current SDFGI max distance"
engine.print("info", "SDFGI max distance:", sdfgi.get_max_distance())
```
