---
title: sdfgi.get_normal_bias
description: Retrieves the current normal bias of the SDFGI post-processing effect
badge: Client
---

### Syntax

```lua
local result = sdfgi.get_normal_bias()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current normal bias value |

<hr/>

### Examples

```lua lineNumbers title="Print current SDFGI normal bias"
engine.print("info", "SDFGI normal bias:", sdfgi.get_normal_bias())
```
