---
title: gfx.sdfgi.get_probe_bias
description: Retrieves the current probe bias of the SDFGI post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.sdfgi.get_probe_bias()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current probe bias value |

<hr/>

### Examples

```lua lineNumbers title="Print current SDFGI probe bias"
engine.print("info", "SDFGI probe bias:", gfx.sdfgi.get_probe_bias())
```
