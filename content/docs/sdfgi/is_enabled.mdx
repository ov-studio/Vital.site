---
title: sdfgi.is_enabled
description: Retrieves whether the SDFGI post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local result = sdfgi.is_enabled()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if SDFGI is enabled, `false` otherwise |

<hr/>

### Examples

```lua lineNumbers title="Print whether SDFGI is enabled"
engine.print("info", "SDFGI enabled:", sdfgi.is_enabled())
```
