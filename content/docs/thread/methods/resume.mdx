---
title: self:resume
description: Resumes an existing suspended thread
badge: Shared
---

### Syntax

```lua
local status = self:resume(options = false)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `table` | options | Options containing the following fields:<br/>• `executions` (number) - number of executions per resume<br/>• `interval` (number) - resume interval in milliseconds |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | Returns `true` on successful execution, or `false` on failure |

<hr/>

### Example

```lua
--Resumes only once
local self = thread:create(function(self)
    for i = 1, 5, 1 do
        engine.print("Thread iteration count", i)
    end
end)

self:resume()
```

```lua
--Automatically resumes once every 1 second
local self = thread:create(function(self)
    for i = 1, 5, 1 do
        thread:pause()
        engine.print("Thread iteration count", i)
    end
end)

self:resume({executions = 1, interval = 1000})
```
