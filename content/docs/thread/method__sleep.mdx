---
title: self:sleep
description: Puts an existing thread on sleep
badge: Shared
---

### Syntax

```lua
local status = self:sleep(
    duration
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `int` | duration | Sleep duration in milliseconds |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Sleep a thread for 5 seconds then continue"
local self = thread:create(function(self)
    engine.print("info", "Thread sleeping for 5 seconds")
    self:sleep(5000)
    engine.print("info", "Thread awaken now")
end)

self:resume()
```
