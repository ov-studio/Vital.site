---
title: thread.create
description: Creates a new thread instance
badge: Shared
---

### Syntax

```lua
local ctx = thread.create(
    exec
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `function` | exec | Function to be executed by the thread |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `thread` \| `bool` | ctx | Created `thread` instance on successful execution, or `false` on failure |

---

### Callback: `exec`

```lua
exec(self)
```

| Type | Name | Description |
|------|------|-------------|
| `thread` | self | Running thread instance |

---

### Examples

```lua lineNumbers title="Create and resume a basic thread"
local ctx = thread.create(function(self)
    engine.print("info", "Hello from thread")
end)

ctx:resume()
```

```lua lineNumbers title="Create a thread and sleep inside it"
local ctx = thread.create(function(self)
    engine.print("info", "Sleeping for 2 seconds")
    self:sleep(2000)
    engine.print("info", "Awake now")
end)

ctx:resume()
```
