---
title: util.thread.create
description: Creates a new thread instance with specified function in suspended state
badge: Shared
---

### Syntax

```lua
local instance = util.thread.create(
    exec
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `function` | exec | Function to be executed by the thread |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `thread` \| `bool` | instance | Created `thread` instance on successful execution, `false` otherwise |

---

### Callback: `exec`

```lua
exec(self)
```

| Type | Name | Description |
|------|------|-------------|
| `thread` | self | Running thread instance |

---

### Examples

```lua lineNumbers title="Create and resume a basic thread"
local entity = util.thread.create(function(self)
    engine.print("info", "Hello from thread")
end)

entity:resume()
```

```lua lineNumbers title="Create a thread and sleep inside it"
local entity = util.thread.create(function(self)
    engine.print("info", "Sleeping for 2 seconds")
    self:sleep(2000)
    engine.print("info", "Awake now")
end)

entity:resume()
```
