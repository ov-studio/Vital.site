---
title: thread:create
description: Creates a new thread context
badge: Shared
---

### Syntax

```lua
local ctx = thread:create(exec)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `function` | exec | Function to be executed by the thread |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `thread` or `bool` | ctx | Returns `thread` instance on successful execution, or `false` on failure |

<hr/>

### Callback

```lua
exec(self)
```

| Type | Name | Description |
|------|------|-------------|
| `thread` | self | Running thread instance |

<hr/>

### Examples

```lua
local self = thread:create(function(self)
    engine.print("Hello from thread")
end)

self:resume()
```
