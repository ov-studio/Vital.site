---
title: thread:create
description: Creates a new thread context
badge: Shared
---

### Syntax

```lua
local ctx = thread:create(
    exec
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `function` | exec | Function to be executed by the thread |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `thread` \| `bool` | ctx | `thread` instance on successful execution, or `false` on failure |

<hr/>

### Callback: `exec`

```lua
exec(self)
```

| Type | Name | Description |
|------|------|-------------|
| `thread` | self | Running thread instance |

<hr/>

### Examples

```lua lineNumbers title="Create and resume a basic thread"
local self = thread:create(function(self)
    engine.print("info", "Hello from thread")
end)

self:resume()
```
