---
title: self:get_type
description: Returns the type name of the instance
badge: Shared
---

### Syntax

```lua
local result = self:get_type()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` \| `bool` | result | Type of the instance, or `false` if unavailable |

---

### Examples

```lua lineNumbers title="Get the type of a thread instance"
local ctx = thread.create(function(self)
    engine.print("info", self:get_type()) --'thread'
end)

ctx:resume()
```
