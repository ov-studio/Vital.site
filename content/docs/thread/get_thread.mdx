---
title: thread:get_thread
description: Retrieves currently running thread
badge: Shared
---

### Syntax

```lua
local ctx = thread:get_thread()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `thread` / `bool` | ctx | Currently running `thread` instance, or `false` if none is active |

<hr/>

### Examples

```lua lineNumbers title="Get the currently running thread"
engine.print("info", "Running thread:", thread:get_thread())
```
