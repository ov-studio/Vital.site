---
title: thread:pause
description: Pauses currently running thread
badge: Shared
---

### Syntax

```lua
local ctx = thread:pause()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | ctx | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Pause a thread then resume it manually"
local self = thread:create(function(self)
    engine.print("info", "Paused thread")
    thread:pause()
    engine.print("info", "Hello from thread")
end)

self:resume()
```
