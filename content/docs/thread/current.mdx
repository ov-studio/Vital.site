---
title: thread.current
description: Retrieves the currently running thread instance
badge: Shared
---

### Syntax

```lua
local ctx = thread.current()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `thread` \| `bool` | ctx | Currently running `thread` instance, or `false` if no thread is active |

---

### Examples

```lua lineNumbers title="Retrieve the currently running thread from within a thread"
local ctx = thread.create(function(self)
    local running = thread.current()
    engine.print("info", "Running thread:", running)
end)

ctx:resume()
```

```lua lineNumbers title="Check whether code is running inside a thread"
local running = thread.current()

if not running then
    engine.print("info", "Not inside a thread")
end
```
