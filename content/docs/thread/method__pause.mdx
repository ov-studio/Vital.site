---
title: self:pause
description: Yields the currently running thread, suspending execution until it is resumed
badge: Shared
---

### Syntax

```lua
self:pause()
```

<Callout type="info">
  **This function must be called from within a running thread.** Calling it outside of a thread context has no effect and returns `false`.

  - **Inside a thread** — execution is suspended at the call site and resumes from that point when [`self:resume`](/docs/thread/method/resume) is called
  - **Outside a thread** — the call is a no-op and returns `false`
</Callout>

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

---

### Examples

```lua lineNumbers title="Pause a thread then resume it manually"
local ctx = thread.create(function(self)
    engine.print("info", "Before pause")
    self:pause()
    engine.print("info", "After resume")
end)

ctx:resume()
ctx:resume()
```
