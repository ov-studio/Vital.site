---
title: self:status
description: Retrieves the status of an existing thread
badge: Shared
---

### Syntax

```lua
local status = self:status()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` | status | Thread's current status |

<hr/>

### Examples

```lua lineNumbers title="Get the current thread status"
local self = thread:create(function(self)
    engine.print("info", "Thread status:", self:status())
end)

self:resume()
```
