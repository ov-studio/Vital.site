---
title: self:destroy
description: Destroys the thread instance and frees its resources
badge: Shared
---

### Syntax

```lua
local status = self:destroy()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Destroy a thread instance"
local ctx = thread.create(function(self)
    engine.print("info", "Hello from thread")
end)

ctx:resume()
ctx:destroy()
```
