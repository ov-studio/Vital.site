---
title: self:resume
description: Resumes a suspended thread instance
badge: Shared
---

### Syntax

```lua
local status = self:resume()
```

<Callout type="info">
  **A thread cannot be resumed while it is sleeping or awaiting a promise.** Both states must be cleared before `self:resume` can run successfully.

  - **Sleeping** — the thread is currently inside a [`self:sleep`](/docs/thread/method/sleep) call and has not yet woken up
  - **Awaiting** — the thread is currently suspended inside a [`self:await`](/docs/thread/method/await) call waiting for a promise to settle
</Callout>

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

---

### Examples

```lua lineNumbers title="Create and resume a basic thread"
local ctx = thread.create(function(self)
    engine.print("info", "Hello from thread")
end)

ctx:resume()
```

```lua lineNumbers title="Attempt to resume a sleeping thread"
local ctx = thread.create(function(self)
    self:sleep(5000)
end)

ctx:resume()
engine.print("info", ctx:resume()) --false
```
