---
title: discord.is_connected
description: Retrieves whether the Discord client is currently authenticated and connected
badge: Client
---

### Syntax

```lua
local result = discord.is_connected()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if the Discord client is authenticated and connected, `false` otherwise |

<hr/>

### Examples

```lua
engine.print("info", "Discord connected:", discord.is_connected())
```
