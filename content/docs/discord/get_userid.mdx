---
title: discord.get_userid
description: Retrieves the userid of the currently connected Discord account
badge: Client
---

### Syntax

```lua
local result = discord.get_userid()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` | result | Userid of the connected Discord account, or `false` if not connected |

<hr/>

### Examples

```lua lineNumbers title="Get the userid of the connected discord account"
local userid = discord.get_userid()

engine.print("info", "Discord user ID:", userid)
```
