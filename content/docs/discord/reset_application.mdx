---
title: discord.reset_application
description: Resets the Discord client to the default application-id and activity
badge: Client
---

### Syntax

```lua
local status = discord.reset_application()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Reset the discord client to its default application"
discord.reset_application()
```
