---
title: discord.set_activity
description: Sets the Discord rich presence activity state and details
badge: Client
---

### Syntax

```lua
local status = discord.set_activity(state, details)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | state | The activity state text shown in the presence |
| `string` | details | The activity details text shown in the presence |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
--Set activity
discord.set_activity("In a match", "Playing ranked")
```
