---
title: discord.get_username
description: Retrieves the username of the currently connected Discord account
badge: Client
---

### Syntax

```lua
local result = discord.get_username()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` | result | Username of the connected Discord account, or `false` if not connected |

<hr/>

### Examples

```lua lineNumbers title="Get the username of the connected discord account"
local username = discord.get_username()

engine.print("info", "Discord username:", username)
```
