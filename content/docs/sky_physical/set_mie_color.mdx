---
title: gfx.sky.physical.set_mie_color
description: Sets the Mie scattering color of the physical sky material
badge: Client
---

### Syntax

```lua
local status = gfx.sky.physical.set_mie_color(
    color
)
```

```lua
local status = gfx.sky.physical.set_mie_color(
    hex
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `color` | color | Mie scattering color as `{r, g, b, a}` |
| `string` | hex | Alternatively, a valid HTML hex color string (e.g. `"#ffffff"`) |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set physical sky Mie color using a color table"
gfx.sky.physical.set_mie_color({0.69, 0.86, 1.0, 1.0})
```

```lua lineNumbers title="Set physical sky Mie color using a hex string"
gfx.sky.physical.set_mie_color("#b0dcff")
```
