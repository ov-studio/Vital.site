---
title: gfx.sky.physical.set_mie_coefficient
description: Sets the Mie scattering coefficient of the physical sky material
badge: Client
---

### Syntax

```lua
local status = gfx.sky.physical.set_mie_coefficient(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Mie scattering coefficient to apply |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set physical sky Mie coefficient"
gfx.sky.physical.set_mie_coefficient(0.005)
```
