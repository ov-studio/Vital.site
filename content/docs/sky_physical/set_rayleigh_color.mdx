---
title: gfx.sky.physical.set_rayleigh_color
description: Sets the Rayleigh scattering color of the physical sky material
badge: Client
---

### Syntax

```lua
local status = gfx.sky.physical.set_rayleigh_color(
    color
)
```

```lua
local status = gfx.sky.physical.set_rayleigh_color(
    hex
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `color` | color | Rayleigh scattering color as `{r, g, b, a}` |
| `string` | hex | Alternatively, a valid HTML hex color string (e.g. `"#ffffff"`) |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set physical sky Rayleigh color using a color table"
gfx.sky.physical.set_rayleigh_color({0.26, 0.41, 0.58, 1.0})
```

```lua lineNumbers title="Set physical sky Rayleigh color using a hex string"
gfx.sky.physical.set_rayleigh_color("#426894")
```
