---
title: gfx.sky.physical.get_rayleigh_color
description: Retrieves the current Rayleigh scattering color of the physical sky material
badge: Client
---

### Syntax

```lua
local color = gfx.sky.physical.get_rayleigh_color()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `color` | color | Current Rayleigh scattering color as `{r, g, b, a}` |

---

### Examples

```lua lineNumbers title="Print current physical sky Rayleigh color"
core.engine.print("info", "Rayleigh color:", util.table.unpack(gfx.sky.physical.get_rayleigh_color()))
```
