---
title: gfx.environment.get_tonemap_white
description: Retrieves the current tonemap white point of the environment
badge: Client
---

### Syntax

```lua
local result = gfx.environment.get_tonemap_white()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current tonemap white point value |

---

### Examples

```lua lineNumbers title="Print current environment tonemap white point"
core.engine.print("info", "Tonemap white:", gfx.environment.get_tonemap_white())
```
