---
title: gfx.env.set_ambient_source
description: Sets the ambient light source of the environment
badge: Client
---

### Syntax

```lua
local status = gfx.env.set_ambient_source(
    source
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `int` | source | Ambient light source to apply<br/>Refer [**gfx.env.ambient_source**](/docs/environment/enum/ambient_source) enum |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set environment ambient source to background"
gfx.env.set_ambient_source(gfx.env.ambient_source.BG)
```
