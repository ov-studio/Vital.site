---
title: gfx.env.set_ambient_source
description: Sets the ambient light source of the environment
badge: Client
---

### Syntax

```lua
local result = gfx.env.set_ambient_source(source)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `int` | source | Ambient light source to apply |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set environment ambient source"
gfx.env.set_ambient_source(0)
```
