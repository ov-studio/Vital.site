---
title: gfx.env.set_background_energy
description: Sets the background energy multiplier of the environment
badge: Client
---

### Syntax

```lua
local status = gfx.env.set_background_energy(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Background energy multiplier to apply |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set environment background energy"
gfx.env.set_background_energy(1.0)
```
