---
title: gfx.env.get_ambient_color
description: Retrieves the current ambient light color of the environment
badge: Client
---

### Syntax

```lua
local result = gfx.env.get_ambient_color()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `color` | result | Ambient light color as `{r, g, b, a}` |

---

### Examples

```lua lineNumbers title="Print current environment ambient color"
local color = gfx.env.get_ambient_color()

core.engine.print("info", "Ambient color:", color[1], color[2], color[3], color[4])
```
