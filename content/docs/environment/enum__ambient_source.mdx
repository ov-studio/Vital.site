---
title: gfx.env.ambient_source
description: Enumeration of supported ambient light sources for the environment
badge: Client
---

### Values

| Constant | Description |
|----------|-------------|
| `gfx.env.ambient_source.BG` | Derives ambient light from the current background — see [`gfx.env.set_background_mode`](/docs/environment/set_background_mode) |
| `gfx.env.ambient_source.DISABLED` | Disables ambient light entirely |
| `gfx.env.ambient_source.COLOR` | Uses a flat custom color as the ambient light source — see [`gfx.env.set_ambient_color`](/docs/environment/set_ambient_color) |
| `gfx.env.ambient_source.SKY` | Derives ambient light from the configured sky |
