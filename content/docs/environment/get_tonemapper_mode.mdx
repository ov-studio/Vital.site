---
title: gfx.env.get_tonemapper_mode
description: Retrieves the current tonemapper mode of the environment
badge: Client
---

### Syntax

```lua
local result = gfx.env.get_tonemapper_mode()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| [`gfx.env.tonemapper_mode`](/docs/environment/enum/tonemapper_mode) | result | Current tonemapper mode |

---

### Examples

```lua lineNumbers title="Print current environment tonemapper mode"
core.engine.print("info", "Tonemapper mode:", gfx.env.get_tonemapper_mode())
```
