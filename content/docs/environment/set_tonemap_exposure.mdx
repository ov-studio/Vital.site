---
title: gfx.env.set_tonemap_exposure
description: Sets the tonemap exposure of the environment
badge: Client
---

### Syntax

```lua
local result = gfx.env.set_tonemap_exposure(value)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Tonemap exposure to apply |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set environment tonemap exposure"
gfx.env.set_tonemap_exposure(1.0)
```
