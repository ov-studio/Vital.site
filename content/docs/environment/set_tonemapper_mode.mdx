---
title: gfx.env.set_tonemapper_mode
description: Sets the tonemapper mode of the environment
badge: Client
---

### Syntax

```lua
local status = gfx.env.set_tonemapper_mode(
    mode
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `int` | mode | Tonemapper mode to apply<br/>Refer [**gfx.env.tonemapper_mode**](/docs/environment/enum/tonemapper_mode) enum |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set environment tonemapper to linear"
gfx.env.set_tonemapper_mode(gfx.env.tonemapper_mode.LINEAR)
```
