---
title: gfx.env.set_tonemapper_mode
description: Sets the tonemapper mode of the environment
badge: Client
---

### Syntax

```lua
local status = gfx.env.set_tonemapper_mode(
    mode
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `int` | mode | Tonemapper mode to apply |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set environment tonemapper mode"
gfx.env.set_tonemapper_mode(0)
```
