---
title: gfx.environment.set_ambient_energy
description: Sets the ambient light energy of the environment
badge: Client
---

### Syntax

```lua
local result = gfx.environment.set_ambient_energy(value)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Ambient light energy to apply |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if the ambient energy was set successfully |

---

### Examples

```lua lineNumbers title="Set environment ambient energy"
gfx.environment.set_ambient_energy(1.0)
```
