---
title: gfx.env.set_background_mode
description: Sets the background mode of the environment
badge: Client
---

### Syntax

```lua
local status = gfx.env.set_background_mode(
    mode
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| [`gfx.env.background_mode`](/docs/environment/enum/background_mode) | mode | Background mode to apply |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set environment background mode to sky"
gfx.env.set_background_mode(gfx.env.background_mode.SKY)
```
