---
title: gfx.environment.get_background_color
description: Retrieves the current background color of the environment
badge: Client
---

### Syntax

```lua
local color = gfx.environment.get_background_color()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `color` | color | Background color as `{r, g, b, a}` |

---

### Examples

```lua lineNumbers title="Print current environment background color"
core.engine.print("info", "Background color:", util.table.unpack(gfx.environment.get_background_color()))
```
