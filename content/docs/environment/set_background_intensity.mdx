---
title: gfx.env.set_background_intensity
description: Sets the background luminance intensity of the environment
badge: Client
---

### Syntax

```lua
local result = gfx.env.set_background_intensity(value)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Background luminance intensity to apply |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set environment background intensity"
gfx.env.set_background_intensity(30000.0)
```
