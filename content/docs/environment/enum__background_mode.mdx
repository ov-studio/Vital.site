---
title: gfx.env.background_mode
description: Enumeration of supported environment background modes
badge: Client
---

### Values

| Constant | Description |
|----------|-------------|
| `gfx.env.background_mode.CLEAR_COLOR` | Uses the renderer's default clear color as the background |
| `gfx.env.background_mode.COLOR` | Uses a flat custom color as the background<br/>Refer [`gfx.env.set_background_color`](/docs/environment/set_background_color) |
| `gfx.env.background_mode.SKY` | Renders the configured sky as the background |
