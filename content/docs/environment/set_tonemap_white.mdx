---
title: gfx.env.set_tonemap_white
description: Sets the tonemap white point of the environment
badge: Client
---

### Syntax

```lua
local result = gfx.env.set_tonemap_white(value)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Tonemap white point to apply |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set environment tonemap white point"
gfx.env.set_tonemap_white(1.0)
```
