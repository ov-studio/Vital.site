---
title: gfx.env.reflection_source
description: Enumeration of supported reflection sources for the environment
badge: Client
---

### Values

| Constant | Description |
|----------|-------------|
| `gfx.env.reflection_source.BG` | Derives reflections from the current background<br/>Refer [`gfx.env.set_background_mode`](/docs/environment/set_background_mode) |
| `gfx.env.reflection_source.DISABLED` | Disables environment reflections entirely |
| `gfx.env.reflection_source.SKY` | Derives reflections from the configured sky |
