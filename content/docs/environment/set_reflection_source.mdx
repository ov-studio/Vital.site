---
title: gfx.env.set_reflection_source
description: Sets the reflection source of the environment
badge: Client
---

### Syntax

```lua
local status = gfx.env.set_reflection_source(
    source
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `int` | source | Reflection source to apply |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set environment reflection source"
gfx.env.set_reflection_source(0)
```
