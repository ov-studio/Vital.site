---
title: gfx.env.get_reflection_source
description: Retrieves the current reflection source of the environment
badge: Client
---

### Syntax

```lua
local result = gfx.env.get_reflection_source()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| [`gfx.env.reflection_source`](/docs/environment/enum/reflection_source) | result | Current reflection source |

---

### Examples

```lua lineNumbers title="Print current environment reflection source"
core.engine.print("info", "Reflection source:", gfx.env.get_reflection_source())
```
