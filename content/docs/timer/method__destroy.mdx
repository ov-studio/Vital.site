---
title: self:destroy
description: Destroys the timer instance and frees its resources
badge: Shared
---

### Syntax

```lua
local status = self:destroy()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Destroy a timer instance"
local entity = timer.create(function(count)
    engine.print("info", "Tick: "..engine.get_tick())
end, 500, 0)

entity:destroy()
```
