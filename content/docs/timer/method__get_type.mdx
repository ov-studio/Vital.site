---
title: self:get_type
description: Returns the type name of the instance
---

### Syntax

```lua
local result = self:get_type()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` / `bool` | result | Type of the instance, or `false` if unavailable |

<hr/>

### Examples

```lua lineNumbers title="Get the type of a timer instance"
local tm = timer.create(function(count)
    engine.print("info", "Counter:", count)
end, 1000, 1)

engine.print("info", tm:get_type()) --'timer'
```
