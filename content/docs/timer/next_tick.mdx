---
title: timer.next_tick
description: Schedules a function to execute on the next engine tick
badge: Shared
---

### Syntax

```lua
local status = timer.next_tick(exec)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `function` | exec | Function to be executed on the next engine tick |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` if the callback was successfully scheduled, `false` otherwise |

---

```lua lineNumbers title="Execute after resource starts"
resource.start("my_resource")

timer.next_tick(function()
    engine.print("info", "Resource running?:", resource.is_running("my_resource"))
end)
```
