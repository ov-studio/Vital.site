---
title: timer.create
description: Creates a new timer instance that executes specified function on a given interval
badge: Shared
---

### Syntax

```lua
local ctx = timer.create(exec, interval, executions)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `function` | exec | Function to be executed by the timer |
| `int` | interval | Interval in milliseconds between each execution<br/>Must be `>= 0` |
| `int` | executions | Number of times to execute (`0` means infinite)<br/>Must be `>= 0` |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `timer` / `bool` | ctx | `timer` instance on successful execution, or `false` on failure |

<hr/>

### Callback: `exec`

```lua
exec(...)
```

| Type | Name | Description |
|------|------|-------------|
| `...` | ... | Additional arguments |

```lua lineNumbers title="Create a one-shot timer"
local tm = timer.create(function(count)
    engine.print("info", "Counter:", count)
end, 1000, 1)
```

```lua lineNumbers title="Create a repeating timer"
local tm = timer.create(function(count)
    engine.print("info", "Counter:", count)
end, 500, 0)
```

```lua lineNumbers title="Create a timer that fires 3 times then auto-destroys"
local tm = timer.create(function(count)
    engine.print("info", count.."/3")
end, 1000, 3)
```
