---
title: db_query:select
description: Specifies which columns to retrieve in a fetch query
badge: Server
---

### Syntax

```lua
local query = db_query:select(...)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | ... | One or more column names to include in the result<br/>If omitted, all columns are returned (`*`) |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `db_query` / `bool` | query | Query builder instance on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
local db = database.create("127.0.0.1", "root", "", "vital_sandbox")

--Fetch specific columns from the players table
local rows = db:table("players")
    :select("id", "name", "score")
    :fetch()

--Fetch all columns from the players table
local rows = db:table("players")
    :fetch()
```
