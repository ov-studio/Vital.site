---
title: db_query:insert
description: Appends a row insertion to the query
badge: Server
---

### Syntax
```lua
local query = db_query:insert(data)
```

<Callout type="info">
  **This function only prepares the query and does not execute it.** Chain [`db_query:execute`](/docs/database/method/execute) at the end to commit the query to the database.

  - **Data** — accepts a key-value table where each key is a column name and each value is the data to insert
  - **Chaining** — compatible with `:where` to conditionally apply the insert
  - **Multiple insertions** — chain multiple `:insert` calls to append additional row insertions
  - **Multiple conditions** — chain multiple `:where` calls to apply additional filters to the insertion
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `table` | data | Map of column names to the values to insert<br/>Accepted value types are `string` and `number` |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `db_query` / `bool` | query | Query builder instance on successful execution, or `false` on failure |

<hr/>

### Examples
```lua
local db = database.create("127.0.0.1", "root", "", "vital_sandbox")

--Insert a new player record
db:table("players")
    :insert({ name = "aviril", score = 0 })
    :execute()
```
