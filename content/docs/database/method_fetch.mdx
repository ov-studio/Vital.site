---
title: self:fetch
description: Retrieves all rows matching the current query from the table
badge: Server
---

### Syntax

```lua
local rows = self:fetch()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `table` | rows | Array of matching rows, where each row is a table of column name to value pairs |

<hr/>

### Examples

```lua
local db = database.create("127.0.0.1", "root", "", "vital_sandbox")

local rows = db:table("players"):select("id", "name", "score"):where("score", ">", 100):fetch()

for i = 1, #rows do
    engine.print("info", rows[i].id, rows[i].name, rows[i].score)
end
```
