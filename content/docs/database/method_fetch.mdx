---
title: db_query:fetch
description: Retrieves all rows matching the prepared query from the table
badge: Server
---

### Syntax

```lua
local rows = db_query:fetch()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `table` | rows | Array of matching rows, where each row is a table of column name to value pairs |

### Query Builders

| Builder | Description |
|---------|-------------|
| [`db_query:select`](/docs/database/method/select) | Specifies the columns to retrieve |
| [`db_query:where`](/docs/database/method/where) | Filters rows based on a given condition |
| [`db_query:insert`](/docs/database/method/insert) | Inserts a new row into the table |
| [`db_query:delete`](/docs/database/method/delete) | Deletes rows from the table |
| [`db_query:update`](/docs/database/method/update) | Updates existing rows in the table |

<hr/>

### Examples

```lua
local db = database.create("127.0.0.1", "root", "", "vital_sandbox")

--Fetch all players with a score greater than 100
local rows = db:table("players")
    :select("id", "name", "score")
    :where("score", ">", 100)
    :fetch()

--Iterate over the results and print each row
for i = 1, #rows do
    engine.print("info", rows[i].id, rows[i].name, rows[i].score)
end
```
