---
title: db_query:truncate
description: Removes all rows from the table
badge: Server
---

### Syntax

```lua
local promise = db_query:truncate()
```

<Callout type="info">
  **This function returns a promise and starts processing the query.** Use [`thread:await`](/docs/thread/method/await) inside a thread to retrieve the result.

  - **Inside a thread** — `self:await(db_query:truncate())` yields until the operation completes, then returns the resolved values
  - **Outside a thread** — the promise can still be created, but must be awaited from within a thread to retrieve the result
</Callout>

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `promise` | promise | Refer [**Settlements**](#settlements) section |

---

### Settlements

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on success, `false` on failure |
| `bool` | resolved | true |
| `string` | rejected | Error message describing the failure |

---

### Examples

```lua lineNumbers title="Truncate all rows from the 'players' table"
local db = database.create("127.0.0.1", "root", "", "vital_sandbox")

thread.create(function(self)
    local ok, err = self:await(db:table("players")
        :truncate())

    if not ok then
        engine.print("error", err)
    end
end):resume()
```
