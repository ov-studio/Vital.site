---
title: self:delete
description: Prepares a DELETE query
badge: Server
---

### Syntax

```lua
local query = self:delete()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `database_query` / `bool` | query | Query builder instance on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
local db = database.create("127.0.0.1", "root", "", "vital_sandbox")

db:table("players"):delete():where("name", "=", "aviril"):execute()
```
