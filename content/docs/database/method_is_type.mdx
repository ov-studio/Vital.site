---
title: self:is_type
description: Checks whether the instance belongs to the specified type
badge: Server
---

### Syntax

```lua
local result = self:is_type(name)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | name | Type to check against |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if the instance matches the given type, `false` otherwise |

<hr/>

### Examples

```lua
local db = database.create("127.0.0.1", "root", "", "vital_sandbox")

engine.print("info", db:is_type("database")) --true
engine.print("info", db:is_type("font")) --false
```
