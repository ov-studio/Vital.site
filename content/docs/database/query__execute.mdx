---
title: db_query:execute
description: Commits the prepared query on the database
badge: Server
---

### Syntax

```lua
local promise = db_query:execute()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `promise` | promise | Refer [**Settlements**](#settlements) section |

---

### Settlements

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on success, `false` on failure |
| `bool` | resolved | true |
| `string` | rejected | Error message describing the failure |

<hr/>

### Examples

```lua
local db = database.create("127.0.0.1", "root", "", "vital_sandbox")

thread.create(function(self)
    --Insert a new player record
    self:await(db:table("players")
        :insert({ name = "aviril", score = 0 })
        :execute())

    --Update the score for a specific player
    self:await(db:table("players")
        :update({ score = 500 })
        :where("name", "=", "aviril")
        :execute())

    --Delete a specific player record
    self:await(db:table("players")
        :delete()
        :where("name", "=", "aviril")
        :execute())
end):resume()
```
