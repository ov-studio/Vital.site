---
title: db_query:execute
description: Commits the prepared query on the database
badge: Server
---

### Syntax

```lua
local promise = db_query:execute()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `promise` | promise | Refer [**Settlements**](#settlements) section |

---

### Settlements

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on success, `false` on failure |
| `bool` | resolved | true |
| `string` | rejected | Error message describing the failure |

<hr/>

### Examples

```lua lineNumbers title="Insert a new player record"
local db = database.create("127.0.0.1", "root", "", "vital_sandbox")

thread.create(function(self)
    local ok, err = self:await(db:table("players")
        :insert({ name = "aviril", score = 0 })
        :execute())

    if not ok then
        engine.print("error", "Insert failed: "..err)
    end
end):resume()
```

```lua lineNumbers title="Update the score for a specific player"
local db = database.create("127.0.0.1", "root", "", "vital_sandbox")

thread.create(function(self)
    local ok, err = self:await(db:table("players")
        :update({ score = 500 })
        :where("name", "=", "aviril")
        :execute())

    if not ok then
        engine.print("error", "Update failed: "..err)
    end
end):resume()
```

```lua lineNumbers title="Delete a specific player record"
local db = database.create("127.0.0.1", "root", "", "vital_sandbox")

thread.create(function(self)
    local ok, err = self:await(db:table("players")
        :delete()
        :where("name", "=", "aviril")
        :execute())

    if not ok then
        engine.print("error", "Delete failed: "..err)
    end
end):resume()
```
