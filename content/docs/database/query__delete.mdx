---
title: db_query:delete
description: Appends a row deletion to the query
badge: Server
---

### Syntax
```lua
local query = db_query:delete()
```

<Callout type="info">
  **This function only prepares the query and does not execute it.** Chain [`db_query:execute`](/docs/database/query/execute) at the end to commit the query to the database.

  - **Chaining** — compatible with `:where` to target specific rows for deletion
  - **Multiple deletions** — chain multiple `:delete` calls to append additional row deletions
  - **Multiple conditions** — chain multiple `:where` calls to apply additional filters to the deletion
  - **Without conditions** — omitting `:where` will delete all rows in the table
</Callout>
### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `db_query` / `bool` | query | Query builder instance on successful execution, or `false` on failure |

<hr/>

### Examples
```lua
local db = database.create("127.0.0.1", "root", "", "vital_sandbox")

--Delete a specific player record
db:table("players")
    :delete()
    :where("name", "=", "aviril")
    :execute()
```
