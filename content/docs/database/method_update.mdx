---
title: self:update
description: Prepares an UPDATE query with the given column values
badge: Server
---

### Syntax

```lua
local query = self:update(data)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `table` | data | Map of column names to their new values<br/>Values can be `string` or `number` |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `database_query` / `bool` | query | Query builder instance on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
local db = database.create("127.0.0.1", "root", "", "vital_sandbox")

--Update the score for a specific player
db:table("players")
    :update({ score = 500 })
    :where("name", "=", "aviril")
    :execute()
```
