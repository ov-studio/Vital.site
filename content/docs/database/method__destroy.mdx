---
title: self:destroy
description: Destroys the database instance, closes the connection and frees its resources
badge: Server
---

### Syntax

```lua
local status = self:destroy()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

---

### Examples

```lua lineNumbers title="Destroy a database instance"
local db = database.create("127.0.0.1", "root", "", "vital_sandbox")

db:destroy()
```
