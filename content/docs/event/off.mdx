---
title: event.off
description: Unregisters a previously registered handler from the specified event
badge: Shared
---

### Syntax

```lua
local status = event.off(
    name,
    exec
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | name | Name of the event to unsubscribe from |
| `function` | exec | Handler function reference to remove<br/>Must be the same function value passed to [`event.on`](/docs/event/on) |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` if the handler was found and removed, `false` otherwise |

---

### Examples

```lua lineNumbers title="Remove a previously registered handler"
local function on_player_join(player)
    engine.print("info", "Player joined:", player)
end

event.on("on_player_join", on_player_join)
engine.print("info", "Handler removed:", event.off("on_player_join", on_player_join))
```

```lua lineNumbers title="Conditionally unsubscribe based on state"
local handler = nil
handler = function(value)
    if value == "done" then
        event.off("on_signal", handler)
    end
end

event.on("on_signal", handler)
```
