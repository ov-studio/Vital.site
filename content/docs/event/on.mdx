---
title: event.on
description: Registers a handler function for the specified event
badge: Shared
---

### Syntax

```lua
local status = event.on(
    name,
    exec,
    config = nil
)
```

<Callout type="info">
  **The `thread` global is only available inside async handlers.**

  - **When `async = true`** — handler runs inside a dedicated thread; `thread` is available as a global within the callback scope
  - **When `async = false`** — handler runs inline on the calling thread; no `thread` global is injected
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | name | Name of the event to subscribe to |
| `function` | exec | Handler function to invoke when the event fires |
| `table` | config | Configuration table (see [Config](#config)) |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful registration, `false` otherwise |

---

### Config

| Type | Key | Default | Description |
|------|-----|---------|-------------|
| `bool` | async | `false` | • When `true` - runs the handler inside a dedicated thread<br/>• When `false` - runs the handler inline on the calling thread |
| `number` | subscription_limit | `nil` | Maximum number of times this handler will fire before being automatically removed<br/>• Must be `>= 1` if set<br/>• Omit for unlimited invocations |

---

### Callback: `exec`

```lua
exec(...)
```

| Type | Name | Description |
|------|------|-------------|
| `any` | ... | Arguments forwarded from the emitting call |

---

### Examples

```lua lineNumbers title="Register a basic event handler"
event.on("on_player_join", function(player)
    engine.print("info", "Player joined:", player)
end)
```

```lua lineNumbers title="Register a handler with a subscription limit"
event.on("on_pickup_collected", function(pickup)
    engine.print("info", "Collected:", pickup)
end, { subscription_limit = 1 })
```

```lua lineNumbers title="Register an async handler that sleeps"
event.on("on_round_start", function()
    engine.print("info", "Round started, waiting 5 seconds...")
    thread:sleep(5000)
    engine.print("info", "5 seconds elapsed")
end, { async = true })
```
