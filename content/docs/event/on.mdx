---
title: util.event.on
description: Registers a handler function for the specified event
badge: Shared
---

### Syntax

```lua
local status = util.event.on(
    name,
    exec,
    config = nil
)
```

<Callout type="info">
  **The `thread` global is only available inside async handlers.**

  - **When `async = true`** — handler runs inside a dedicated thread; thread can be accessed via [`util.thread.current()`](/docs/thread/current)
  - **When `async = false`** — handler runs inline on the calling thread
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | name | Name of the event to subscribe to |
| `function` | exec | Handler function to invoke when the event fires |
| `table` | config | Configuration table<br/>Refer [Config](#config) section |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful registration, `false` otherwise |

---

### Config

| Type | Key | Default | Description |
|------|-----|---------|-------------|
| `bool` | async | `false` | • When `true` - runs the handler inside a dedicated thread<br/>• When `false` - runs the handler inline on the calling thread |
| `int` | subscription_limit | `nil` | Maximum number of times this handler will fire before being automatically removed<br/>• Must be `>= 1` if set<br/>• Omit for unlimited invocations |

---

### Callback: `exec`

```lua
exec(...)
```

| Type | Name | Description |
|------|------|-------------|
| `any` | ... | Arguments forwarded from the emitting call |

---

### Examples

```lua lineNumbers title="Register a basic event handler"
util.event.on("on_player_join", function(player)
    core.engine.print("info", "Player joined:", player)
end)
```

```lua lineNumbers title="Register a handler with a subscription limit"
util.event.on("on_pickup_collected", function(pickup)
    core.engine.print("info", "Collected:", pickup)
end, { subscription_limit = 1 })
```

```lua lineNumbers title="Register an async handler that sleeps"
util.event.on("on_round_start", function()
    local thread = util.thread.current()
    core.engine.print("info", "Round started, waiting 5 seconds...")
    thread:sleep(5000)
    core.engine.print("info", "5 seconds elapsed")
end, { async = true })
```
