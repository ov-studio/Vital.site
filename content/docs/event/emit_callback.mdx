---
title: event.emit_callback
description: Fires an event and returns a promise that resolves with the handler's return values
badge: Shared
---

### Syntax

```lua
local result = event.emit_callback(
    name,
    options = nil,
    ...
)
```

<Callout type="info">
  **`options` is positional — pass `nil` or omit it entirely when no routing is needed.**

  - **When routing options are needed** — pass the options table as the second argument
  - **When no routing options are needed** — omit `options` and pass arguments directly after `name`
</Callout>

<Callout type="warn">
  **Only serializable values survive remote transmission. Non-serializable values are silently dropped.**

  - **Safe to send** — `nil`, `bool`, `number`, `string`, `table` (including nested tables)
  - **Silently dropped** — functions, userdata, threads, and any other non-serializable Lua values
  - **Local-only emit** (`remote = false`) — no serialization occurs; all Lua values are forwarded as-is
  - **Nested tables** — fully supported, but deeply nested structures increase payload size and serialization cost
</Callout>

<Callout type="info">
  **For remote calls, only a single `promise` is returned regardless of how many handlers exist on the remote side.**

  - **Local dispatch** — returns a single `promise` when one handler is registered, or a `table` of `promise` instances when multiple handlers are registered
  - **Remote dispatch** — always returns a single `promise` that resolves with the return values of the **first** handler that fires on the remote side
  - **No handlers registered** — returns an already-resolved empty `promise`
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | name | Name of the event to fire |
| `table` | options | Routing options table<br/>Refer [Options](#options) section |
| `any` | ... | Arguments forwarded to every handler |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `promise` \| `table` | result | • **When `remote = true`:**<br/>&nbsp;&nbsp;◦ Always a single `promise` - resolves with the return values of the first handler on the remote side<br/>• **When `remote = false`:**<br/>&nbsp;&nbsp;◦ A single `promise` when one handler is registered<br/>&nbsp;&nbsp;◦ A `table` of `promise` instances - one per handler (when multiples are registered)<br/>&nbsp;&nbsp;◦ An already-resolved empty `promise` when no handlers are registered |

### Examples

```lua lineNumbers title="Await a return value from a local handler"
event.on("on_get_score", function()
    return 42
end)

thread.create(function(self)
    local ok, score = self:await(event.emit_callback("on_get_score"))
    engine.print("info", "Score:", score)
end):resume()
```

```lua lineNumbers title="Send a callback from client to server and await the reply"
if engine.get_platform() == "client" then
    thread.create(function(self)
        local ok, result, data = self:await(event.emit_callback("on_fetch_inventory", { remote = true }, "weapons"))
        engine.print("info", "Server replied:", result)
        engine.iprint(data)
    end):resume()
else
    event.on("on_fetch_inventory", function(category)
        local thread = thread.current()
        thread:sleep(2000)
        return "ok", { category = category, items = { "sword", "bow" } }
    end, { async = true })
end
```

```lua lineNumbers title="Send a callback from server to a specific client and await the reply"
if engine.get_platform() == "client" then
    event.on("on_request_position", function()
        local thread = thread.current()
        thread:sleep(500)
        return 12.5, 0.0, 87.3
    end, { async = true })
else
    thread.create(function(self)
        local ok, x, y, z = self:await(event.emit_callback("on_request_position", { remote = true, peer = targetPeerId }))
        engine.print("info", "Client position:", x, y, z)
    end):resume()
end
```

```lua lineNumbers title="Collect return values from multiple local handlers"
event.on("on_query", function() return "handler_a" end)
event.on("on_query", function() return "handler_b" end)

thread.create(function(self)
    local promises = event.emit_callback("on_query")

    for i, j in ipairs(promises) do
        local ok, value = self:await(j)
        engine.print("info", "Handler", i, "returned:", value)
    end
end):resume()
```
