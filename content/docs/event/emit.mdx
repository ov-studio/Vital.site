---
title: event.emit
description: Fires an event, invoking all registered handlers with the supplied arguments
badge: Shared
---

### Syntax

```lua
local status = event.emit(
    name,
    options = nil,
    ...
)
```

<Callout type="info">
  **`options` is positional — pass `nil` or omit it entirely when no routing is needed.**

  - **When routing options are needed** — pass the options table as the second argument
  - **When no routing options are needed** — omit `options` and pass arguments directly after `name`
</Callout>

<Callout type="warn">
  **Only serializable values survive remote transmission. Non-serializable values are silently dropped.**

  - **Safe to send** — `nil`, `bool`, `number`, `string`, `table` (including nested tables)
  - **Silently dropped** — functions, userdata, threads, and any other non-serializable Lua values
  - **Local-only emit** (`remote = false`) — no serialization occurs; all Lua values are forwarded as-is
  - **Nested tables** — fully supported, but deeply nested structures increase payload size and serialization cost
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | name | Name of the event to fire |
| `table` | options | Routing options table (see [Options](#options)) |
| `any` | ... | Arguments forwarded to every handler |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution |

---

### Options

| Type | Key | Default | Side | Description |
|------|-----|---------|------|-------------|
| `bool` | remote | `false` | Shared | • When `true` - transmits the event to the remote side; client sends to server, server broadcasts to all peers<br/>• When `false` - fires handlers on the current side only |
| `number` | peer | `0` | Server | Target peer ID for directed server-side remote dispatch (ignored on client)<br/>• When `0` - broadcasts to all connected peers<br/>• When `> 0` - sends to that specific peer only |

---

### Examples

```lua lineNumbers title="Fire a local event with arguments"
event.on("on_damage", function(amount)
    engine.print("info", "Damage received:", amount)
end)

event.emit("on_damage", 50)
```

```lua lineNumbers title="Client: emit an event to the server"
event.emit("on_client_ready", { remote = true }, localPlayer)
```

```lua lineNumbers title="Server: broadcast an event to all peers"
event.emit("on_server_broadcast", { remote = true }, "Hello everyone")
```

```lua lineNumbers title="Server: send an event to a specific peer"
event.emit("on_server_message", { remote = true, peer = targetPeerId }, "Hello")
```
