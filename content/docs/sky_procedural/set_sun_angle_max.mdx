---
title: gfx.sky.procedural.set_sun_angle_max
description: Sets the maximum sun angle of the procedural sky material
badge: Client
---

### Syntax

```lua
local result = gfx.sky.procedural.set_sun_angle_max(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Maximum sun angle of the procedural sky material to apply |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set procedural sky sun angle max"
gfx.sky.procedural.set_sun_angle_max(30.0)
```
