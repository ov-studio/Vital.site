---
title: gfx.sky.procedural.set_sky_energy_multiplier
description: Sets the energy multiplier of the procedural sky
badge: Client
---

### Syntax

```lua
local status = gfx.sky.procedural.set_sky_energy_multiplier(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Energy multiplier of the procedural sky to apply |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set procedural sky sky energy multiplier"
gfx.sky.procedural.set_sky_energy_multiplier(1.0)
```
