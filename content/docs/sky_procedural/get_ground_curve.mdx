---
title: gfx.sky.procedural.get_ground_curve
description: Retrieves the current curve blend factor of the procedural ground
badge: Client
---

### Syntax

```lua
local result = gfx.sky.procedural.get_ground_curve()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current curve blend factor of the procedural ground value |

---

### Examples

```lua lineNumbers title="Print current procedural sky ground curve"
core.engine.print("info", "Ground curve:", gfx.sky.procedural.get_ground_curve())
```
