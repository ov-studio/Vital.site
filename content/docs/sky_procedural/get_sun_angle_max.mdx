---
title: gfx.sky.procedural.get_sun_angle_max
description: Retrieves the current maximum sun angle of the procedural sky material
badge: Client
---

### Syntax

```lua
local result = gfx.sky.procedural.get_sun_angle_max()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current maximum sun angle of the procedural sky material value |

---

### Examples

```lua lineNumbers title="Print current procedural sky sun angle max"
core.engine.print("info", "Sun angle max:", gfx.sky.procedural.get_sun_angle_max())
```
