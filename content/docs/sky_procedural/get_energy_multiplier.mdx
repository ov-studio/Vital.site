---
title: gfx.sky.procedural.get_energy_multiplier
description: Retrieves the current overall energy multiplier of the procedural sky material
badge: Client
---

### Syntax

```lua
local result = gfx.sky.procedural.get_energy_multiplier()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current overall energy multiplier of the procedural sky material value |

---

### Examples

```lua lineNumbers title="Print current procedural sky energy multiplier"
core.engine.print("info", "Energy multiplier:", gfx.sky.procedural.get_energy_multiplier())
```
