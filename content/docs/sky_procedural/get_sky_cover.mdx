---
title: gfx.sky.procedural.get_sky_cover
description: Retrieves the current sky cover texture path of the procedural sky material
badge: Client
---

### Syntax

```lua
local result = gfx.sky.procedural.get_sky_cover()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` \| `bool` | result | Path of the currently assigned sky cover texture, `false` otherwise |

---

### Examples

```lua lineNumbers title="Print current procedural sky cover texture path"
core.engine.print("info", "Sky cover:", gfx.sky.procedural.get_sky_cover())
```
