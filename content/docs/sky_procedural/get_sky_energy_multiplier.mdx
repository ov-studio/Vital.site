---
title: gfx.sky.procedural.get_sky_energy_multiplier
description: Retrieves the current energy multiplier of the procedural sky
badge: Client
---

### Syntax

```lua
local result = gfx.sky.procedural.get_sky_energy_multiplier()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current energy multiplier of the procedural sky value |

---

### Examples

```lua lineNumbers title="Print current procedural sky sky energy multiplier"
core.engine.print("info", "Sky energy multiplier:", gfx.sky.procedural.get_sky_energy_multiplier())
```
