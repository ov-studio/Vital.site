---
title: gfx.sky.procedural.get_ground_bottom_color
description: Retrieves the current bottom color of the procedural ground
badge: Client
---

### Syntax

```lua
local color = gfx.sky.procedural.get_ground_bottom_color()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `color` | color | Current bottom color of the procedural ground as `{r, g, b, a}` |

---

### Examples

```lua lineNumbers title="Print current procedural sky ground bottom color"
core.engine.print("info", "Ground bottom color:", util.table.unpack(gfx.sky.procedural.get_ground_bottom_color()))
```
