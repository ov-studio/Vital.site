---
title: gfx.sky.procedural.set_sky_cover_modulate
description: Sets the sky cover modulate color of the procedural sky material
badge: Client
---

### Syntax

```lua
local status = gfx.sky.procedural.set_sky_cover_modulate(
    color
)
```

```lua
local status = gfx.sky.procedural.set_sky_cover_modulate(
    hex
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `color` | color | Sky cover modulate color of the procedural sky material as `{r, g, b, a}` |
| `string` | hex | Alternatively, a valid HTML hex color string (e.g. `"#ffffff"`) |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set procedural sky cover modulate using a color table"
gfx.sky.procedural.set_sky_cover_modulate({1.0, 1.0, 1.0, 1.0})
```

```lua lineNumbers title="Set procedural sky cover modulate using a hex string"
gfx.sky.procedural.set_sky_cover_modulate("#ffffff")
```
