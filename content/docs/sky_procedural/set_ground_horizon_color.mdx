---
title: gfx.sky.procedural.set_ground_horizon_color
description: Sets the horizon color of the procedural ground
badge: Client
---

### Syntax

```lua
local result = gfx.sky.procedural.set_ground_horizon_color(
    color
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `color` | color | Horizon color of the procedural ground to apply as `{r, g, b, a}` |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set procedural sky ground horizon color"
gfx.sky.procedural.set_ground_horizon_color({0.3, 0.25, 0.2, 1.0})
```
