---
title: gfx.sky.procedural.set_ground_horizon_color
description: Sets the horizon color of the procedural ground
badge: Client
---

### Syntax

```lua
local status = gfx.sky.procedural.set_ground_horizon_color(
    color
)
```

```lua
local status = gfx.sky.procedural.set_ground_horizon_color(
    hex
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `color` | color | Horizon color of the procedural ground as `{r, g, b, a}` |
| `string` | hex | Alternatively, a valid HTML hex color string (e.g. `"#ffffff"`) |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set procedural sky ground horizon color using a color table"
gfx.sky.procedural.set_ground_horizon_color({0.3, 0.25, 0.2, 1.0})
```

```lua lineNumbers title="Set procedural sky ground horizon color using a hex string"
gfx.sky.procedural.set_ground_horizon_color("#4d4033")
```
