---
title: gfx.sky.procedural.set_ground_bottom_color
description: Sets the bottom color of the procedural ground
badge: Client
---

### Syntax

```lua
local status = gfx.sky.procedural.set_ground_bottom_color(
    color
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `color` | color | Bottom color of the procedural ground to apply as `{r, g, b, a}` |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set procedural sky ground bottom color"
gfx.sky.procedural.set_ground_bottom_color({0.1, 0.07, 0.034, 1.0})
```
