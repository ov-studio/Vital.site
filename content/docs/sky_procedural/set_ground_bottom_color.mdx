---
title: gfx.sky.procedural.set_ground_bottom_color
description: Sets the bottom color of the procedural ground
badge: Client
---

### Syntax

```lua
local status = gfx.sky.procedural.set_ground_bottom_color(
    color
)
```

```lua
local status = gfx.sky.procedural.set_ground_bottom_color(
    hex
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `color` | color | Bottom color of the procedural ground as `{r, g, b, a}` |
| `string` | hex | Alternatively, a valid HTML hex color string (e.g. `"#ffffff"`) |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set procedural sky ground bottom color using a color table"
gfx.sky.procedural.set_ground_bottom_color({0.1, 0.07, 0.034, 1.0})
```

```lua lineNumbers title="Set procedural sky ground bottom color using a hex string"
gfx.sky.procedural.set_ground_bottom_color("#1a1208")
```
