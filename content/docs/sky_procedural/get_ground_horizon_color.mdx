---
title: gfx.sky.procedural.get_ground_horizon_color
description: Retrieves the current horizon color of the procedural ground
badge: Client
---

### Syntax

```lua
local color = gfx.sky.procedural.get_ground_horizon_color()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `color` | color | Current horizon color of the procedural ground as `{r, g, b, a}` |

---

### Examples

```lua lineNumbers title="Print current procedural sky ground horizon color"
core.engine.print("info", "Ground horizon color:", util.table.unpack(gfx.sky.procedural.get_ground_horizon_color()))
```
