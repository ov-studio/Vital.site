---
title: gfx.vol_fog.set_anisotropy
description: The anisotropy to be set for the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.vol_fog.set_anisotropy(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Anisotropy value to be applied<br/>Controls the directional bias of light scattering within the gfx.fog. Positive values scatter light forward in the direction of the light source, producing a glowing halo effect. Negative values scatter light backward, creating a more diffuse appearance |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Bias scattering forward for a sun halo effect"
gfx.vol_fog.set_anisotropy(0.5)
```
