---
title: gfx.vol_fog.set_gi_inject
description: Sets the GI injection amount of the volumetric fog
badge: Client
---

### Syntax

```lua
local status = gfx.vol_fog.set_gi_inject(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | GI inject value to apply<br/>Controls how much global illumination light is injected into the volumetric gfx.fog. Higher values make the fog respond to and carry indirect light throughout the scene |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Allow GI to illuminate the fog volume"
gfx.vol_fog.set_gi_inject(1.0)
```
