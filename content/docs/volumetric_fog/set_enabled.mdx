---
title: volumetric_fog.set_enabled
description: Toggles the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local status = volumetric_fog.set_enabled(state)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `bool` | state | Determines volumetric fog's state:<br/>• When `true` - volumetric fog is enabled<br/>• When `false` - volumetric fog is disabled |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Enable and disable volumetric fog"
--Enable volumetric fog
volumetric_fog.set_enabled(true)

--Disable volumetric fog
volumetric_fog.set_enabled(false)
```
