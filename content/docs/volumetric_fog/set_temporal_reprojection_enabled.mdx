---
title: gfx.vol_fog.set_temporal_reprojection_enabled
description: Toggles temporal reprojection for the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.vol_fog.set_temporal_reprojection_enabled(
    state
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `bool` | state | Determines whether temporal reprojection is used:<br/>• When `true` - temporal reprojection is enabled, reusing data from previous frames to smooth out the fog at a lower per-frame cost<br/>• When `false` - temporal reprojection is disabled |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Enable temporal reprojection for smoother fog at lower cost"
gfx.vol_fog.set_temporal_reprojection_enabled(true)
```
