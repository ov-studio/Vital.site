---
title: vol_fog.get_length
description: Retrieves the current draw length of the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = vol_fog.get_length()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current length value |

---

### Examples

```lua lineNumbers title="Print current volumetric fog draw length"
engine.print("info", "Volumetric fog length:", vol_fog.get_length())
```
