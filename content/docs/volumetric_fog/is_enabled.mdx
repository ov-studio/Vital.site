---
title: vol_fog.is_enabled
description: Retrieves whether the volumetric fog post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local result = vol_fog.is_enabled()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if volumetric fog is enabled, `false` otherwise |

---

### Examples

```lua lineNumbers title="Print whether volumetric fog is enabled"
engine.print("info", "Volumetric fog enabled:", vol_fog.is_enabled())
```
