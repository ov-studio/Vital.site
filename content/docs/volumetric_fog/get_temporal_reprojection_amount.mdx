---
title: volumetric_fog.get_temporal_reprojection_amount
description: Retrieves the current temporal reprojection blend amount of the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = volumetric_fog.get_temporal_reprojection_amount()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current temporal reprojection amount |

<hr/>

### Examples

```lua lineNumbers title="Print current temporal reprojection blend amount"
engine.print("info", "Temporal reprojection amount:", volumetric_fog.get_temporal_reprojection_amount())
```
