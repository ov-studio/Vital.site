---
title: gfx.vol_fog.set_length
description: The draw length to set for the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.vol_fog.set_length(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Length value to apply<br/>Sets the maximum distance from the camera at which volumetric fog is rendered. Larger values extend the fog further into the scene at the cost of performance |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Extend volumetric fog across a large distance"
gfx.vol_fog.set_length(512.0)
```
