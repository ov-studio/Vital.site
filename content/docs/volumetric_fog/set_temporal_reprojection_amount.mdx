---
title: gfx.vol_fog.set_temporal_reprojection_amount
description: Sets the temporal reprojection blend amount of the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.vol_fog.set_temporal_reprojection_amount(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Temporal reprojection amount value to apply<br/>Controls how heavily previous frames are blended into the current frame. Higher values produce smoother fog with less flickering but may introduce ghosting artifacts during fast camera movement |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Set a high reprojection blend for smooth fog"
gfx.vol_fog.set_temporal_reprojection_amount(0.9)
```
