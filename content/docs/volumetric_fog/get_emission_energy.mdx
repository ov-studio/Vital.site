---
title: volumetric_fog.get_emission_energy
description: Retrieves the current emission energy of the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = volumetric_fog.get_emission_energy()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current emission energy value |

<hr/>

### Examples

```lua lineNumbers title="Print current volumetric fog emission energy"
engine.print("info", "Volumetric fog emission energy:", volumetric_fog.get_emission_energy())
```
