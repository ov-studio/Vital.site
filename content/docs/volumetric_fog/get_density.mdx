---
title: volumetric_fog.get_density
description: Retrieves the current density of the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = volumetric_fog.get_density()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current density value |

<hr/>

### Examples

```lua lineNumbers title="Print current volumetric fog density"
engine.print("info", "Volumetric fog density:", volumetric_fog.get_density())
```
