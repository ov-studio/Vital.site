---
title: gfx.vol_fog.set_sky_affect
description: Sets the sky influence amount of the volumetric fog
badge: Client
---

### Syntax

```lua
local status = gfx.vol_fog.set_sky_affect(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Sky influence value to apply<br/>Controls how much the volumetric fog blends into and affects the appearance of the sky. A value of `0` leaves the sky unaffected while higher values cause the fog to visibly tint and obscure the skybox |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Allow volumetric fog to partially tint the sky"
gfx.vol_fog.set_sky_affect(0.5)
```
