---
title: volumetric_fog.set_ambient_inject
description: The ambient light injection amount to be set for the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local status = volumetric_fog.set_ambient_inject(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Ambient inject value to be applied<br/>Controls how much ambient light is injected into the volumetric fog volume. Higher values allow the fog to be lit by the scene's ambient light, producing a softer and more naturally integrated appearance |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Allow ambient light to fill the fog volume"
volumetric_fog.set_ambient_inject(0.5)
```
