---
title: gfx.vol_fog.get_emission
description: Retrieves the current emission color of the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.vol_fog.get_emission()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `color` | result | Volumetric fog emission color as `{r, g, b, a}` |

---

### Examples

```lua lineNumbers title="Print current volumetric fog emission color"
core.engine.print("info", "Volumetric fog emission:", util.table.unpack(gfx.vol_fog.get_emission()))
```
