---
title: volumetric_fog.get_ambient_inject
description: Retrieves the current ambient light injection amount of the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = volumetric_fog.get_ambient_inject()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current ambient inject value |

<hr/>

### Examples

```lua lineNumbers title="Print current volumetric fog ambient inject amount"
engine.print("info", "Volumetric fog ambient inject:", volumetric_fog.get_ambient_inject())
```
