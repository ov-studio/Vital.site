---
title: volumetric_fog.get_detail_spread
description: Retrieves the current detail spread of the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = volumetric_fog.get_detail_spread()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current detail spread value |

<hr/>

### Examples

```lua lineNumbers title="Print current volumetric fog detail spread"
engine.print("info", "Volumetric fog detail spread:", volumetric_fog.get_detail_spread())
```
