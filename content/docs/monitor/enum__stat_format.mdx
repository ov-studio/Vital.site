---
title: util.monitor.stat_format
description: Enumeration of display format types for custom stats
badge: Client
---

### Values

| Constant | Description |
|----------|-------------|
| `util.monitor.stat_format.QUANTITY` | Displayed as a plain number — suitable for counts and discrete values |
| `util.monitor.stat_format.MEMORY` | Displayed as a memory size — value is interpreted as bytes |
| `util.monitor.stat_format.TIME` | Displayed as a duration — value is interpreted as seconds |
| `util.monitor.stat_format.PERCENTAGE` | Displayed as a percentage — value is interpreted as a ratio between `0.0` and `1.0` |
