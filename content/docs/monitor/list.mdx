---
title: util.monitor.list
description: Retrieves all available performance monitors, grouped by native and custom
badge: Client
---

### Syntax

```lua
local result = util.monitor.list()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `table` | result | Table containing sub-tables:<br/>• `native` - all built-in Godot performance monitors listed<br/>• `custom` - all currently registered custom monitors listed<br/>Refer [**Structure**](#structure) section |

---

### Structure

`result.native` — Array of entries for all built-in Godot performance monitors

| Field | Type | Description |
|-------|------|-------------|
| `id` | `int` | Enum value usable with [`util.monitor.get`](/docs/monitor/get) and [`util.monitor.has`](/docs/monitor/has) |
| `name` | `string` | Constant name from the [`util.monitor.stat_native`](/docs/monitor/enum/stat_native) enum |

`result.custom` — Array of entries for all currently registered custom monitors

| Field | Type | Description |
|-------|------|-------------|
| `id` | `string` | Registered string id usable with [`util.monitor.get`](/docs/monitor/get) and [`util.monitor.has`](/docs/monitor/has) |
| `name` | `string` | Human-readable display name provided at registration |

---

### Examples

```lua lineNumbers title="Print all available monitors"
local monitors = util.monitor.list()

core.engine.print("info", "Native monitors:", #monitors.native)
core.engine.print("info", "Custom monitors:", #monitors.custom)
```

```lua lineNumbers title="Iterate over native monitors"
local monitors = util.monitor.list()

for index, entry in ipairs(monitors.native) do
    core.engine.print("info", entry.name, "=", util.monitor.get(entry.id))
end
```

```lua lineNumbers title="Iterate over custom monitors"
local monitors = util.monitor.list()

for index, entry in ipairs(monitors.custom) do
    core.engine.print("info", entry.name, "=", util.monitor.get(entry.id))
end
```

```lua lineNumbers title="Check if any custom monitors are registered"
local monitors = util.monitor.list()

if #monitors.custom == 0 then
    core.engine.print("warn", "No custom monitors registered")
end
```
