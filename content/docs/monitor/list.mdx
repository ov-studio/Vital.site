---
title: util.monitor.list
description: Retrieves all available stats, grouped by native and custom
badge: Client
---

### Syntax

```lua
local result = util.monitor.list()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `table` | result | Table containing sub-tables:<br/>• `native` - all built-in Godot stats listed<br/>• `custom` - all currently registered custom stats listed<br/>Refer [**Structure**](#structure) section |

---

### Structure

`result.native` — Array of entries for all built-in Godot stats

| Field | Type | Description |
|-------|------|-------------|
| `id` | `int` | Enum value usable with [`util.monitor.get`](/docs/monitor/get) and [`util.monitor.has`](/docs/monitor/has) |
| `name` | `string` | Constant name from the [`util.monitor.stat_native`](/docs/monitor/enum/stat_native) enum |
| `format` | `string` | Display format of the stat, matching one of the [`util.monitor.stat_format`](/docs/monitor/enum/stat_format) enum |

`result.custom` — Array of entries for all currently registered custom stats

| Field | Type | Description |
|-------|------|-------------|
| `id` | `string` | Registered string id usable with [`util.monitor.get`](/docs/monitor/get) and [`util.monitor.has`](/docs/monitor/has) |
| `name` | `string` | Human-readable display name provided at registration |
| `format` | `string` | Display format of the stat, matching one of the [`util.monitor.stat_format`](/docs/monitor/enum/stat_format) enum |

---

### Examples

```lua lineNumbers title="Print all available stats"
local stats = util.monitor.list()

core.engine.print("info", "Native stats:", #stats.native)
core.engine.print("info", "Custom stats:", #stats.custom)
```

```lua lineNumbers title="Iterate over native stats"
local stats = util.monitor.list()

for index, entry in ipairs(stats.native) do
    core.engine.print("info", entry.name, "=", util.monitor.get(entry.id), "(" .. entry.format .. ")")
end
```

```lua lineNumbers title="Iterate over custom stats"
local stats = util.monitor.list()

for index, entry in ipairs(stats.custom) do
    core.engine.print("info", entry.name, "=", util.monitor.get(entry.id), "(" .. entry.format .. ")")
end
```

```lua lineNumbers title="Check if any custom stats are registered"
local stats = util.monitor.list()

if #stats.custom == 0 then
    core.engine.print("warn", "No custom stats registered")
end
```
