---
title: util.monitor.has
description: Checks whether a stat exists, either native or custom
badge: Client
---

### Syntax

```lua
local result = util.monitor.has(
    id
)
```

<Callout type="info">
  **Accepts both native and custom stat identifiers.**

  - **Native stats** — pass a [`util.monitor.stat_native`](/docs/monitor/enum/stat_native) enum value to check whether a built-in Godot stat exists.
  - **Custom stats** — pass a `string` id to check whether a stat registered via [`util.monitor.register`](/docs/monitor/register) exists.
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|
| [`util.monitor.stat_native`](/docs/monitor/enum/stat_native) \| `string` | id | Stat to look up:<br/>• When [`util.monitor.stat_native`](/docs/monitor/enum/stat_native) - checks a native stat by enum value<br/>• When `string` - checks a custom stat by its registered id |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if the stat exists, `false` otherwise |

---

### Examples

```lua lineNumbers title="Check for a native stat"
if util.monitor.has(util.monitor.stat_native.TIME_FPS) then
    core.engine.print("info", "FPS stat is available")
end
```

```lua lineNumbers title="Check for a custom stat before reading it"
if util.monitor.has("my_resource:entity_count") then
    core.engine.print("info", "Entity count:", util.monitor.get("my_resource:entity_count"))
end
```

```lua lineNumbers title="Guard registration with an existence check"
if not util.monitor.has("my_resource:cache_size") then
    util.monitor.register("my_resource:cache_size", "Cache Size", function()
        return cache.byte_size()
    end, util.monitor.stat_format.MEMORY)
end
```
