---
title: util.resource.current
description: Returns the name of the resource the calling script belongs to
badge: Shared
---

### Syntax

```lua
local result = util.resource.current()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` \| `bool` | result | Name of the calling resource, `false` otherwise |

---

### Examples

```lua lineNumbers title="Get the current resource name"
core.engine.print("info", "Running resource:", util.resource.current())
```
