---
title: util.resource.list
description: Returns a list of all loaded resources
badge: Shared
---

### Syntax

```lua
local result = util.resource.list()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `table` | result | Array of resource name strings for all loaded resources; empty table if none are loaded |

---

### Examples

```lua lineNumbers title="Print all loaded resources"
engine.iprint(util.resource.list())
```

```lua lineNumbers title="Count loaded resources"
local list = util.resource.list()

engine.print("info", "Loaded resources:", #list)
```
