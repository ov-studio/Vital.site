---
title: ssr.is_enabled
description: Retrieves whether the SSR post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local result = ssr.is_enabled()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if SSR is enabled, `false` otherwise |

<hr/>

### Examples

```lua lineNumbers title="Print whether ssr is currently enabled"
engine.print("info", "SSR enabled:", ssr.is_enabled())
```
