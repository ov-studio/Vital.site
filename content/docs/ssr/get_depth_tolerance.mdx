---
title: ssr.get_depth_tolerance
description: Retrieves the current depth tolerance of the SSR post-processing effect
badge: Client
---

### Syntax

```lua
local result = ssr.get_depth_tolerance()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current depth tolerance value |

<hr/>

### Examples

```lua lineNumbers title="Get the current ssr depth tolerance"
engine.print("info", "SSR depth tolerance:", ssr.get_depth_tolerance())
```
