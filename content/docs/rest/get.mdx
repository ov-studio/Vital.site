---
title: rest.get
description: Executes a GET request on specified url
badge: Shared
---

### Syntax

```lua
local promise = rest.get(
    url,
    headers = {
        "Content-Type: application/json"
    },
    timeout = 60
)
```

<Callout type="info">
  **This function returns a promise — use [`thread:await`](/docs/thread/method/await) inside a thread to retrieve the result.**

  - **Inside a thread** — `self:await(rest.get(...))` yields until the response is received, then returns the resolved values
  - **Outside a thread** — the promise can still be created, but must be awaited from within a thread to retrieve the result
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | url | Request URL to query |
| `table` | headers | Request headers for query |
| `int` | timeout | Read timeout in seconds |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `promise` | promise | Resolves with the response `string` on success, or rejects with an error `string` on failure |

<hr/>

### Examples

```lua lineNumbers title="Send a basic GET request"
local t = thread.create(function(self)
    local ok, result = self:await(rest.get("https://api.example.com/info"))
    if ok then
        engine.print("info", "Response:", result)
    else
        engine.print("error", "Request failed:", result)
    end
end)
t:resume()
```

```lua lineNumbers title="GET with custom headers and timeout"
local t = thread.create(function(self)
    local ok, result = self:await(rest.get(
        "https://api.example.com/data",
        {
            "User-Agent: MyApp",
            "Accept: application/json"
        },
        30
    ))
    engine.print("info", "Response:", result)
end)
t:resume()
```
