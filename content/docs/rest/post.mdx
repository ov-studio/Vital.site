---
title: rest.post
description: Executes a POST request on specified url
badge: Shared
---

### Syntax

```lua
local response, error = rest.post(url, body, headers = {
    "Content-Type: application/json"
})
```

<Callout type="warn">
Requires execution within a valid thread context for asynchronous, non-blocking operation.<br/>
Calls made outside a thread will fail to process.
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | url | Request URL to query |
| `string` | body | Request body for query |
| `table` | headers | Request headers for query |


### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` or `bool` | response | Response on successful execution, or `false` on failure |
| `string` or `bool` | error | `false` on successful execution, or error on failure |

<hr/>

### Examples

```lua
engine.print("Response:", rest.post("https://api.example.com/users", table.encode({
    name: "John Doe",
    email: "john@example.com"
}, "JSON")))
```
