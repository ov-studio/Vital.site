---
title: rest.post
description: Executes a POST request on specified url
badge: Shared
---

### Syntax

```lua
local promise = rest.post(
    url,
    body,
    headers = {
        "Content-Type: application/json"
    },
    timeout = 60
)
```

<Callout type="info">
  **This function returns a promise and starts processing the query.** Use [`thread:await`](/docs/thread/method/await) inside a thread to retrieve the result.

  - **Inside a thread** — `self:await(rest.post(...))` yields until the response is received, then returns the resolved values
  - **Outside a thread** — the promise can still be created, but must be awaited from within a thread to retrieve the result
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | url | Request URL to query |
| `string` | body | Request body for query |
| `table` | headers | Request headers for query |
| `int` | timeout | Read timeout in seconds |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `promise` | promise | Refer [**Settlements**](#settlements) section |

---

### Settlements

| Type | Name | Description |
|---------|------|-------------|
| `bool` | status | `true` on success, `false` on failure |
| `string` | resolved | Response body returned by the server |
| `string` | rejected | Error message describing the failure |

<hr/>

### Examples

```lua lineNumbers title="Send a basic POST request with a JSON body"
thread.create(function(self)
    local ok, result = self:await(rest.post(
        "https://api.example.com/users",
        table.encode({
            name = "John Doe",
            email = "john@example.com"
        }, "JSON")
    ))

    if ok then
        engine.print("info", "Response:", result)
    else
        engine.print("error", "Request failed:", result)
    end
end):resume()
```

```lua lineNumbers title="Send a POST with custom authorization header and timeout"
thread.create(function(self)
    local ok, result = self:await(rest.post(
        "https://api.example.com/users",
        table.encode({
            name = "John Doe",
            email = "john@example.com"
        }, "JSON"),
        {
            "Authorization: Bearer my-token"
        },
        30
    ))

    engine.print("info", "Response:", result)
end):resume()
```
