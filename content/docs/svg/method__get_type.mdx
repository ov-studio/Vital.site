---
title: self:get_type
description: Returns the type name of the instance
badge: Client
---

### Syntax

```lua
local result = self:get_type()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` / `bool` | result | Type of the instance, or `false` if unavailable |

<hr/>

### Examples

```lua lineNumbers title="Get the type of an SVG instance"
local icon = svg.create("assets/icon.svg")

engine.print("info", icon:get_type()) --'svg'
```
