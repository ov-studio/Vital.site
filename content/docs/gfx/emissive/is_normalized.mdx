---
title: gfx.emissive.is_normalized
description: Retrieves whether glow level normalization is currently enabled for the emissive post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.emissive.is_normalized()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if glow normalization is enabled, `false` otherwise |

<hr/>

### Examples

```lua
gfx.print("Emissive normalized:", gfx.emissive.is_normalized())
```
