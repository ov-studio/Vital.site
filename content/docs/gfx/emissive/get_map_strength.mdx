---
title: gfx.emissive.get_map_strength

badge: Client
---

### Syntax

```lua
local result = gfx.emissive.get_map_strength()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current glow map strength value |

<hr/>

### Examples

```lua lineNumbers title="Print emissive glow map strength"
engine.print("info", "Emissive map strength:", gfx.emissive.get_map_strength())
```
