---
title: gfx.emissive.is_enabled
description: Retrieves whether the emissive glow post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local result = gfx.emissive.is_enabled()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if emissive glow is enabled, `false` otherwise |

<hr/>

### Examples

```lua
engine.print("Emissive enabled:", gfx.emissive.is_enabled())
```
