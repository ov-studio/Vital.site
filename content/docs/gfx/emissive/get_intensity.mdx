---
title: gfx.emissive.get_intensity

badge: Client
---

### Syntax

```lua
local result = gfx.emissive.get_intensity()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current intensity value |

<hr/>

### Examples

```lua
gfx.print("Emissive intensity:", gfx.emissive.get_intensity())
```
