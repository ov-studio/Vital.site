---
title: gfx.emissive.get_bloom
description: Retrieves the current bloom threshold of the emissive post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.emissive.get_bloom()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current bloom value |

<hr/>

### Examples

```lua
engine.print("info", "Emissive bloom:", gfx.emissive.get_bloom())
```
