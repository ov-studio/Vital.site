---
title: gfx.emissive.get_mix
description: Retrieves the current mix amount of the emissive post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.emissive.get_mix()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current mix value |

<hr/>

### Examples

```lua lineNumbers title="Print emissive mix amount"
engine.print("info", "Emissive mix:", gfx.emissive.get_mix())
```
