---
title: gfx.emissive.get_hdr_luminance_cap
description: Retrieves the current HDR luminance cap of the emissive post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.emissive.get_hdr_luminance_cap()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current HDR luminance cap value |

<hr/>

### Examples

```lua lineNumbers title="Print emissive HDR luminance cap"
engine.print("info", "Emissive HDR luminance cap:", gfx.emissive.get_hdr_luminance_cap())
```
