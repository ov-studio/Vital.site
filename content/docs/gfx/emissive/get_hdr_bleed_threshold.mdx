---
title: gfx.emissive.get_hdr_bleed_threshold
description: Retrieves the current HDR bleed threshold of the emissive post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.emissive.get_hdr_bleed_threshold()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current HDR bleed threshold value |

<hr/>

### Examples

```lua lineNumbers title="Print emissive HDR bleed threshold"
engine.print("info", "Emissive HDR bleed threshold:", gfx.emissive.get_hdr_bleed_threshold())
```
