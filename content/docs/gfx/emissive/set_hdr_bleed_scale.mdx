---
title: gfx.emissive.set_hdr_bleed_scale
description: The HDR bleed scale to be set for the emissive post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.emissive.set_hdr_bleed_scale(value)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | HDR bleed scale value to be applied<br/>Controls how strongly pixels above the bleed threshold contribute to the glow. Higher values amplify the bleed from bright HDR areas, producing a more intense halo around very bright surfaces |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Amplify bleed from bright HDR surfaces"
gfx.emissive.set_hdr_bleed_scale(2.0)
```
