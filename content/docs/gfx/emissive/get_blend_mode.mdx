---
title: gfx.emissive.get_blend_mode
description: Retrieves the current blend mode of the emissive post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.emissive.get_blend_mode()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `int` | result | Current blend mode |

<hr/>

### Examples

```lua
engine.print("info", "Emissive blend mode:", gfx.emissive.get_blend_mode())
```
