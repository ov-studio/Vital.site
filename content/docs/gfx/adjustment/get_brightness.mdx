```lua

```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current brightness value |

<hr/>

### Examples

```lua
gfx.print("Current brightness:", gfx.adjustment.get_brightness())
```
