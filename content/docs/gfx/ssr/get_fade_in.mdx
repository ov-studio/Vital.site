---
title: gfx.ssr.get_fade_in

badge: Client
---

### Syntax

```lua
local result = gfx.ssr.get_fade_in()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current fade-in value |

<hr/>

### Examples

```lua lineNumbers title="Get the current SSR fade-in distance"
engine.print("info", "SSR fade in:", gfx.ssr.get_fade_in())
```
