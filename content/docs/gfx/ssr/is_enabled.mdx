---
title: gfx.ssr.is_enabled
description: Retrieves whether the SSR post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local result = gfx.ssr.is_enabled()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if SSR is enabled, `false` otherwise |

<hr/>

### Examples

```lua lineNumbers title="Print whether SSR is currently enabled"
engine.print("info", "SSR enabled:", gfx.ssr.is_enabled())
```
