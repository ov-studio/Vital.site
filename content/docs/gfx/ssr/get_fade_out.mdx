---
title: gfx.ssr.get_fade_out
description: Retrieves the current fade-out distance of the SSR post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.ssr.get_fade_out()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current fade-out value |

<hr/>

### Examples

```lua lineNumbers title="Get the current ssr fade-out distance"
engine.print("info", "SSR fade out:", gfx.ssr.get_fade_out())
```
