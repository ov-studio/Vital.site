---
title: gfx.fog.get_aerial_perspective
description: Retrieves the current aerial perspective blend amount of the fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.fog.get_aerial_perspective()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current aerial perspective value |

<hr/>

### Examples

```lua lineNumbers title="Print current fog aerial perspective"
engine.print("info", "Fog aerial perspective:", gfx.fog.get_aerial_perspective())
```
