---
title: gfx.fog.get_light_energy
description: Retrieves the current light energy of the fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.fog.get_light_energy()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current light energy value |

<hr/>

### Examples

```lua
engine.print("info", "Fog light energy:", gfx.fog.get_light_energy())
```
