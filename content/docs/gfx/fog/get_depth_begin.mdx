---
title: gfx.fog.get_depth_begin
description: Retrieves the current depth begin distance of the fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.fog.get_depth_begin()
```

<Callout type="warn">
  **Only available when the fog mode is set to depth mode.** See [`gfx.fog.set_mode`](/docs/gfx/fog/set_mode) to configure the active fog mode.

  - **Depth mode required** — calling this function in any other fog mode will return an unexpected result
  - **Recommended usage** — use alongside [`gfx.fog.get_depth_end`](/docs/gfx/fog/get_depth_end) to inspect the full fog transition range
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current depth begin value |

<hr/>

### Examples

```lua
engine.print("info", "Fog depth begin:", gfx.fog.get_depth_begin())
```
