---
title: gfx.fog.get_sky_affect

badge: Client
---

### Syntax

```lua
local result = gfx.fog.get_sky_affect()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current sky influence value |

<hr/>

### Examples

```lua
engine.print("info", "Fog sky affect:", gfx.fog.get_sky_affect())
```
