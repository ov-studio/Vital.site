---
title: gfx.fog.get_sun_scatter
description: Retrieves the current sun scatter amount of the fog post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.fog.get_sun_scatter()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current sun scatter value |

<hr/>

### Examples

```lua
engine.print("info", "Fog sun scatter:", gfx.fog.get_sun_scatter())
```
