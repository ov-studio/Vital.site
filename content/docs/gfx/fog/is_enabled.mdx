---
title: gfx.fog.is_enabled
description: Retrieves whether the fog post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local result = gfx.fog.is_enabled()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if fog is enabled, `false` otherwise |

<hr/>

### Examples

```lua
engine.print("info", "Fog enabled:", gfx.fog.is_enabled())
```
