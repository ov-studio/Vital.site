---
title: gfx.ssao.set_power
description: Sets the power curve of the SSAO post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.ssao.set_power(value)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Power value to be applied<br/>Controls the exponent applied to the occlusion factor, affecting the contrast between occluded and non-occluded areas |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Increase occlusion contrast"
gfx.ssao.set_power(2.0)
```
