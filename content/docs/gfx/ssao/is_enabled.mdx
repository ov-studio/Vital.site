---
title: gfx.ssao.is_enabled
description: Retrieves whether the SSAO post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local result = gfx.ssao.is_enabled()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if SSAO is enabled, `false` otherwise |

<hr/>

### Examples

```lua lineNumbers title="Print whether SSAO is currently enabled"
engine.print("info", "SSAO enabled:", gfx.ssao.is_enabled())
```
