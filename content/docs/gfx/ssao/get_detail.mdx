---
title: gfx.ssao.get_detail

badge: Client
---

### Syntax

```lua
local result = gfx.ssao.get_detail()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current detail value |

<hr/>

### Examples

```lua
gfx.print("SSAO detail:", gfx.ssao.get_detail())
```
