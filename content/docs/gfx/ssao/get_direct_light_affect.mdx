---
title: gfx.ssao.get_direct_light_affect
description: Retrieves the current direct lighting influence of the SSAO post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.ssao.get_direct_light_affect()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current direct light influence value |

<hr/>

### Examples

```lua
engine.print("info", "SSAO direct light affect:", gfx.ssao.get_direct_light_affect())
```
