---
title: gfx.ssao.get_radius
description: Retrieves the current sampling radius of the SSAO post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.ssao.get_radius()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current radius value |

<hr/>

### Examples

```lua
engine.print("info", "SSAO radius:", gfx.ssao.get_radius())
```
