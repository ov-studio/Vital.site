title: gfx.ssao.get_channel_affect

badge: Client
---

### Syntax

```lua
local result = gfx.ssao.get_channel_affect()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current channel influence value |

<hr/>

### Examples

```lua lineNumbers title="Get the current SSAO channel affect"
engine.print("info", "SSAO channel affect:", gfx.ssao.get_channel_affect())
```
