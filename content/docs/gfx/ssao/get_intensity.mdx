badge: Client

### Syntax

```lua
local result = gfx.ssao.get_intensity()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current intensity value |

<hr/>

### Examples

```lua lineNumbers title="Get the current SSAO intensity"
engine.print("info", "SSAO intensity:", gfx.ssao.get_intensity())
```
