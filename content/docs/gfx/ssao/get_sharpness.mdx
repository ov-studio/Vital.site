---
title: gfx.ssao.get_sharpness
description: Retrieves the current sharpness of the SSAO post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.ssao.get_sharpness()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current sharpness value |

<hr/>

### Examples

```lua lineNumbers title="Get the current ssao sharpness"
engine.print("info", "SSAO sharpness:", gfx.ssao.get_sharpness())
```
