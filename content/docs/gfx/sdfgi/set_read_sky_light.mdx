---
title: gfx.sdfgi.set_read_sky_light
description: Toggles sky light sampling for the SDFGI post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.sdfgi.set_read_sky_light(state)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `bool` | state | Determines whether sky light is sampled:<br/>• When `true` - SDFGI reads and incorporates sky light into global illumination<br/>• When `false` - sky light contribution is ignored |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Include sky light in GI calculation"
--Include sky light in GI calculation
gfx.sdfgi.set_read_sky_light(true)
```
