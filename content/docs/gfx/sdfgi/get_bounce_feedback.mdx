### Syntax

```lua
local result = gfx.sdfgi.get_bounce_feedback()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current bounce feedback value |

<hr/>

### Examples

```lua
engine.print("info", "SDFGI bounce feedback:", gfx.sdfgi.get_bounce_feedback())
```
