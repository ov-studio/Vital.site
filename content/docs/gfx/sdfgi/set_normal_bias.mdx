---
title: gfx.sdfgi.set_normal_bias
description: The normal bias to be set for the SDFGI post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.sdfgi.set_normal_bias(value)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Normal bias value to be applied<br/>Offsets the GI sampling position along the surface normal to reduce self-occlusion and light leaking artifacts on geometry surfaces |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Apply a small normal bias to reduce surface artifacts"
gfx.sdfgi.set_normal_bias(1.1)
```
