---
title: gfx.sdfgi.get_cascades
description: Retrieves the current number of cascades of the SDFGI post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.sdfgi.get_cascades()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `int` | result | Current cascade count |

<hr/>

### Examples

```lua lineNumbers title="Print current SDFGI cascade count"
engine.print("info", "SDFGI cascades:", gfx.sdfgi.get_cascades())
```
