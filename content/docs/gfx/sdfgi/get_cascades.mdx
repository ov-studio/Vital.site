---

### Syntax

```lua
local result = gfx.sdfgi.get_cascades()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `int` | result | Current cascade count |

<hr/>

### Examples

```lua
engine.print("SDFGI cascades:", gfx.sdfgi.get_cascades())
```
