### Syntax

```lua
local result = gfx.sdfgi.get_normal_bias()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current normal bias value |

<hr/>

### Examples

```lua
engine.print("info", "SDFGI normal bias:", gfx.sdfgi.get_normal_bias())
```
