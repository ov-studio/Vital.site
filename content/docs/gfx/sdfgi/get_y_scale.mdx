---
title: gfx.sdfgi.get_y_scale
description: Retrieves the current Y-axis scale mode of the SDFGI post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.sdfgi.get_y_scale()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `int` | result | Current Y scale mode |

<hr/>

### Examples

```lua
engine.print("info", "SDFGI Y scale:", gfx.sdfgi.get_y_scale())
```
