---
title: gfx.sdfgi.is_enabled
description: Retrieves whether the SDFGI post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local result = gfx.sdfgi.is_enabled()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if SDFGI is enabled, `false` otherwise |

<hr/>

### Examples

```lua
gfx.print("SDFGI enabled:", gfx.sdfgi.is_enabled())
```
