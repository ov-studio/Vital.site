```lua

```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current energy value |

### Examples

```lua
engine.print("info", "SDFGI energy:", gfx.sdfgi.get_energy())
```
