---
title: gfx.volumetric_fog.set_emission
description: The emission color to be set for the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.volumetric_fog.set_emission(color)
local status = gfx.volumetric_fog.set_emission(hex)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `color` | color | Emission color as `{r, g, b, a}` |
| `string` | hex | Alternatively, a valid HTML hex color string (e.g. `"#ffffff"`) |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Set a warm orange emission color using a color table"
--Set a warm orange emission color using a color table
gfx.volumetric_fog.set_emission({1.0, 0.4, 0.1, 1.0})
```

```lua lineNumbers title="Set emission color using a hex string"
--Set emission color using a hex string
gfx.volumetric_fog.set_emission("#ff6619")
```
