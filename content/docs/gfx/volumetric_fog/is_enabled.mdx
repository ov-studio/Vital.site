---
title: gfx.volumetric_fog.is_enabled
description: Retrieves whether the volumetric fog post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local result = gfx.volumetric_fog.is_enabled()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if volumetric fog is enabled, `false` otherwise |

<hr/>

### Examples

```lua
engine.print("info", "Volumetric fog enabled:", gfx.volumetric_fog.is_enabled())
```
