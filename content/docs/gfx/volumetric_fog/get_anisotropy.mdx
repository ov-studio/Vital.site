```lua

```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current anisotropy value |

engine.print("info", "Volumetric fog anisotropy:", gfx.volumetric_fog.get_anisotropy())
