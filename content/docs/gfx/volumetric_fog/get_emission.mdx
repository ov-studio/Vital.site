---
title: gfx.volumetric_fog.get_emission
description: Retrieves the current emission color of the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local r, g, b, a = gfx.volumetric_fog.get_emission()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | r | Red channel value of the current emission color |
| `float` | g | Green channel value of the current emission color |
| `float` | b | Blue channel value of the current emission color |
| `float` | a | Alpha channel value of the current emission color |

<hr/>

### Examples

```lua
local r, g, b, a = gfx.volumetric_fog.get_emission()
engine.print("info", "Volumetric fog emission:", r, g, b, a)
```
