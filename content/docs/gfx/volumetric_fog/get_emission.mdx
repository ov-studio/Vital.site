---
title: gfx.volumetric_fog.get_emission
description: Retrieves the current emission color of the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local color = gfx.volumetric_fog.get_emission()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `color` | color | Volumetric fog emission color as `{r, g, b, a}` |

<hr/>

### Examples

```lua lineNumbers title="Print current volumetric fog emission color"
engine.print("info", "Volumetric fog emission:", table.unpack(gfx.volumetric_fog.get_emission()))
```
