---
title: gfx.volumetric_fog.get_albedo
description: Retrieves the current albedo color of the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local color = gfx.volumetric_fog.get_albedo()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `color` | color | Volumetric fog albedo color as `{r, g, b, a}` |

<hr/>

### Examples

```lua lineNumbers title="Print current volumetric fog albedo color"
engine.print("info", "Volumetric fog albedo:", table.unpack(gfx.volumetric_fog.get_albedo()))
```
