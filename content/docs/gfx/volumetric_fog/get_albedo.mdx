---
title: gfx.volumetric_fog.get_albedo
description: Retrieves the current albedo color of the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local r, g, b, a = gfx.volumetric_fog.get_albedo()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | r | Red channel value of the current albedo color |
| `float` | g | Green channel value of the current albedo color |
| `float` | b | Blue channel value of the current albedo color |
| `float` | a | Alpha channel value of the current albedo color |

<hr/>

### Examples

```lua
local r, g, b, a = gfx.volumetric_fog.get_albedo()
engine.print("Volumetric fog albedo:", r, g, b, a)
```
