---
title: gfx.volumetric_fog.set_albedo
description: The albedo color to be set for the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.volumetric_fog.set_albedo(color)
local status = gfx.volumetric_fog.set_albedo(hex)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `color` | color | Albedo color as `{r, g, b, a}` |
| `string` | hex | Alternatively, a valid HTML hex color string (e.g. `"#ffffff"`) |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Set a cool blue-tinted fog albedo using a color table"
--Set a cool blue-tinted fog albedo using a color table
gfx.volumetric_fog.set_albedo({0.6, 0.7, 1.0, 1.0})
```

```lua lineNumbers title="Set albedo color using a hex string"
--Set albedo color using a hex string
gfx.volumetric_fog.set_albedo("#99b3ff")
```
