---
title: gfx.volumetric_fog.set_density
description: The density to be set for the volumetric fog post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.volumetric_fog.set_density(value)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Density value to be applied<br/>Controls the base thickness of the volumetric fog. Higher values produce denser, more opaque fog that scatters and absorbs light more strongly throughout the volume |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
--Set a moderate volumetric fog density
gfx.volumetric_fog.set_density(0.05)
```
