---
title: gfx.volumetric_fog.get_ambient_inject

badge: Client
---

### Syntax

```lua
local result = gfx.volumetric_fog.get_ambient_inject()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current ambient inject value |

<hr/>

### Examples

```lua lineNumbers title="Print current volumetric fog ambient inject amount"
engine.print("info", "Volumetric fog ambient inject:", gfx.volumetric_fog.get_ambient_inject())
```
