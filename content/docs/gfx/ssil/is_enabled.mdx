---
title: gfx.ssil.is_enabled
description: Retrieves whether the SSIL post-processing effect is currently enabled
badge: Client
---

### Syntax

```lua
local result = gfx.ssil.is_enabled()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if SSIL is enabled, `false` otherwise |

<hr/>

### Examples

```lua
engine.print("SSIL enabled:", gfx.ssil.is_enabled())
```
