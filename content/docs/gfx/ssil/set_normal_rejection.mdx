---
title: gfx.ssil.set_normal_rejection
description: The normal rejection strength to be set for the SSIL post-processing effect
badge: Client
---

### Syntax

```lua
local status = gfx.ssil.set_normal_rejection(value)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Normal rejection value to be applied<br/>Controls how aggressively surfaces with diverging normals are excluded from contributing indirect light to one another, reducing light leaking across sharp geometric transitions |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
--Reduce light leaking at sharp geometry transitions
gfx.ssil.set_normal_rejection(1.0)
```
