### Syntax

```lua
local result = gfx.ssil.get_intensity()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current intensity value |

<hr/>

### Examples

```lua lineNumbers title="Print current ssil intensity"
engine.print("info", "SSIL intensity:", gfx.ssil.get_intensity())
```
