---
title: gfx.ssil.get_normal_rejection
description: Retrieves the current normal rejection strength of the SSIL post-processing effect
badge: Client
---

### Syntax

```lua
local result = gfx.ssil.get_normal_rejection()
```

### Parameters

| No parameters are accepted by this function |
|----------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current normal rejection value |

<hr/>

### Examples

```lua lineNumbers title="Print current ssil normal rejection"
engine.print("info", "SSIL normal rejection:", gfx.ssil.get_normal_rejection())
```
