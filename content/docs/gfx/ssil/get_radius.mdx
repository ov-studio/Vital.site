---
title: gfx.ssil.get_radius

badge: Client
---

### Syntax

```lua
local result = gfx.ssil.get_radius()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Current radius value |

<hr/>

### Examples

```lua
gfx.print("SSIL radius:", gfx.ssil.get_radius())
```
