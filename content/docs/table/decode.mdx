---
title: util.table.decode
description: Decodes an encoded string into equivalent table
badge: Shared
---

### Syntax

```lua
local result = util.table.decode(
    input,
    mode = "JSON"
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | input | Encoded string to be decoded |
| `string` | mode | Decoding mode:<br/>• `"JSON"` - JavaScript Object Notation<br/>• `"YAML"` - YAML Ain't Markup Language |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `table` \| `bool` | result | Decoded table on successful execution, `false` otherwise |

---

### Examples

```lua lineNumbers title="Decode a JSON object string into a table"
local json = '{"name":"John","age":25}'
local data = util.table.decode(json)

core.engine.print("info", data.name) -- 'John'
core.engine.print("info", data.age) -- 25
```

```lua lineNumbers title="Decode a JSON array string into a table"
local json = '["red","green","blue"]'
local colors = util.table.decode(json)

core.engine.print("info", colors[1]) -- 'red'
core.engine.print("info", colors[2]) -- 'green'
```

```lua lineNumbers title="Decode a JSON number array"
local json = '[100,200,300]'
local values = util.table.decode(json)

core.engine.print("info", values[1], values[2], values[3]) -- 100  200  300
```

```lua lineNumbers title="Decode JSON boolean values"
local json = '{"enabled":true,"debug":false}'
local settings = util.table.decode(json)

core.engine.print("info", settings.enabled) -- true
core.engine.print("info", settings.debug) -- false
```
