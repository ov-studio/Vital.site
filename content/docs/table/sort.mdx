---
title: table.sort
description: Sorts table elements in-place
badge: Shared
---

### Syntax

```lua
table.sort(input, comparison = nil)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `table` | input | Table to be sorted |
| `function` | comparison | Comparison function `compare(a, b)` - used for manipulating sorting algorithm |

### Returns

| Type | Name | Description |
|------|------|-------------|

<hr/>

### Examples

```lua lineNumbers title="Sort numbers in ascending order"
local nums = {5, 2, 8, 1, 9}
table.sort(nums)

engine.print("info", table.concat(nums, ", ")) --"1, 2, 5, 8, 9"
```

```lua lineNumbers title="Sort numbers in descending order"
local nums = {5, 2, 8, 1, 9}
table.sort(nums, function(a, b) return a > b end)

engine.print("info", table.concat(nums, ", ")) --"9, 8, 5, 2, 1"
```

```lua lineNumbers title="Sort strings alphabetically"
local words = {"zebra", "apple", "mango", "banana"}
table.sort(words)

engine.print("info", table.concat(words, ", ")) --"apple, banana, mango, zebra"
```

```lua lineNumbers title="Sort strings by length"
local words = {"hi", "hello", "a", "world"}
table.sort(words, function(a, b) return #a < #b end)

engine.print("info", table.concat(words, ", ")) --"a, hi, hello, world"
```

```lua lineNumbers title="Sort strings case-insensitively"
local names = {"Bob", "alice", "Charlie", "david"}
table.sort(names, function(a, b) 
    return string.lower(a) < string.lower(b) 
end)

engine.print("info", table.concat(names, ", ")) --"alice, Bob, Charlie, david"
```

```lua lineNumbers title="Sort mixed-case strings uniformly"
local items = {"Apple", "banana", "Cherry", "date"}
table.sort(items, function(a, b)
    return string.lower(a) < string.lower(b)
end)

engine.print("info", table.concat(items, ", ")) --"Apple, banana, Cherry, date"
```
