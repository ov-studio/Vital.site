---
title: file.write
description: Writes content to a specified file
badge: Shared
---

### Syntax

```lua
local status = file.write(path, content)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | path | File path to write the contents to |
| `string` | content | Content to be written |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
--Writes content to a file within current resource
engine.print("info", "File written:", file.write("files/data.json", "{}"))
```

```lua
--Writes content to a file path within current resource
engine.print("info", "File written:", file.write("files/logs/session.log", "Session started"))
```
