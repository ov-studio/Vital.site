---
title: file.write
description: Writes content to specified file
badge: Shared
---

### Syntax

```lua
local status = file.write(path, content)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | path | File path to write the contents in |
| `string` | content | Content to be written |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua
engine.print("info", "File written:", file.read("files/images/test.png"))
```
