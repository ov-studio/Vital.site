---
title: file.size
description: Retrieves size of specified file
badge: Shared
---

### Syntax

```lua
local state = file.delete(path)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | path | File path for size computation |


### Returns

| Type | Name | Description |
|------|------|-------------|
| `int` or `bool` | valid | Computed size of specified file on successful execution, or `false` on failure |

<hr/>

### Example

```lua
engine.print("File size:", file.size("test.png"))
```
