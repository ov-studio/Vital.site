---
title: file.exists
description: Retrieves the existence of a specified file
badge: Shared
---

### Syntax

```lua
local valid = file.exists(path)
```

<Callout type="idea">
  **Prefix the path with `:resourcename` to access files from another running resource.**

  - **Local path** — resolves relative to the current resource's directory
  - **Cross-resource path** — prefix with `:resourcename` to resolve relative to another running resource's directory
  - **Example** — `:resource_a/files/images/test.png` accesses `files/images/test.png` inside `resource_a`
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | path | File path to verify for existence |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | valid | Specified file's existence |

<hr/>

### Examples

```lua
--Queries a file within current resource
engine.print("info", "File exists:", file.exists("files/images/test.png"))
```

```lua
--Queries a file from another running resource
engine.print("info", "File exists:", file.exists(":resource_a/files/images/test.png"))
```
