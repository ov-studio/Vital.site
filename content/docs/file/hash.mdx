---
title: file.hash
description: Retrieves the hash of a specified file
badge: Shared
---

### Syntax

```lua
local hash = file.hash(
    path,
    mode
)
```

<Callout type="idea">
  **Prefix the path with `:resourcename` to access files from another running resource.**

  - **Local path** — resolves relative to the current resource's directory
  - **Cross-resource path** — prefix with `:resourcename` to resolve relative to another running resource's directory
  - **Example** — `:resource_a/files/images/test.png` accesses `files/images/test.png` inside `resource_a`
</Callout>

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | path | File path to hash |
| `string` | mode | Hashing algorithm to use - same modes as used in [`crypto.hash`](/docs/crypto/hash) |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` / `bool` | hash | Computed file hash on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Hash a file within the current resource"
engine.print("info", "File hash:", file.hash("files/images/test.png", "SHA256"))
```

```lua lineNumbers title="Hash a file from another running resource"
engine.print("info", "File hash:", file.hash(":resource_a/files/images/test.png", "SHA256"))
```
