---
title: file.delete
description: Deletes specified file
badge: Shared
---

### Syntax

```lua
local status = file.delete(path)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | path | File path to delete |


### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | Returns `true` on successful execution, or `false` on failure |

<hr/>

### Example

```lua
engine.print("File deleted:", file.delete("files/images/test.png"))
```
