---
title: math.floor
description: Rounds a number down to the nearest integer
badge: Shared
---

### Syntax

```lua
local result = math.floor(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `number` | value | Number to round down |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `int` | result | Largest integer less than or equal to value |

---

### Examples

```lua lineNumbers title="Floor a positive float"
local result = math.floor(4.9)

core.engine.print("info", result) -- 4
```

```lua lineNumbers title="Floor a negative float"
local result = math.floor(-4.2)

core.engine.print("info", result) -- -5
```

```lua lineNumbers title="Floor an exact integer returns itself"
local result = math.floor(3.0)

core.engine.print("info", result) -- 3
```

```lua lineNumbers title="Truncate a float to integer"
local result = math.floor(9.99)

core.engine.print("info", result) -- 9
```

```lua lineNumbers title="Compute integer division"
local result = math.floor(17 / 5)

core.engine.print("info", result) -- 3
```

```lua lineNumbers title="Snap position to grid"
local pos_x = 137.6
local grid_size = 32
local snapped = math.floor(pos_x / grid_size) * grid_size

core.engine.print("info", snapped) -- 128
```

```lua lineNumbers title="Convert seconds to whole minutes"
local seconds = 185
local minutes = math.floor(seconds / 60)

core.engine.print("info", minutes) -- 3
```
