### Value

```lua
math.pi -- 3.1415926535898
```

A constant — not a function. Represents the ratio of a circle's circumference to its diameter.

<hr/>

### Examples

```lua lineNumbers title="Print the value of pi"
engine.print("info", math.pi) --3.1415926535898
```

```lua lineNumbers title="Compute the circumference of a circle"
local radius = 5
local circumference = 2 * math.pi * radius

engine.print("info", circumference) --31.415926535898
```

```lua lineNumbers title="Compute the area of a circle"
local radius = 3
local area = math.pi * radius * radius

engine.print("info", area) --28.274333882308
```

```lua lineNumbers title="Convert 180 degrees to radians using pi"
local radians = math.pi

engine.print("info", radians) --3.1415926535898
```

```lua lineNumbers title="Generate a full circle of evenly spaced points"
local n = 8
for i = 0, n - 1 do
    local angle = 2 * math.pi * i / n
    local x = math.cos(angle)
    local y = math.sin(angle)
    engine.print("info", x, y)
end
```

```lua lineNumbers title="Compute the volume of a sphere"
local r = 4
local volume = (4 / 3) * math.pi * r * r * r

engine.print("info", volume) --268.08257310942
```
