---
title: math.abs
description: Returns the absolute value of a number
badge: Shared
---

### Syntax

```lua
local result = math.abs(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `number` | value | Number to get the absolute value of |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `number` | result | Absolute (non-negative) value of the input |

---

### Examples

```lua lineNumbers title="Absolute value of a negative number"
local result = math.abs(-5)

core.engine.print("info", result) -- 5
```

```lua lineNumbers title="Absolute value of a positive number"
local result = math.abs(42)

core.engine.print("info", result) -- 42
```

```lua lineNumbers title="Absolute value of zero"
local result = math.abs(0)

core.engine.print("info", result) -- 0
```

```lua lineNumbers title="Absolute value of a negative float"
local result = math.abs(-3.14)

core.engine.print("info", result) -- 3.14
```

```lua lineNumbers title="Calculate distance between two numbers"
local a = 10
local b = 35
local distance = math.abs(a - b)

core.engine.print("info", distance) -- 25
```

```lua lineNumbers title="Clamp a delta to avoid overshoot"
local delta = -0.75
local speed = math.abs(delta) * 10

core.engine.print("info", speed) -- 7.5
```

```lua lineNumbers title="Check if a value is within tolerance"
local value = -0.0001
local tolerance = 0.001

if math.abs(value) < tolerance then
    core.engine.print("info", "Within tolerance")
end
```
