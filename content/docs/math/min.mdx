---
title: util.math.min
description: Returns the smallest value among the given arguments
badge: Shared
---

### Syntax

```lua
local result = util.math.min(
    value,
    ...
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | First number |
| `float` | ... | Additional numbers to compare |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | The smallest of all given values |

---

### Examples

```lua lineNumbers title="Min of two numbers"
local result = util.math.min(3, 7)

core.engine.print("info", result) -- 3
```

```lua lineNumbers title="Min of multiple numbers"
local result = util.math.min(10, 5, 8, 1, 6)

core.engine.print("info", result) -- 1
```

```lua lineNumbers title="Min with negative numbers"
local result = util.math.min(-1, -5, -3)

core.engine.print("info", result) -- -5
```

```lua lineNumbers title="Min of a single value returns itself"
local result = util.math.min(99)

core.engine.print("info", result) -- 99
```

```lua lineNumbers title="Clamp a value to a maximum ceiling"
local speed = 150
local clamped = util.math.min(speed, 100)

core.engine.print("info", clamped) -- 100
```

```lua lineNumbers title="Clamp a value between a min and max"
local value = 120
local clamped = util.math.max(0, util.math.min(value, 100))

core.engine.print("info", clamped) -- 100
```

```lua lineNumbers title="Cap items rendered to available slots"
local item_count = 25
local slot_count = 10
local render_count = util.math.min(item_count, slot_count)

core.engine.print("info", render_count) -- 10
```
