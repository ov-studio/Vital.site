---
title: math.min
description: Returns the smallest value among the given arguments
badge: Shared
---

### Syntax

```lua
local result = math.min(
    x,
    ...
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `number` | x | First number |
| `number` | ... | Additional numbers to compare |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `number` | result | The smallest of all given values |

<hr/>

### Examples

```lua lineNumbers title="Min of two numbers"
local result = math.min(3, 7)

engine.print("info", result) --3
```

```lua lineNumbers title="Min of multiple numbers"
local result = math.min(10, 5, 8, 1, 6)

engine.print("info", result) --1
```

```lua lineNumbers title="Min with negative numbers"
local result = math.min(-1, -5, -3)

engine.print("info", result) ---5
```

```lua lineNumbers title="Min of a single value returns itself"
local result = math.min(99)

engine.print("info", result) --99
```

```lua lineNumbers title="Clamp a value to a maximum"
local speed = 150
local clamped = math.min(speed, 100)

engine.print("info", clamped) --100
```

```lua lineNumbers title="Clamp a value between a min and max"
local value = 120
local clamped = math.max(0, math.min(value, 100))

engine.print("info", clamped) --100
```

```lua lineNumbers title="Cap items rendered to available slots"
local items = 25
local slots = 10
local render = math.min(items, slots)

engine.print("info", render) --10
```
