---
title: util.math.deg
description: Converts an angle from radians to degrees
badge: Shared
---

### Syntax

```lua
local result = util.math.deg(
    angle
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `number` | angle | Angle in radians to convert |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `number` | result | Equivalent angle in degrees |

---

### Examples

```lua lineNumbers title="Convert pi radians to degrees"
local result = util.math.deg(util.math.pi)

core.engine.print("info", result) -- 180.0
```

```lua lineNumbers title="Convert pi/2 to degrees"
local result = util.math.deg(util.math.pi / 2)

core.engine.print("info", result) -- 90.0
```

```lua lineNumbers title="Convert 0 radians to degrees"
local result = util.math.deg(0)

core.engine.print("info", result) -- 0.0
```

```lua lineNumbers title="Convert 2*pi to degrees"
local result = util.math.deg(2 * util.math.pi)

core.engine.print("info", result) -- 360.0
```

```lua lineNumbers title="Convert an atan2 result to readable degrees"
local angle = util.math.atan(1, 0)
local degrees = util.math.deg(angle)

core.engine.print("info", degrees) -- 90.0
```

```lua lineNumbers title="Print common radian values in degrees"
for _, rad in ipairs({0, util.math.pi/6, util.math.pi/4, util.math.pi/3, util.math.pi/2}) do
    core.engine.print("info", util.math.deg(rad))
end

--[[
Output:
0.0
30.0
45.0
60.0
90.0
]]
```

```lua lineNumbers title="Verify deg and rad are inverses"
local original = 135
local result = util.math.deg(util.math.rad(original))

core.engine.print("info", result) -- 135.0
```
