---
title: util.math.mininteger
description: The minimum value a Lua integer can hold
badge: Shared
---

### Constant

```lua
util.math.mininteger -- -9223372036854775808
```

<Callout type="info">
    Represents the smallest (most negative) integer value representable by Lua's 64-bit signed integer type (-2^63).
</Callout>

---

### Examples

```lua lineNumbers title="Print the minimum integer value"
core.engine.print("info", util.math.mininteger) -- -9223372036854775808
```

```lua lineNumbers title="mininteger minus 1 wraps around to maxinteger"
local result = util.math.mininteger - 1

core.engine.print("info", result) -- 9223372036854775807
```

```lua lineNumbers title="Use as a lower sentinel for maximum searching"
local values = {100, 200, 50, 400}
local largest = util.math.mininteger

for index, v in ipairs(values) do
    if v > largest then largest = v end
end

core.engine.print("info", largest) -- 400
```

```lua lineNumbers title="mininteger is less than any other integer"
core.engine.print("info", util.math.mininteger < -1000000000) -- true
```

```lua lineNumbers title="Safely take absolute value near the boundary"
local safe = util.math.mininteger + 1

core.engine.print("info", util.math.abs(safe)) -- 9223372036854775807
```
