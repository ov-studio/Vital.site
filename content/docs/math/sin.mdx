---
title: math.sin
description: Returns the sine of an angle given in radians
badge: Shared
---

### Syntax

```lua
local result = math.sin(
    angle
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `number` | angle | Angle in radians |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `number` | result | Sine of angle, in the range [-1, 1] |

---

### Examples

```lua lineNumbers title="Sine of 0"
local result = math.sin(0)

core.engine.print("info", result) -- 0.0
```

```lua lineNumbers title="Sine of pi/2 is 1"
local result = math.sin(math.pi / 2)

core.engine.print("info", result) -- 1.0
```

```lua lineNumbers title="Sine of a 30 degree angle"
local result = math.sin(math.rad(30))

core.engine.print("info", result) -- 0.5
```

```lua lineNumbers title="Sine of a 90 degree angle"
local result = math.sin(math.rad(90))

core.engine.print("info", result) -- 1.0
```

```lua lineNumbers title="Animate vertical position with a sine wave"
local elapsed = 0.5
local amplitude = 10
local frequency = 2
local pos_y = amplitude * math.sin(frequency * elapsed)

core.engine.print("info", pos_y) -- 8.4147
```

```lua lineNumbers title="Generate a circular position"
local angle = math.pi / 4
local radius = 5
local pos_x = radius * math.cos(angle)
local pos_y = radius * math.sin(angle)

core.engine.print("info", pos_x, pos_y) -- 3.5355  3.5355
```

```lua lineNumbers title="Compute sine for each cardinal angle"
for _, deg in ipairs({0, 90, 180, 270, 360}) do
    core.engine.print("info", deg, math.sin(math.rad(deg)))
end
```
