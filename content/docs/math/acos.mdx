---
title: util.math.acos
description: Returns the arc cosine of a number in radians
badge: Shared
---

### Syntax

```lua
local result = util.math.acos(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `number` | value | Cosine value in the range [-1, 1] |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `number` | result | Arc cosine in radians, in the range [0, pi] |

---

### Examples

```lua lineNumbers title="Arc cosine of 1 is 0"
local result = util.math.acos(1)

core.engine.print("info", result) -- 0.0
```

```lua lineNumbers title="Arc cosine of 0 is pi/2"
local result = util.math.acos(0)

core.engine.print("info", result) -- 1.5707963267949
```

```lua lineNumbers title="Arc cosine of -1 is pi"
local result = util.math.acos(-1)

core.engine.print("info", result) -- 3.1415926535898
```

```lua lineNumbers title="Arc cosine of 0.5 is 60 degrees"
local result = util.math.deg(util.math.acos(0.5))

core.engine.print("info", result) -- 60.0
```

```lua lineNumbers title="Compute angle between two normalized vectors"
local dot_product = 0.5
local angle_deg = util.math.deg(util.math.acos(dot_product))

core.engine.print("info", angle_deg) -- 60.0
```

```lua lineNumbers title="Verify acos and cos are inverses"
local angle = util.math.pi / 3
local result = util.math.acos(util.math.cos(angle))

core.engine.print("info", result) -- 1.0471975511966
```
