---
title: math.exp
description: Returns e raised to the power of x
badge: Shared
---

### Syntax

```lua
local result = math.exp(
    x
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `number` | x | Exponent to raise e to |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `number` | result | e^x (approximately 2.71828^x) |

<hr/>

### Examples

```lua lineNumbers title="e to the power of 1"
local result = math.exp(1)

engine.print("info", result) --2.718281828459
```

```lua lineNumbers title="e to the power of 0"
local result = math.exp(0)

engine.print("info", result) --1.0
```

```lua lineNumbers title="e to the power of 2"
local result = math.exp(2)

engine.print("info", result) --7.3890560989307
```

```lua lineNumbers title="e to a negative power"
local result = math.exp(-1)

engine.print("info", result) --0.36787944117144
```

```lua lineNumbers title="Exponential decay formula"
local initial = 100
local rate = 0.05
local time = 10
local remaining = initial * math.exp(-rate * time)

engine.print("info", remaining) --60.653065971263
```

```lua lineNumbers title="Verify exp and log are inverses"
local x = 3.5
local result = math.exp(math.log(x))

engine.print("info", result) --3.5
```
