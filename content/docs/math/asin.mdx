---
title: util.math.asin
description: Returns the arc sine of a number in radians
badge: Shared
---

### Syntax

```lua
local result = util.math.asin(
    value
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `float` | value | Sine value in the range [-1, 1] |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `float` | result | Arc sine in radians, in the range [-pi/2, pi/2] |

---

### Examples

```lua lineNumbers title="Arc sine of 0"
local result = util.math.asin(0)

core.engine.print("info", result) -- 0.0
```

```lua lineNumbers title="Arc sine of 1 is pi/2"
local result = util.math.asin(1)

core.engine.print("info", result) -- 1.5707963267949
```

```lua lineNumbers title="Arc sine of 0.5 is 30 degrees"
local result = util.math.deg(util.math.asin(0.5))

core.engine.print("info", result) -- 30.0
```

```lua lineNumbers title="Arc sine of -1 is -pi/2"
local result = util.math.asin(-1)

core.engine.print("info", result) -- -1.5707963267949
```

```lua lineNumbers title="Recover angle from a vertical direction component"
local dir_y = 0.866 -- sin(60 degrees)
local angle_deg = util.math.deg(util.math.asin(dir_y))

core.engine.print("info", angle_deg) -- 59.999
```

```lua lineNumbers title="Verify asin and sin are inverses"
local angle = util.math.pi/6
local result = util.math.asin(util.math.sin(angle))

core.engine.print("info", result) -- 0.5235987755983
```
