---
title: core.font.create
description: Creates a new font instance from a file path
badge: Client
---

### Syntax

```lua
local instance = core.font.create(
    path
)
```

### Parameters

| Type | Name | Description |
|------|------|-------------|
| `string` | path | File path to a supported font asset<br/>Refer [**Formats**](#formats) section |

### Returns

| Type | Name | Description |
|------|------|-------------|
| `font` | instance | Created font instance |

---

### Formats

| Format | Extension | Description |
|--------|-----------|-------------|
| TrueType | `.ttf` | Standard scalable font format |
| OpenType | `.otf` | Extended scalable font format |
| Web Open Font | `.woff` | Compressed web font format |
| Web Open Font 2 | `.woff2` | Modern compressed web font format |

---

### Examples

```lua lineNumbers title="Create a font instance from a TTF file"
local entity = core.font.create("assets/core.font.ttf")
```

```lua lineNumbers title="Create a font instance from an OTF file"
local entity = core.font.create("assets/core.font.otf")
```

```lua lineNumbers title="Create a font instance from a WOFF2 file"
local entity = core.font.create("assets/core.font.woff2")
```
