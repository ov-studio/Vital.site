---
title: self:destroy
description: Destroys the font instance and frees its resources
badge: Client
---

### Syntax

```lua
local status = self:destroy()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | status | `true` on successful execution, or `false` on failure |

<hr/>

### Examples

```lua lineNumbers title="Destroy a font instance"
local ft = font.create("fonts/Roboto-Regular.ttf")

ft:destroy()
```
