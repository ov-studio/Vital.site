---
title: self:get_type
description: Returns the type name of the instance
badge: Client
---

### Syntax

```lua
local result = self:get_type()
```

### Parameters

| Type | Name | Description |
|------|------|-------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `string` / `bool` | result | Type of the instance, or `false` if unavailable |

<hr/>

### Examples

```lua lineNumbers title="Get the type of a font instance"
local fontface = font.create("fonts/Roboto-Regular.ttf")

engine.print("info", fontface:get_type()) --"font"
```
