---
title: self:is_antialiased
description: Retrieves the antialiasing state of the font
badge: Client
---

### Syntax

```lua
local result = self:is_antialiased()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if antialiasing is enabled, `false` otherwise |

---

### Examples

```lua lineNumbers title="Get the antialiasing state of a font"
local entity = core.font.create("fonts/Roboto-Regular.ttf")

core.engine.print("info", "Font antialiasing:", entity:is_antialiased())
```
