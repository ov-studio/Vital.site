---
title: self:get_antialiasing
description: Retrieves the antialiasing state of the font
badge: Client
---

### Syntax

```lua
local result = self:get_antialiasing()
```

### Parameters

| No parameters are accepted by this function |
|---------------------------------------------|

### Returns

| Type | Name | Description |
|------|------|-------------|
| `bool` | result | `true` if antialiasing is enabled, `false` otherwise |

<hr/>

### Examples

```lua lineNumbers title="Get the antialiasing state of a font"
local ft = font.create("fonts/Roboto-Regular.ttf")

engine.print("info", "Font antialiasing:", ft:get_antialiasing())
```
